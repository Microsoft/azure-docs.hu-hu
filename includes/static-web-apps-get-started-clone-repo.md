## <a name="clone-the-repository"></a>A tárház klónozása

A GitHub-fiókban létrehozott adattárral a következő parancs használatával klónozott a projektet a helyi gépre.

```bash
git clone https://github.com/<YOUR_GITHUB_ACCOUNT_NAME>/my-first-static-web-app.git
```

Ügyeljen arra, hogy a `<YOUR_GITHUB_ACCOUNT_NAME>` GitHub-felhasználónevét cserélje le.

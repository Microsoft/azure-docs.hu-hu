[Az Azure App Service](/azure/app-service/overview) hozhat létre és üzemeltethet kód lehetővé teszi. Ez a kód a web apps, mobil háttérszolgáltatások és RESTful API-k formájában is lehet. App Service használatával kód, az Ön által választott programozási nyelv használatával infrastruktúra kezelése nélkül is fejleszthet. App Service-ben a Windows és Linux egyaránt támogatja. Automatikus telepítéseket a bármely egyéb Git-adattárból támogatottak, például a GitHub és az Azure DevOps.

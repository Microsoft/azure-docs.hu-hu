## <a name="quickstarts-and-tutorials"></a>Rövid útmutatók és oktatóanyagok

Az alábbi rövid útmutatók és oktatóanyagok segítségével megismerheti, hogyan fejleszthet Resource Manager-sablonokat:

- Gyors útmutatók

    |Beosztás|Leírás|
    |------|-----|
    |[Az Azure Portal használata](../articles/azure-resource-manager/resource-manager-quickstart-create-templates-use-the-portal.md)|Hozzon létre egy sablont a portálról, és megismerheti a szerkesztési és helyezi üzembe a sablont.|
    |[A Visual Studio Code használata](../articles/azure-resource-manager/resource-manager-quickstart-create-templates-use-visual-studio-code.md)|Sablonok létrehozása és szerkesztése a Visual Studio Code használatával, és a sablonok üzembe helyezése az Azure Cloud Shell-lel.|
    |[A Visual Studio használata](../articles/azure-resource-manager/vs-azure-tools-resource-groups-deployment-projects-create-deploy.md)|Sablonok létrehozása, szerkesztése és üzembe helyezése a Visual Studióval.|

- Oktatóanyagok

    |Beosztás|Leírás|
    |------|-----|
    |[A sablonreferencia felhasználása](../articles/azure-resource-manager/resource-manager-tutorial-create-encrypted-storage-accounts.md)|A sablon referenciadokumentációjának használata sablonok fejlesztéséhez. Az oktatóanyagban megtalálja a tárfiók sémáját, és létrehoz egy titkosított tárfiókot az információk felhasználásával.|
    |[Több példány létrehozása](../articles/azure-resource-manager/resource-manager-tutorial-create-multiple-instances.md)|Azure-erőforrások több példányának létrehozása. Az oktatóanyagban több tárfiókpéldányt hoz létre.|
    |[Erőforrások üzembehelyezési sorrendjének beállítása](../articles/azure-resource-manager/resource-manager-tutorial-create-templates-with-dependent-resources.md)|Erőforrás-függőségek meghatározása. Az oktatóanyagban létrehoz egy virtuális hálózatot, egy virtuális gépet és a függő Azure-erőforrásokat. Megismerheti a függőségek meghatározásának a módját.|
    |[Feltételek használata](../articles/azure-resource-manager/resource-manager-tutorial-use-conditions.md)|Erőforrások üzembe helyezése paraméterértékek alapján. Az oktatóanyagban meghatároz egy sablont egy új tárfiók létrehozásához, vagy egy meglévő tárfiókot használ egy paraméterérték alapján.|
    |[A Key Vault integrálása](../articles/azure-resource-manager/resource-manager-tutorial-use-key-vault.md)|Titkos kulcsok/jelszavak lekérése az Azure Key Vaultból. Az oktatóanyagban létrehoz egy virtuális gépet.  A virtuális gép rendszergazdai jelszavának lekérése egy Key Vaultból történik.|
    |[Csatolt sablonok létrehozása](../articles/azure-resource-manager/resource-manager-tutorial-create-linked-templates.md)|Sablonok modularizálása, és más sablonok meghívása egy sablonból. Az oktatóanyagban létrehoz egy virtuális hálózatot, egy virtuális gépet és a függő erőforrásokat.  A függő tárfiók egy hivatkozott sablonban van meghatározva. |
    |[Virtuális gépi bővítmények üzembe helyezése](../articles/azure-resource-manager/resource-manager-tutorial-deploy-vm-extensions.md)|Üzembe helyezés utáni feladatok végrehajtása a bővítmények segítségével. Az oktatóanyagban egy ügyfél szkriptbővítmény webalkalmazás-kiszolgáló telepítése a virtuális gép üzembe helyezése. |
    |[SQL-bővítmények telepítése](../articles/azure-resource-manager/resource-manager-tutorial-deploy-sql-extensions-bacpac.md)|Üzembe helyezés utáni feladatok végrehajtása a bővítmények segítségével. Az oktatóanyagban egy ügyfél szkriptbővítmény webalkalmazás-kiszolgáló telepítése a virtuális gép üzembe helyezése. |
    |[Biztonságos összetevők](../articles/azure-resource-manager/resource-manager-tutorial-secure-artifacts.md)|Tegye biztonságossá a telepítés befejezéséhez szükséges összetevőket. Az oktatóanyagban elsajátíthatja, hogyan védheti meg az SQL telepítése bővítmények az oktatóanyagban használt összetevő. |
    |[Biztonságos üzembehelyezési gyakorlatok](../articles/azure-resource-manager/deployment-manager-tutorial.md)|Az Azure Deployment Manager használata. |

Ezekben az oktatóanyagokban használható önállóan, vagy egy Resource Manager sablon fejlesztési főbb fogalmakat további.

A virtuális gépeket API-k használatával hozhatja létre és kezelheti Azure Stack peremhálózati eszközön. Ezek az API-k standard Azure Resource Manager API-k, amelyek a helyi Azure Stack Edge-végpont használatával lettek meghívva. A Azure Resource Manager API-k konzisztens felügyeleti réteget biztosítanak, amely ebben az esetben lehetővé teszi a virtuális gépek létrehozását, frissítését és törlését az eszközön található helyi előfizetésekben. Azure PowerShell-parancsmagok segítségével csatlakozhat az Azure Stack Edge-ben futó Azure Resource Managerhoz.

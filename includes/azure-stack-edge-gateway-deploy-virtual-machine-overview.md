Az Azure Stack Edge-eszközökön API-kkal hozhat létre és felügyelhet virtuális gépeket (VM-eket). Ezek az API-k standard Azure Resource Manager API-k, amelyek a helyi Azure Stack Edge-végpont használatával lettek meghívva. Az Azure Resource Manager API-k konzisztens felügyeleti réteget biztosítanak, amely ebben az esetben virtuális gépek létrehozását, frissítését és törlését teszi lehetővé az eszközön lévő helyi előfizetésben. Azure PowerShell-parancsmagokkal csatlakozhat az Azure Stack Edge-en futó Azure Resource Managerhez.

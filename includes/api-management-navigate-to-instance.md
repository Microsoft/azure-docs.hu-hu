1. A Azure Portal keresse meg és válassza a API Management **szolgáltatásokat.**

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-1.png" alt-text="Válassza a API Management szolgáltatásokat":::

1. A API Management **szolgáltatások lapon** válassza ki a API Management példányát.

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-2.png" alt-text="Válassza ki a API Management-példányt":::

- **[Monitorozás Azure Data Studio irányítópultokkal](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> A Azure Data Studio egy platformfüggetlen adatbázis-eszköz olyan adatszakemberek számára, akik a helyszíni és a Felhőbeli adatplatformok Microsoft-családját használják Windows, MacOS és Linux rendszeren. Azure Data Studio modern szerkesztői élményt nyújt az IntelliSense, a kódrészletek, a verziókövetés integrációja és egy integrált terminál számára. Az adatplatform-felhasználó szem előtt tartva a lekérdezési eredményhalmaz és a testreszabható irányítópultok beépített ábrázolásával.

**[Figyelés a Grafana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> A Grafana egy többplatformos nyílt forráskódú elemzési és interaktív vizualizációs webalkalmazás. Diagramokat, diagramokat és riasztásokat biztosít a weben a támogatott adatforrásokhoz való csatlakozáskor. Ez egy beépülőmodul-rendszeren keresztül bővíthető. A végfelhasználók az interaktív lekérdezés használatával összetett figyelési irányítópultokat hozhatnak létre.

**[A log Analytics és a Kibana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>A Kibana egy nyílt forráskódú adatvizualizációs irányítópult a Elasticsearch számára. Vizualizációs képességeket biztosít a Elasticsearch-fürtön indexelt tartalom felett. A felhasználók sáv-, vonal-és pontdiagram-diagramokat, illetve kördiagramokat és térképeket hozhatnak létre nagy mennyiségű adattal.

## <a name="clean-up-resources"></a>Az erőforrások eltávolítása

Ha folytatja a [következő lépéssel](#next-steps) , és hozzáad egy Azure Storage-üzenetsor-kötést a függvényhez, az összes erőforrást meg kell őriznie, hogy a már elvégzett műveletekre építsen.

Ellenkező esetben a következő lépésekkel törölheti a Function alkalmazást és annak kapcsolódó erőforrásait, hogy elkerülje a további költségek felmerülését.

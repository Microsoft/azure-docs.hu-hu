> [!TIP]
> A cikk információi az eredeti Azure Table Storage-ra vonatkoznak. Azonban most már létezik egy alternatív ajánlat a Table Storage-hoz: a Azure Cosmos DB Table API. Ez az API nagyobb teljesítményt és rendelkezésre állást, globális elosztást és automatikus másodlagos indexeket biztosít. A szolgáltatás fogyasztáson alapuló [kiszolgáló](../articles/cosmos-db/serverless.md) nélküli módban is elérhető. A Azure Cosmos DB és az Azure Table Storage Table API között néhány [szolgáltatásbeli különbség](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage) van. További információ: [Azure Cosmos DB Table API](../articles/cosmos-db/table-introduction.md).

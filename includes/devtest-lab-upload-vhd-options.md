## <a name="prerequisites"></a>Előfeltételek

- [Feltöltés a AzCopy parancssori segédprogram használatával](../articles/lab-services/devtest-lab-upload-vhd-using-azcopy.md)
- [Feltöltés a Microsoft Azure Storage Explorer használatával](../articles/lab-services/devtest-lab-upload-vhd-using-storage-explorer.md)
- [Feltöltés a PowerShell-konzol használatával](../articles/lab-services/devtest-lab-upload-vhd-using-powershell.md)

## <a name="prerequisites"></a>Előfeltételek 
Az oktatóanyag befejezéséhez egy VHD-fájlfeltöltésre kell, hogy legyen feltöltve ahhoz a tesztkörnyezethez társított tárfiókba, amelyben az egyéni lemezképet hoz létre. Válassza ki az alábbi hivatkozások bármelyikét a VHD-fájl feltöltéséhez:

- [Feltöltés az AzCopy parancssori segédprogrammal](../articles/lab-services/devtest-lab-upload-vhd-using-azcopy.md)
- [Feltöltés a Microsoft Azure Storage Explorer használatával](../articles/lab-services/devtest-lab-upload-vhd-using-storage-explorer.md)
- [Feltöltés a PowerShell-konzolhasználatával](../articles/lab-services/devtest-lab-upload-vhd-using-powershell.md)

## <a name="add-a-consumer-group-to-your-iot-hub"></a>Az IoT hub fogyasztói csoport hozzáadása

Adatok lekérése az Azure IoT Hub fogyasztói csoportok alkalmazások használják. Ebben az oktatóanyagban létrehoz egy fogyasztói csoportot adatokat olvasni az IoT hub a következő Azure-szolgáltatás által használandó.

Az IoT hub fogyasztói csoport hozzáadásához kövesse az alábbi lépéseket:

1. Az [Azure Portalon](https://portal.azure.com/) nyissa meg az IoT Hubot.

2. A bal oldali ablaktáblán kattintson a **végpontok**válassza **események** a középső panelen adjon meg egy nevet, a **fogyasztói csoportok** a jobb oldali ablaktáblán, és kattintson a **mentése** .

   ![Az IoT hub fogyasztói csoport létrehozása](./media/iot-hub-get-started-create-consumer-group/iot-hub-create-consumer-group-azure.png)

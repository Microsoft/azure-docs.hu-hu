## <a name="azure-cognitive-service-subscription-types"></a>Az Azure Cognitive Services-szolgáltatás előfizetéstípusok

> [!NOTE]
> Az előfizetés-tulajdonosokat letilthatja a Cognitive Services-fiókok az erőforráscsoportok és előfizetések létrehozása alkalmazásával [az Azure policy](https://docs.microsoft.com/azure/governance/policy/overview#policy-definition), a "nem engedélyezett erőforrástípusok" szabályzatdefiníció hozzárendelése és megadása**Microsoft.CognitiveServices/accounts** a cél erőforrás típusaként.

Az Azure Cognitive Services két különböző előfizetésben keresztül érhető el: Több szolgáltatásos előfizetést, vagy egy egyetlen-szolgáltatás. Ezek az előfizetések lehetővé teszik vagy egyetlen szolgáltatás, vagy több szolgáltatást egyszerre csatlakozni.

### <a name="multi-service-subscription"></a>Több szolgáltatásos előfizetés

>[!WARNING]
> Jelenleg ezek a szolgáltatások **nem** támogatja több szolgáltatásos kulcsokat: A QnA Maker, beszédszolgáltatások, egyéni vizuális és Anomáliadetektálási detector használatával.

Az Azure Cognitive Services több szolgáltatásos előfizetés nagy része az Azure Cognitive Services egy egyetlen előfizetés és Azure-erőforrások használatát teszi lehetővé, és egyesíti a számlázást, a használt szolgáltatásokhoz. Lásd: [Cognitive Services díjszabása](https://azure.microsoft.com/pricing/details/cognitive-services/) további információt.

### <a name="single-service-subscription"></a>Előfizetés – olyan egyetlen szolgáltatást

Egyetlen szolgáltatás, például a Computer Vision, vagy a Speech Services-előfizetés. Előfizetés – olyan egyetlen szolgáltatást ennek az erőforrásnak korlátozódik.

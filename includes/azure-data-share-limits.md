| **Erőforrás** | **Korlát** |
| --- | --- |
| Az Azure-előfizetések esetében az adatmegosztási erőforrások maximális száma |50 |
| Az elküldett megosztások maximális száma az adatmegosztási erőforrásokban |100 |
| Az adatmegosztási erőforrásokban fogadott megosztások maximális száma |100 |
| Meghívások maximális száma egy elküldött megosztásban |100 |
| Megosztási előfizetések maximális száma eljuttatott megosztáson |100 |
| Adatkészletek maximális száma egy megosztáson |100 |
| Pillanatkép-ütemtervek maximális száma egy megosztáson |1 |

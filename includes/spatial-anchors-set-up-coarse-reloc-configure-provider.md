## <a name="configure-the-sensor-fingerprint-provider"></a>Az érzékelő ujjlenyomat-szolgáltatójának konfigurálása

Először hozzon létre és konfiguráljon egy érzékelő ujjlenyomat-szolgáltatót. Ez gondoskodik a platform-specifikus érzékelők beolvasásáról az eszközön, és azokat a Felhőbeli horgony-munkamenet által felhasznált közös képviseletre küldi el.

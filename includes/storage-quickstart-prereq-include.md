Az Azure Storage eléréséhez Azure-előfizetésre lesz szüksége. Ha még nem rendelkezik előfizetéssel, hozzon létre egy [ingyenes fiókot](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) a Kezdés előtt.

Az Azure Storage-hoz való összes hozzáférés egy Storage-fiókon keresztül történik. Ehhez a rövid útmutatóhoz hozzon létre egy Storage-fiókot a [Azure Portal](https://portal.azure.com/), Azure PowerShell vagy az Azure CLI használatával. A fiók létrehozásával kapcsolatos útmutatóért lásd a [tárfiók létrehozását](../articles/storage/common/storage-account-create.md) ismertető szakaszt.

Hozzáférhet az Azure Storage, Azure-előfizetésre lesz szüksége. Ha még nincs előfizetése, hozzon létre egy [ingyenes fiókot](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) megkezdése előtt.

Minden, az Azure Storage-hozzáférés tárfiókon keresztül történik. Ha a tárfiók már nem rendelkezik, könnyedén használatával is létrehozhat egy a [az Azure portal](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM). Storage-fiók az Azure PowerShell vagy az Azure CLI használatával is létrehozhat. A fiók létrehozásával kapcsolatos útmutatóért lásd a [tárfiók létrehozását](../articles/storage/common/storage-quickstart-create-account.md) ismertető szakaszt.

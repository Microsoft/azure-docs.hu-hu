Az Azure Storage eléréséhez szüksége lesz egy Azure-előfizetésre. Ha még nem rendelkezik előfizetéssel, hozzon létre egy [ingyenes fiókot,](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) mielőtt elkezdené.

Az Azure Storage-hoz való hozzáférés egy tárfiókon keresztül történik. Ehhez a rövid útmutatóhoz hozzon létre egy tárfiókot az [Azure Portalon](https://portal.azure.com/), az Azure PowerShell vagy az Azure CLI használatával. A tárfiók létrehozásához a [Tárfiók létrehozása című](../articles/storage/common/storage-account-create.md)témakörben talál segítséget.

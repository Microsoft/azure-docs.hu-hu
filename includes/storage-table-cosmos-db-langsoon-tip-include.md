> [!TIP]
> A cikkben szereplő tartalom az Azure Table Storage szolgáltatásra vonatkozik. Mostantól elérhető azonban egy olyan prémium táblatároló-ajánlat (Azure Cosmos DB Table API), amely teljesítményoptimalizált táblákat, globális elosztást és automatikus másodlagos indexeket kínál. Ha további információra van szüksége, vagy szeretné kipróbálni a prémium ajánlatot, tekintse meg az [Azure Cosmos DB Table API](https://aka.ms/premiumtables)-t ismertető cikket. A cikk programozási nyelve még nem támogatott a prémium szintű ajánlatban, de a jövőben hozzá lesz adva.

> [!TIP]
> Ez a cikk tartalma az Azure Table storage vonatkozik. Mostantól elérhető azonban egy olyan prémium táblatároló-ajánlat (Azure Cosmos DB Table API), amely teljesítményoptimalizált táblákat, globális elosztást és automatikus másodlagos indexeket kínál. Ha további információra van szüksége, vagy szeretné kipróbálni a prémium ajánlatot, tekintse meg az [Azure Cosmos DB Table API](https://aka.ms/premiumtables)-t ismertető cikket. Ez a cikk programozási nyelv még nem támogatja a prémium szintű szolgáltatása, de a jövőben fog bővülni.

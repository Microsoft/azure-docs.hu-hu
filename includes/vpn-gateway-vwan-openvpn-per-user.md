### <a name="open-the-mfa-page"></a><a name="mfa"></a>Az MFA-lap megnyitása

1. Jelentkezzen be az Azure portálra.
2. Keresse meg az **Azure Active Directory -> Minden felhasználó .**
3. A **többtényezős hitelesítési** lap megnyitásához válassza a többtényezős hitelesítéslehetőséget.

   ![Bejelentkezés](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Felhasználók kijelölése

1. A **többtényezős hitelesítés** i. lapon válassza ki azt a felhasználó(ka)t, akinek engedélyezni szeretné az MFA-t.
2. Válassza az **Engedélyezés** lehetőséget.

   ![Válassza ezt:](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)

Az Azure Cognitive Services a-ra előfizetett Azure-erőforrások képviselik. Hozzon létre egy erőforrást az anomália-detektorhoz a [Azure Portal](../articles/cognitive-services/cognitive-services-apis-create-account.md) vagy az [Azure CLI](../articles/cognitive-services/cognitive-services-apis-create-account-cli.md) használatával a helyi gépen. További lehetőségek:

* A [próbaverziós kulcs](https://azure.microsoft.com/try/cognitive-services/#decision) ingyenes hét napig érvényes. A regisztráció után elérhető lesz az [Azure webhelyén](https://azure.microsoft.com/try/cognitive-services/my-apis/).  
* Az erőforrás megtekintése a [Azure Portal](https://portal.azure.com/)

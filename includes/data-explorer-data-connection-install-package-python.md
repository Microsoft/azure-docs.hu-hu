## <a name="install-python-package"></a>Python-csomag telepítése

A Python-csomag azure Data Explorer (Kusto) telepítéséhez nyisson meg egy parancssort, amely a Python az útjába. Futtassa az alábbi parancsot:

```python
pip install azure-common
pip install azure-mgmt-kusto
```

Az űrlap adatokat kér Önnek, a vállalatnak és a felhasználói forgatókönyvnek, amelyhez a tárolót fogja használni. Miután elküldte az űrlapot, az Azure Cognitive Services csapata ellenőrzi, hogy megfelelnek-e a Private Container registryhez való hozzáférés feltételeinek.

> [!IMPORTANT]
> Az űrlapon egy Microsoft-fiókkal (MSA) vagy Azure Active Directory (Azure AD-fiókkal) társított e-mail-címet kell használnia. 

Ha jóváhagyja a kérelmét, egy e-mailt fog kapni, amely leírja, hogyan kérheti le a hitelesítő adatokat, és hogyan érheti el a privát tároló beállításjegyzékét.

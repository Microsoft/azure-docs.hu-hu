> [!IMPORTANT]
> * Azure Security Center jelenleg nem tudja végrehajtani a [képbiztonsági rések vizsgálatát](../articles/security-center/defender-for-container-registries-introduction.md?bc=%2fazure%2fcontainer-registry%2fbreadcrumb%2ftoc.json&toc=%2fazure%2fcontainer-registry%2ftoc.json) egy olyan beállításjegyzékben, amely korlátozza a privát végpontokhoz, a kiválasztott alhálózatokhoz vagy IP-címekhez való hozzáférést. 
> * Az Azure-szolgáltatások, például az Azure DevOps Services, a Web Apps és a Azure Container Instances példányai nem férnek hozzá a hálózatra korlátozott tárolók beállításjegyzékéhez. 
> * Bizonyos egyéb Azure-szolgáltatási példányok biztonságosan hozzáférhetnek a hálózatra korlátozott tárolók beállításjegyzékéhez. További információ: [a megbízható szolgáltatások engedélyezése a hálózatra korlátozott tárolók beállításjegyzékének biztonságos eléréséhez](../articles/container-registry/allow-access-trusted-services.md).

> A IoT Hub számos eszköz platformjának és nyelvének (például C, Java, JavaScript és Python) támogatásával rendelkezik az [Azure IoT Device SDK](https://github.com/azure/azure-iot-sdks)-k használatával. Az eszköznek az oktatóanyag kódjához való csatlakoztatásához, illetve általában az Azure IoT Hub-hoz való használatáról az [Azure IoT PYTHON SDK](https://github.com/Azure/azure-iot-sdk-python)-ban talál útmutatást.

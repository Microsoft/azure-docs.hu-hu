### <a name="azure-active-directory-authentication"></a>Azure Active Directory-hitelesítés

Az Azure térbeli horgonyok lehetővé teszik az alkalmazások számára a felhasználói Azure AD-(Active Directory-) tokenek hitelesítését is. Használhatja például az Azure AD-jogkivonatokat az Azure térbeli Horgonyokkal való integrációhoz. Ha egy vállalat megtartja az Azure AD-ban lévő felhasználókat, megadhat egy felhasználói Azure AD-jogkivonatot az Azure térbeli horgonyok SDK-ban. Ez lehetővé teszi, hogy közvetlenül az Azure térbeli horgonyok szolgáltatással hitelesítse az adott Azure AD-bérlő részét képező fiókot.

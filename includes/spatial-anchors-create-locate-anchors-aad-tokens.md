### <a name="azure-active-directory-authentication"></a>Azure Active Directory-hitelesítés

Az Azure Spatial Anchors lehetővé teszi az alkalmazások hitelesítését a felhasználói Azure AD (Active Directory) jogkivonatokkal. Például használhatja az Azure AD-jogkivonatok integrálása az Azure Spatial Anchors. Ha egy enterprise-i felhasználókat tart fenn az Azure AD-ben, az Azure Spatial Anchors SDK-ban megadhat egy felhasználói Azure AD-jogkivonatot. Ezzel lehetővé teszi, hogy közvetlenül az Azure Spatial Anchors szolgáltatás egy fiók, amely ugyanannak az Azure AD-bérlőnek a része.

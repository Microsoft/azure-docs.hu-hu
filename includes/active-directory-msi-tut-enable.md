1. [Bejelentkezés Azure Portal](https://portal.azure.com)

2. [Virtuális gép létrehozása rendszer által hozzárendelt identitással](../articles/active-directory/managed-identities-azure-resources/qs-configure-portal-windows-vm.md#system-assigned-managed-identity)

## <a name="create-a-repository"></a>Adattár létrehozása

Ez a cikk egy GitHub-sablon tárházát használja, amely megkönnyíti az első lépéseket. A sablon az Azure statikus Web Apps használatával történő üzembe helyezéshez használt alapszintű alkalmazást tartalmaz.

# <a name="no-framework"></a>[Nincs keretrendszer](#tab/vanilla-javascript)

1. Új tárház létrehozásához navigáljon a következő helyre:
    1. [https://github.com/staticwebdev/vanilla-basic/generate](https://github.com/login?return_to=/staticwebdev/vanilla-basic/generate)
1. Nevezze el az adattárat **az első-statikus-Web-App**

> [!NOTE]
> Az Azure statikus Web Apps egy webalkalmazás létrehozásához legalább egy HTML-fájl szükséges. Az ebben a lépésben létrehozott tárház egyetlen _index.html_ fájlt tartalmaz.

**Q**: Miért a hely a meglévő átjáró módosítás? <br/>
**A**: 2017. május 3. előtt létrehozott átjáró-erőforrásokat az Azure AD-bérlő az eredeti helyükre lett áthelyezve, a munkahelyi vagy iskolai fiók által létrehozott hozzájuk. Azonban ezeket a módosításokat nem lehet hatással jelenleg futó logikai alkalmazások, amelyek továbbra is a szokásos módon működik. Az átjáró májusban általánosan elérhető, az átjáró-erőforrások helyeként logic app helyek eltérhet.

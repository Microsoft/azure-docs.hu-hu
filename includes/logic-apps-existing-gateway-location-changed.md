**Q**: A hely a meglévő átjáró megváltoztak <br/>
**A**: 2017. május 3. az eredeti helyükre, az Azure AD-bérlő lett áthelyezve a a munkahelyi vagy iskolai fiókkal, amely előtt létre az átjáró-erőforrások létre hozzájuk. Azonban ezeket a módosításokat nem lehet hatással jelenleg futó logikai alkalmazások, amelyek továbbra is a szokásos módon működik. Az átjáró májusban általánosan elérhető, az átjáró-erőforrások helyeként logic app helyek eltérhet.

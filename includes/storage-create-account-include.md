## <a name="create-an-azure-storage-account"></a>Azure-tárfiók létrehozása

Az első Azure Storage-fiók létrehozásának legegyszerűbb módja a [Azure Portal](https://portal.azure.com)használata. További tudnivalókért lásd: [Create a storage account](../articles/storage/common/storage-account-create.md) (Tárfiók létrehozása).

Ezenkívül az [Azure PowerShell](/powershell/module/az.storage/), az [Azure CLI](../articles/storage/blobs/storage-quickstart-blobs-cli.md), illetve a [.NET-keretrendszerhez készült Azure Storage erőforrás-szolgáltató](https://azure.microsoft.com/resources/samples/storage-dotnet-resource-provider-getting-started/) használatával is létrehozhat egy Azure-tárfiókot.

Ha jelenleg nem szeretne Storage-fiókot létrehozni az Azure-ban, akkor a Azurite-tároló emulátorával is futtathatja és tesztelheti a kódot egy helyi környezetben. További információ: [a Azurite Emulator használata a helyi Azure Storage-fejlesztéshez](../articles/storage/common/storage-use-azurite.md).

## <a name="create-an-azure-storage-account"></a>Azure Storage-fiók létrehozása

Legegyszerűbben az [Azure Portal](https://portal.azure.com) segítségével hozhatja létre első Azure Storage-fiókját. További tudnivalókért lásd: [Create a storage account](../articles/storage/common/storage-account-create.md) (Tárfiók létrehozása).

Ezenkívül az [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md), az [Azure CLI](../articles/storage/common/storage-azure-cli.md), illetve a [.NET-keretrendszerhez készült Azure Storage erőforrás-szolgáltató](https://azure.microsoft.com/resources/samples/storage-dotnet-resource-provider-getting-started/) használatával is létrehozhat egy Azure-tárfiókot.

Ha jelenleg nem szeretne Storage-fiókot létrehozni az Azure-ban, az Azure Storage Emulator használatával futtathatja és tesztelheti a programkódot egy helyi környezetben. További információkért lásd: [Use the Azure Storage Emulator for Development and Testing](../articles/storage/common/storage-use-emulator.md) (Fejlesztés és tesztelés az Azure Storage Emulatorral).

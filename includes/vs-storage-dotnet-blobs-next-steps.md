Most, hogy megismerte az Azure Blob Storage alapjait, az alábbi hivatkozásokat követve megismerheti az összetettebb tárolási feladatokat.

* Tekintse meg a [.net-hez készült Azure Storage ügyféloldali kódtár](/dotnet/api/overview/azure/storage) blob Service dokumentációját, amely részletesen ismerteti az elérhető API-kat.
* Annak megismeréséhez, hogy miként egyszerűsíthető az Azure Storage-hoz való együttműködéshez írt kód, tekintse meg [a mi az a Azure WEBJOBS SDK](https://github.com/Azure/azure-webjobs-sdk/wiki)
* Az Azure-ban való adattárolás további lehetőségeiről tekintse meg a többi szolgáltatás-útmutatót.
  * Az Azure Storage-táblák használatához lásd: [Az Azure Table Storage használatának első lépései a .NET használatával](../articles/cosmos-db/tutorial-develop-table-dotnet.md).
  * Az Azure Storage-várólistákkal való együttműködéshez az [Azure Queue Storage használatának első lépései a .NET használatával](../articles/storage/queues/storage-dotnet-how-to-use-queues.md).
  * A kapcsolati információk tárolásához lásd: [Kapcsolódás a SQL Databasehoz .net (C#) használatával](../articles/azure-sql/database/connect-query-dotnet-visual-studio.md).

> [!TIP]
> 
> Az Azure Blob storage-erőforrások az Azure Storage Explorer használatával kezelheti. 
> [Az Azure Storage Explorer](https://azure.microsoft.com/features/storage-explorer/) van egy ingyenes, önálló alkalmazás, amely lehetővé teszi, hogy a Microsofttól [Azure Blob storage-erőforrások kezelése](../articles/vs-azure-tools-storage-explorer-blobs.md). Azure Storage Explorer használatával, is vizuálisan létrehozása, olvasása, frissítése, és törölhet blobtárolókat és blobokat, valamint blobok hozzáférésének kezelése a tárolókhoz és blobokhoz.

A virtuális gép (VM) automatikusan hozzárendel egy magánhálózati IP-címet a megadott tartományhoz azon alhálózat alapján, amelyben a virtuális gép telepítve van. A virtuális gép a virtuális gép törlése előtt megőrzi a címeket. Az Azure dinamikusan rendeli hozzá a következő elérhető magánhálózati IP-címet az alhálózatról, amelyben létrehoz egy virtuális gépet. Ha szeretne egy adott IP-címet a virtuális géphez hozzárendelt alhálózatból, rendeljen hozzá egy statikus IP-címet.

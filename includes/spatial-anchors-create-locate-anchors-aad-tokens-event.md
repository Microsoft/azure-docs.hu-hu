A hozzáférési tokenekhez hasonlóan, ha nincs beállítva Azure AD-jogkivonat, a TokenRequired eseményt kell kezelnie, vagy a delegált protokollon végre kell hajtania a tokenRequired metódust.

Az eseményt szinkron módon kezelheti az Event argumentumok tulajdonságának beállításával.

Az Azure Data Lake Analytics megkönnyíti az elosztott infrastruktúra és az összetett kód kezelésének összetett feladatát. Dinamikusan szolgál erőforrásokat, és használhatja az exabájtnyi adat elemzéséhez. Amikor a feladat befejeződik, automatikusan leállítja az erőforrásokat. Csak a felhasznált feldolgozási teljesítményért kell fizetnie. A tárolt adatok méretének vagy a felhasznált számítási mennyiségnek a növelésével vagy csökkentésével nem kell újraírnia a kódot. Az előfizetés alapértelmezett korlátainak növeléséhez forduljon az ügyfélszolgálathoz.

| **Erőforrás** | **Korlát** | **Megjegyzések** |
| --- | --- | --- |
| Egyidejű feladatok maximális száma |20 | |
| Az elemzési egységek (AUs) maximális száma fiókonként |250 | Legfeljebb 250 AUs kombinációját használhatja 20 feladat között. A korlát növeléséhez forduljon a Microsoft támogatási szolgálatához. |
| A feladat elküldéséhez a parancsfájl maximális mérete | 3 MB | |
| A Data Lake Analytics-fiókok maximális száma régiónként előfizetésenként | 5 | A korlát növeléséhez forduljon a Microsoft támogatási szolgálatához. |

## <a name="install-the-azure-functions-extension"></a>Az Azure Functions-bővítmény telepítése

A Azure Functions bővítmény használatával létrehozhat és tesztelheti a függvényeket, és üzembe helyezheti őket az Azure-ban.

1. A Visual Studio Code-ban nyissa meg a **bővítmények** elemet, keresse meg az **Azure functions**kifejezést, vagy [válassza ezt a hivatkozást a Visual Studio Code-ban](vscode:extension/ms-azuretools.vscode-azurefunctions)

1. A Visual Studio Code-bővítmény telepítéséhez válassza a **telepítés** lehetőséget:

    ![Az Azure Functions bővítményének telepítése](./media/functions-install-vs-code-extension/vscode-install-extension.png)

1. A telepítés után válassza ki az Azure ikont a tevékenység sávján. Ekkor megjelenik egy Azure Functions terület az oldalsávon.

    ![Az Azure Functions terület az oldalsávon](./media/functions-install-vs-code-extension/azure-functions-window-vscode.png)

## <a name="install-the-azure-functions-extension"></a>Az Azure Functions bővítmény telepítése

Az Azure Functions bővítmény segítségével létrehozhat és tesztelhet függvényeket, és üzembe helyezheti őket az Azure-ban.

1. A Visual Studio-kódban nyissa meg a **Bővítmények et,** és keressen **azure-függvényeket,** vagy [válassza ezt a hivatkozást a Visual Studio-kód ban.](vscode:extension/ms-azuretools.vscode-azurefunctions)

1. A Visual Studio-kód bővítményének telepítéséhez válassza a **Telepítés** lehetőséget:

    ![Az Azure Functions bővítményének telepítése](./media/functions-install-vs-code-extension/vscode-install-extension.png)

1. A telepítés után válassza ki az Azure ikont a tevékenységsávon. Ekkor megjelenik egy Azure Functions terület az oldalsávon.

    ![Az Azure Functions terület az oldalsávon](./media/functions-install-vs-code-extension/azure-functions-window-vscode.png)

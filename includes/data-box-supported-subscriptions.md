Mielőtt hozzákezd, győződjön meg az alábbiakról:

* A Microsoft Azure Storage-fiókja rendelkezik hozzáférési hitelesítő adatokkal, például a Storage-fiók nevét és a hozzáférési kulcsot.

* A Data Box szolgáltatáshoz használt előfizetés a következő típusok egyike:
  * Microsoft nagyvállalati szerződés (EA). További információk az [EA-előfizetésekről](https://azure.microsoft.com/pricing/enterprise-agreement/).
  * Felhőszolgáltató (CSP). További információk az [Azure CSP programjáról](https://docs.microsoft.com/azure/cloud-solution-provider/overview/azure-csp-overview).
  * Microsoft Azure szponzorálás. További információk az [Azure szponzorálási programjáról](https://azure.microsoft.com/offers/ms-azr-0036p/).
  * Microsoft Partner Network (MPN). További tudnivalók a [Microsoft Partner Networkről](https://partner.microsoft.com/commercial#).

* Eszközrendelések létrehozásához tulajdonosi vagy közreműködői jogosultsággal kell rendelkeznie az előfizetésen.

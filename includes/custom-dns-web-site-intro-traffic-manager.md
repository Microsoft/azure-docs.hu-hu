A Microsoft Azure Traffic Manager forgalom terheléselosztása betöltése az Azure websites használatakor a webhely majd használatával lehet elérni a  **\*. trafficmanager.net** Azure által hozzárendelt tartománynév. Egy egyéni tartománynevet, például www.contoso.com, a webhelyet, annak érdekében, hogy a felhasználók számára egy további felismerhető tartománynevet is társíthat.

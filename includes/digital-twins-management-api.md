Az alábbi példákban `YOUR_MANAGEMENT_API_URL` URI-ját a digitális Twins API-k hivatkozik:

```plaintext
https://YOUR_INSTANCE_NAME.YOUR_LOCATION.azuresmartspaces.net/management/api/v1.0
```

| Name (Név) | Csere erre |
| --- | --- |
| YOUR_INSTANCE_NAME | Az Azure digitális Twins-példány nevét |
| YOUR_LOCATION | A régió, a példány üzemelteti |

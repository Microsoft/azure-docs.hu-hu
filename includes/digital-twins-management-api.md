Az alábbi példákban `yourManagementApiUrl` URI-ját a digitális Twins API-k hivatkozik:

```plaintext
https://yourInstanceName.yourLocation.azuresmartspaces.net/management/api/v1.0/
```

| Egyedi név | Csere erre |
| --- | --- |
| Példánynév | Az Azure digitális Twins-példány nevét |
| yourLocation | Melyik kiszolgáló régióban lévő üzemeltetett a példány |

A klasszikus üzembe helyezési modell használata során nem használhatja az Azure Cloud Shellt. Ehelyett telepítenie kell az Azure Service Management (SM) PowerShell-parancsmagok legújabb verzióját a számítógépre. Ezek a parancsmagok eltérnek az AzureRM vagy az Az parancsmagok. Az SM-parancsmagok telepítéséről a [Szolgáltatáskezelési parancsmagok telepítése](/powershell/azure/servicemanagement/install-azure-ps)című témakörben látható. Az Azure PowerShellről általában az [Azure PowerShell dokumentációjában](/powershell/azure/)olvashat bővebben.

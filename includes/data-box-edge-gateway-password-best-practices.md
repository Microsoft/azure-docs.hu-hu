Tartsa szem előtt az alábbi gyakorlati tanácsokat:

- Javasoljuk, hogy az összes jelszót biztonságos helyen tárolja, hogy ne kelljen visszaállítania a jelszót, ha elfelejti. A felügyeleti szolgáltatás nem tudja beolvasni a meglévő jelszavakat. Csak az Azure Portalon keresztül állíthatja alaphelyzetbe őket. Ha alaphelyzetbe állít egy jelszót, az alaphelyzetbe állítás előtt mindenképpen értesítse az összes felhasználót.
- Az eszköz Windows PowerShell-felületét http-n keresztül távolról is elérheti. Biztonsági tanácsként a HTTP protokollt csak megbízható hálózatokon kell használni.
- Győződjön meg arról, hogy az eszköz jelszavak erős és jól védett. Kövesse a [jelszóval kapcsolatos gyakorlati tanácsokat.](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management)

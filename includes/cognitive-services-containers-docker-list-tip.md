> [!TIP]
> A [docker-rendszerképek](https://docs.docker.com/engine/reference/commandline/images/) paranccsal felsorolhatja a letöltött tárolórendszerképeket. A következő parancs például felsorolja az egyes letöltött tárolóképek azonosítóját, tárházát és címkéjét, táblázatként formázva:
>
>  ```
>  docker images --format "table {{.ID}}\t{{.Repository}}\t{{.Tag}}"
>
>  IMAGE ID         REPOSITORY                TAG
>  <image-id>       <repository-path/name>    <tag-name>
>  ```

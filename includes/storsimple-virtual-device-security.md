A StorSimple virtuális eszköz használatakor tartsa szem előtt az alábbi biztonsági szempontokat:

* A virtuális eszköz a Microsoft Azure-előfizetésen keresztül védett. Ez azt jelenti, hogy ha a virtuális eszköz használata, és az Azure-előfizetés sérül, a virtuális eszközön tárolt adatok is érzékeny.
* Az Azure StorSimple-ben tárolt adatok titkosításához használt tanúsítvány nyilvános kulcsa biztonságosan elérhetővé válik az Azure klasszikus portálon, és a személyes kulcs megmarad a StorSimple-eszközzel. A StorSimple virtuális eszközön a nyilvános és a személyes kulcsok az Azure-ban tárolódnak.
* A virtuális eszköz a Microsoft Azure adatközpontban található.

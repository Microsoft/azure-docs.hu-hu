A StorSimple virtuális eszköz használata esetén tartsa szem előtt a következő biztonsági szempontokat:

* A virtuális eszközt a Microsoft Azure-előfizetése biztosítja. Ez azt jelenti, hogy ha a virtuális eszközt használja, és az Azure-előfizetése biztonságban van, a virtuális eszközön tárolt adatai is fogékonyak.
* Az Azure StorSimple tárolt adattitkosításhoz használt tanúsítvány nyilvános kulcsa biztonságosan elérhető a klasszikus Azure portálon, és a titkos kulcs megmarad a StorSimple eszközzel. A StorSimple virtuális eszközön a nyilvános és a titkos kulcs is az Azure-ban tárolódik.
* A virtuális eszközt a Microsoft Azure adatközpont üzemelteti.

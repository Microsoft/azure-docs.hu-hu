## <a name="create-an-azure-resource"></a>Azure-erőforrás létrehozása

A Bing Spell Check API használatának megkezdéséhez hozza létre a következő Azure-erőforrások egyikét:

[Erőforrás Bing Spell Check](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSpellCheck-v7)
   * A Azure Portalon keresztül érhető el, amíg nem törli az erőforrást.
   * Az ingyenes díjszabási csomaggal próbálja ki a szolgáltatást, és később frissítsen egy fizetős szintre az éles környezetben.
   * A Bing Spell Check API a [Bing Search v7-erőforrás](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSearch-v7)egyes szintjein is elérhető.
    
[Több szolgáltatásból álló erőforrás](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * A Azure Portalon keresztül érhető el, amíg nem törli az erőforrást.  
   * Használja ugyanazt a kulcsot és végpontot az alkalmazásaihoz, több Cognitive Services között.

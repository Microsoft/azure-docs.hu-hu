## <a name="prepare-your-environment"></a>A környezet előkészítése

1. Az Azure CLI-parancsok futtatásához hajtsa végre az Azure CLI [helyi telepítését](/cli/azure/install-azure-cli) , vagy indítsa el a [Azure Cloud shellt](/azure/cloud-shell/quickstart) a bash-környezet használatával.

1. Ha helyi telepítést használ, jelentkezzen be az Azure CLI használatával az az [login](/cli/azure/reference-index#az-login) paranccsal.  A terminálon megjelenő utasítások alapján végezze el a hitelesítési folyamatot.  További bejelentkezési lehetőségekért lásd: [Bejelentkezés az Azure CLI-vel](/cli/azure/authenticate-azure-cli) .

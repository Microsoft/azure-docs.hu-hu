Ha nem rögzítette a gyorsítótárat az irányítópultra, az [Azure Portal](https://portal.azure.com) **Minden szolgáltatás** részén találhatja meg.

![Az Azure az Azure Cache a Redis – Tallózás panel](media/redis-cache-browse/redis-cache-browse.png)

A gyorsítótárak megtekintéséhez kattintson **minden szolgáltatás** és keressen rá a **for Rediss Azure Cache**. 

Válassza ki a kívánt gyorsítótárat a beállítások megtekintéséhez és konfigurálásához.

![Az Azure az Azure Cache a Redis gyorsítótárak listájának tallózása](media/redis-cache-browse/redis-caches.png)

Megtekintheti és konfigurálhatja a gyorsítótárat a **Azure Cache redis** panelen.

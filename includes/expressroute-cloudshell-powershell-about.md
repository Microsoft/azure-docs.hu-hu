Az Azure Cloud Shell használatával az Azure PowerShell vagy a CLI helyi telepítése helyett a legtöbb PowerShell-parancs- és CLI-parancs futtatásához használhatja. Az Azure Cloud Shell egy ingyenes interaktív rendszerhéj, amely előtelepített és a fiókjával való használatra van konfigurálva.
Ha a cikkben található kódot az Azure Cloud Shellen szeretné futtatni, nyisson meg egy Cloud Shell-munkamenetet, a kódblokk **Másolás** gombjával másolja a kódot, és illessze be a Cloud Shell munkamenetbe a __Ctrl+Shift+V__ alkalmazással Windows és Linux rendszeren, vagy __cmd+shift+v__ macOS rendszeren. A beillesztett szöveg végrehajtása nem történik meg automatikusan, a kód futtatásához nyomja le az **Enter** billentyűt.

Néhány módszer a Cloud Shell indításához:

|-----------------------------------------------|---|
| Kattintson a **Try It** (Kipróbálás) lehetőségre a kódblokk jobb felső sarkában. | ![A Cloud Shell ebben a cikkben](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-powershell-try-it.png) |
| Nyissa meg a Cloud Shellt a böngészőben. | [![https://shell.azure.com/powershell](./media/vpn-gateway-cloud-shell-powershell/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Kattintson a **Cloud Shell** gombra az Azure Portal jobb felső részén található menüben. | [![A Cloud Shell a portálon](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-menu.png)](https://portal.azure.com) |

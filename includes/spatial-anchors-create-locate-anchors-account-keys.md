## <a name="set-up-authentication"></a>Hitelesítés beállítása

A szolgáltatás eléréséhez meg kell adnia egy fiókkulcsot, hozzáférési jogkivonatot vagy az Azure Active Directory hitelesítési token. Erről a [Hitelesítés koncepciólapon](/azure/spatial-anchors/concepts/authentication)is olvashat bővebben.

### <a name="account-keys"></a>Fiókkulcsok

Fiókkulcsok egy hitelesítő adatok, amely lehetővé teszi az alkalmazás hitelesítését az Azure Spatial Anchors szolgáltatással. A Fiókkulcsok célja, hogy segítsen a gyors kezdésben. Különösen az alkalmazás azure spatial anchors-i integrációjának fejlesztési fázisában. Így használhatja a fiókkulcsokat, ha a fejlesztés során beágyazza őket az ügyfélalkalmazásokba. A fejlesztésen túli előrehaladással erősen ajánlott egy éles szintű, hozzáférési jogkivonatok által támogatott hitelesítési mechanizmusra vagy az Azure Active Directory felhasználói hitelesítésére való áttérést. A fiókkulcs fejlesztési beszerezéséhez keresse fel az Azure Spatial Anchors-fiókot, és keresse meg a "Kulcsok" lapot.

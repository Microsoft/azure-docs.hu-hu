Most, hogy megtanulta az Azure-várólista-tárolás alapjait, kövesse ezeket a hivatkozásokat az összetettebb tárolási feladatok megismeréséhez.

* Tekintse meg a várólistaszolgáltatás referenciadokumentációját az [Azure Storage Ügyféltár .NET](https://go.microsoft.com/fwlink/?LinkID=390731) referencia az elérhető API-k részletes adatait.
* További információ a várólista-tárolás használatáról az Azure Queue storage használatával – Első lépések az [Azure Queue storage használatával – ismertetése](../articles/storage/queues/storage-dotnet-how-to-use-queues.md)
* Ha meg szeretné tudni, hogyan egyszerűsítheti az Azure Storage-mal való együttműködésre írt kódot, olvassa el [a Mi az Azure WebJobs SDK című](https://github.com/Azure/azure-webjobs-sdk/wiki) könyvét.
* Az Azure-ban való adattárolás további lehetőségeiről tekintse meg a többi szolgáltatás-útmutatót.
  * Az Azure Storage-táblák használatáról az [Azure Table storage használatával – első lépések a .NET használatával](../articles/cosmos-db/table-storage-how-to-use-dotnet.md).
  * Az Azure Storage Blobs használatához [kezdje el az Azure Blob storage használatát a .NET használatával.](../articles/storage/blobs/storage-dotnet-how-to-use-blobs.md)
  * A relációs adatok tárolásához olvassa el a [Csatlakozás sql adatbázishoz a .NET (C#) használatával](../articles/sql-database/sql-database-develop-dotnet-simple.md)című témakört.

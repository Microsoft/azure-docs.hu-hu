Most, hogy megismerte az Azure üzenetsor-tárolás alapjait, kövesse az alábbi hivatkozásokat az összetettebb tárolási feladatok megismeréséhez.

* Tekintse meg a [.net-hez készült Azure Storage ügyféloldali kódtár](https://go.microsoft.com/fwlink/?LinkID=390731) Queue szolgáltatás dokumentációját, amely részletesen ismerteti az elérhető API-kat.
* További információ a üzenetsor-tároló használatáról: az [Azure üzenetsor-tárolás használatának első lépései a .NET használatával](../articles/storage/queues/storage-dotnet-how-to-use-queues.md)
* Annak megismeréséhez, hogy miként egyszerűsíthető az Azure Storage-hoz való együttműködéshez írt kód, tekintse meg [a mi az a Azure WEBJOBS SDK](https://github.com/Azure/azure-webjobs-sdk/wiki)
* Az Azure-ban való adattárolás további lehetőségeiről tekintse meg a többi szolgáltatás-útmutatót.
  * Az Azure Storage-táblák használatához lásd: [Az Azure Table Storage használatának első lépései a .NET használatával](../articles/cosmos-db/table-storage-how-to-use-dotnet.md).
  * Az Azure Storage-Blobok használatához az [Azure Blob Storage használatának első lépései a .NET használatával](../articles/storage/blobs/storage-dotnet-how-to-use-blobs.md).
  * A kapcsolati információk tárolásához lásd: [Kapcsolódás a SQL Databasehoz .net (C#) használatával](../articles/sql-database/sql-database-develop-dotnet-simple.md).

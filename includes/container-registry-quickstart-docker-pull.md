## <a name="run-image-from-registry"></a>Rendszerkép futtatása a beállításjegyzékből

Most lehívhatja és futtathatja a `hello-world:v1` tároló-beállításjegyzékből a [Docker futtatásával][docker-run]:

```
docker run <acrLoginServer>/hello-world:v1  
```

Példa a kimenetre: 

```
Unable to find image 'mycontainerregistry007.azurecr.io/hello-world:v1' locally
v1: Pulling from hello-world
Digest: sha256:662dd8e65ef7ccf13f417962c2f77567d3b132f12c95909de6c85ac3c326a345
Status: Downloaded newer image for mycontainerregistry007.azurecr.io/hello-world:v1

Hello from Docker!
This message shows that your installation appears to be working correctly.

[...]
```

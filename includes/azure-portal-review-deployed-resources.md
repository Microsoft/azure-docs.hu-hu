Ellenőrizze, hogy látható-e a virtuális gép adatait a Azure Portal irányítópultján.

1. Az Azure Portalon válassza az **Irányítópult** lehetőséget.

    :::image type="content" source="media/azure-portal-review-deployed-resources/navigate-to-dashboards.png" alt-text="Navigálás az irányítópulton Azure Portal":::

1. Az irányítópult lapon válassza az **egyszerű VM-irányítópult** lehetőséget.

    :::image type="content" source="media/azure-portal-review-deployed-resources/select-simple-vm-dashboard.png" alt-text="Navigáljon az egyszerű VM-irányítópultra":::

1. Tekintse át az irányítópultot, amelyet a Azure Resource Manager sablon (ARM-sablon) hozott létre. Láthatja, hogy a tartalom némelyike statikus, de vannak olyan diagramok is, amelyek az elején létrehozott virtuális gép teljesítményét mutatják.

    :::image type="content" source="media/azure-portal-review-deployed-resources/review-simple-vm-dashboard.png" alt-text="Egyszerű VM-irányítópult áttekintése":::

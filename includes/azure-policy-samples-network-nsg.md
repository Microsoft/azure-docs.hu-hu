|  |  |
|---------|---------|
| [NSG X minden hálózati adapteren](../articles/governance/policy/samples/nsg-on-nic.md) | Előírja, hogy minden virtuális hálózati adapterhez egy adott hálózati biztonsági csoportot kell használni. A használandó hálózati biztonsági csoport azonosítóját Ön határozza meg. |

- ARCore XR beépülő modul (Android-eszközök támogatásához): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- ARKit XR beépülő modul (iOS-eszközök támogatásához): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

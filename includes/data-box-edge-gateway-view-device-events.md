Az eszköz eseményeinek megtekintéséhez hajtsa végre a következő lépéseket a Azure Portalban. 

1. A Azure Portal nyissa meg a Azure Stack Edge/Data Box Gateway erőforrást, majd lépjen a **figyelés > eszköz eseményeire**.
2. Válasszon ki egy eseményt, és tekintse meg a riasztás részleteit. Tegye meg a megfelelő lépéseket a riasztási feltétel feloldásához.

    ![Esemény kiválasztása és a részletek megtekintése](media/data-box-edge-gateway-view-device-events/view-device-events.png)

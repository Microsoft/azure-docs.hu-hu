## <a name="prerequisites"></a>Előfeltételek

Az oktatóanyag elvégzéséhez győződjön meg arról, hogy rendelkezik a következőkkel:

* Olvassa el az [Azure Spatial Anchors áttekintését.](../articles/spatial-anchors/overview.md)
* Az 5 [perces rövid összefoglalók egyike befejeződött.](../articles/spatial-anchors/index.yml)
* Alapvető ismeretek a C# és unity.
* Alapvető ismeretek az <a href="https://developers.google.com/ar/discover/" target="_blank">ARCore-on,</a> ha Androidot vagy <a href="https://developer.apple.com/arkit/" target="_blank">ARKit-ot</a> szeretne használni, ha iOS-t szeretne használni.
* Olyan Windows-számítógép, amelyen a <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017-es</a> vagy újabb verzió telepítve van a **ASP.NET és a webfejlesztési** munkaterheléssel.
* A [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download).
* Egy vagy több eszköz (iOS vagy Android), amelyen telepíteni és futtatni egy alkalmazást.
  * Ha Androidot használ, a következőkre van szükség:
    * <a href="https://developer.android.com/studio/" target="_blank">Az Android Studio 3.3</a> vagy újabb, <a href="https://unity3d.com/get-unity/download" target="_blank">a Unity 2019.1 vagy a 2019.2</a>és a Windows rendszerhez <a href="https://git-scm.com/download/win" target="_blank">készült Git</a> telepítve van a Windows rendszerű számítógépen.
    * <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">Fejlesztőáltal engedélyezett</a> és <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore-kompatibilis Android-eszköz.</a>
  * Ha iOS-t használ, a következőkre van szükség:
    * Olyan macOS-számítógép, amelyen az <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10</a> vagy újabb kód, a <a href="https://cocoapods.org" target="_blank">CocoaPods</a>és a <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 vagy a 2019.2</a> telepítve van.
    * Fejlesztői <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">arkit-kompatibilis iOS-eszköz.</a>
    * Git keresztül telepített Homebrew. Írja be a következő parancsot egyetlen `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` sorba a terminálon: Ezután futtassa a parancsot. `brew install git`

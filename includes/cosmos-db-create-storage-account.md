Az Azure storage-fiók létrehozásának legegyszerűbb módja az [Azure Portal](https://portal.azure.com)használata. További tudnivalókért lásd: [Create a storage account](../articles/storage/common/storage-account-create.md) (Tárfiók létrehozása).

Ezenkívül az [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md) vagy az [Azure CLI](../articles/storage/common/storage-azure-cli.md) használatával is létrehozhat egy Azure-tárfiókot.

Ha most nem kíván tárfiókot létrehozni, az Azure Storage Emulatorral helyi környezetben futtathatja és tesztelheti a kódját. További információ: [Az Azure storage emulátor használata fejlesztési és tesztelési célokra.](../articles/storage/common/storage-use-emulator.md)

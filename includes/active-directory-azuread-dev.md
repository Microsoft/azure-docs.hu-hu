> [!IMPORTANT]
> A [Microsoft Identity platform (v 2.0)](/azure/active-directory/develop/v2-overview) a Azure Active Directory (Azure ad) fejlesztői platform (v 1.0) fejlődése. Lehetővé teszi a fejlesztők számára, hogy olyan alkalmazásokat építsenek, amelyek az összes Microsoft-identitás használatával biztonságosan jelentkeznek be, és jogkivonattal hívják meg az olyan Microsoft API-kat, mint a Microsoft Graph, vagy olyan API-kat, amelyeket fejlesztők készítettek.
> Ez a tartalom a régebbi, az Azure AD 1.0-s verziójának végpontja. Javasoljuk, hogy a 2.0-s végpontot használja új projektekhez. További információért olvassa el a [Microsoft Identity platform (v 2.0) frissítésének okát?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) valamint a [Microsoft Identity platform korlátozásai](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).

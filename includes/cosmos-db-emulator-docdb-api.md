Másik lehetőségként használhatja az [Azure Cosmos DB Emulatort](https://docs.microsoft.com/azure/cosmos-db/local-emulator) a következő URI-val: `https://localhost:8081`. A kulcs az emulátorral való használatához lásd: [kérelmek hitelesítése](../articles/cosmos-db/local-emulator.md#authenticating-requests).

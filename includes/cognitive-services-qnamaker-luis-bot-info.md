## <a name="more-information-about-bots"></a>További információ a botokról

A szolgáltatás robotokkal való használatáról a következő forrásokkal kapcsolatos további tudnivalókért:

|Erőforrás|Cél|
|--|--|
|[Azure Bot szolgáltatás](https://dev.botframework.com/)|Az Azure Bot szolgáltatás egy teljes felhőben üzemeltetett webszolgáltatást biztosít egy robot-végpontdal. A szolgáltatások [bot keretrendszert](https://github.com/Microsoft/botframework)használnak , amely több nyelven is elérhető.|
|[Robot keretrendszer](https://github.com/Microsoft/botframework)|A Microsoft Bot Framework egy átfogó platform a nagyvállalati szintű társalgási AI-élmények létrehozásához.|
|[Bot keretemor](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|A Bot Framework Emulator egy platformfüggetlen asztali alkalmazás, amely lehetővé teszi a bot fejlesztők számára, hogy teszteljék és hibakeresés botok segítségével épített Bot Framework SDK. A Bot Framework emulátor segítségével tesztelheti a számítógépen helyileg futó robotokat, vagy csatlakozhat a távolról futó robotokhoz.|
|[Bot eszközök](https://github.com/Microsoft/botbuilder-tools)|A Bot Framework eszközök platformfüggetlen parancssori eszközök gyűjteménye, amelyek a teljes körű robotfejlesztési munkafolyamatot lefedik. |
|[Bot építő minták](https://github.com/Microsoft/BotBuilder-Samples)|A teljes fejlesztésű robotmintákat úgy tervezték, hogy bemutassák azokat a forgatókönyveket, amelyeket nagy robotok létrehozásához kell megvalósítania.|

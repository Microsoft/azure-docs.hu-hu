## <a name="more-information-about-bots"></a>További információ a robotok

További információt az ezzel a szolgáltatással a robotok kezdődik a következőket:

|Resource|Cél|
|--|--|
|[Az Azure Bot service](https://dev.botframework.com/)|Az Azure Bot service robot-végponttal teljes felhőben futtatott webalkalmazás szolgáltatást biztosít. A szolgáltatások által használt [Bot framework](https://github.com/Microsoft/botframework), amely számos nyelven érhető el.|
|[Bot Framework](https://github.com/Microsoft/botframework)|A Microsoft Bot Framework az átfogó platform, amellyel nagyvállalati szintű természetes nyelvi mesterséges Intelligencia élményt.|
|[Bot Framework emulátor](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|A Bot Framework-emulátor egy platformfüggetlen asztali alkalmazás, amely lehetővé teszi, hogy a robot a fejlesztők tesztelése és hibakeresése a Bot keretrendszer SDK-val készített robotokat. A Bot Framework-emulátor is használhat, a gépén helyileg futó robotokat teszteléséhez vagy a távolról futtatott robotokat csatlakozni.|
|[A robot eszközök](https://github.com/Microsoft/botbuilder-tools)|A Bot Framework eszközök, melyek terjed ki a teljes körű robot a fejlesztési munkafolyamat olyan többplatformos parancssori eszközöket. |
|[A robot builder minták](https://github.com/Microsoft/BotBuilder-Samples)|Bot teljes fejlesztésű mintákat mutatja be a forgatókönyv végrehajtásához a nagyszerű robotokat készíthet kell tervezték.|

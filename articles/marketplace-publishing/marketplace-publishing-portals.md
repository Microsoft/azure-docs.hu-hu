# <a name="portals-you-will-need"></a>Szüksége lesz portálok

Az ajánlat közzétételi folyamat megkezdése előtt teheti meg a különböző portálok, amelyeket létre kell bevezetni. Az alábbi, a röviden összefoglalja a portálok – fejlesztői központ, Azure közzétételi portálján és az Azure portal – a ahhoz, hogy Ön működjön velük kapcsolatban.                                                                            

## <a name="developer-center"></a>Fejlesztői központ
[http://dev.windows.com/registration?accountprogram=azure](http://dev.windows.com/registration?accountprogram=azure)

### <a name="description"></a>Leírás
A Microsoft Developer Center-fiók létrehozása az egyszeri feladat. Győződjön meg arról, hogy a vállalat még nem rendelkezik a fejlesztői központban lévő fiókjához előtt hozzon létre egyet. A folyamat során a gyűjtünk a fiók létrehozása, és adózási adatait, valamint vállalati címet.

> [!NOTE]
> Ha csak az ingyenes ajánlatokhoz közzéteszi (vagy bring-your-saját licenc kínál), adózási és banki információt nem szükséges.
> 
> 

### <a name="identityaccount-used"></a>Használt identitás/fiók
Ideális esetben ez az érték-e a terjesztési lista vagy egy biztonsági csoportot (például @ azurepublishing*partnercompany*.com). A terjesztési listát vagy biztonsági csoport **kell** Microsoft-fiókkal regisztrálható.

> [!TIP]
> Azt javasoljuk, egy terjesztési listát vagy biztonsági csoportot eltávolítja a függő bármely személy, mert bár egy egyéni fiók is használható.
> 
> 

## <a name="publishing-portal"></a>Közzétételi portálon
[https://publish.windowsazure.com](https://publish.windowsazure.com)

### <a name="description"></a>Leírás

Ezt az értéket a portálon, hogy az ajánlat dolgozhatnak, és tegye közzé (marketing, díjszabás, közzététel, ha van ilyen hitelesítő stb.) a rendszer.

### <a name="identityaccount-used"></a>Használt identitás/fiók
A fenti terjesztési listát vagy biztonsági csoportnak kell használható először jelentkezzen be a közzétételi portálon. Később más felhasználók, társadminisztrátorként felveheti. Ez a lista hogyan, a fejlesztői központ regisztrációs adatok leképezése beolvasása.

## <a name="azure-portal"></a>Azure Portal
[Azure Portal](https://portal.azure.com)

### <a name="description"></a>Leírás
Ez az érték a portálon, ahol megtekintheti az előkészített és közzétett ajánlatok az Azure Marketplace (virtuális gép, megoldássablonok és Azure Resource Manager-alapú fejlesztői szolgáltatások esetében alkalmazható).

### <a name="identityaccount-used"></a>Használt identitás/fiók
A közzétételi portálon egy ajánlatot még átmeneti, amíg egy előfizetés-azonosító kell lennie a szerepel az engedélyezési listán. A manuálisan előkészített ajánlat teszteléséhez használható ezen a portálon való bejelentkezéshez kell ugyanabban az előfizetésben (van egy felhasználónevet és a hozzá társított jelszót).

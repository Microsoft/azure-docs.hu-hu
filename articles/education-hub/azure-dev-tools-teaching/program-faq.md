# <a name="frequently-asked-questions-about-azure-dev-tools-for-teaching"></a>Gyakori kérdések az Azure fejlesztői eszközökről az oktatási célokra

Ez a cikk az Azure diákoknak, az Azure diákoknak kezdőknek és az Azure fejlesztői eszközöknek az oktatási célokra való gyakori kérdéseit tárgyalja.

## <a name="azure-for-students"></a>Azure for Students

### <a name="what-happens-after-i-use-my-100-credit-or-im-at-the-end-of-12-months"></a>Mi történik a $100-kreditek használata, vagy a 12 hónapos időszak lejártakor?

A rendelkezésre álló kreditek kimerítése vagy a 12 hónapos időszak lejárta után az Azure-előfizetése le lesz tiltva. Az [Azure](https://account.azure.com/)-t továbbra is használhatja utólagos elszámolású előfizetésre az Azure-fiók portálon. Az Azure diákoknak szóló részletes használati feltételeinek megtekintéséhez tekintse meg az [ajánlat feltételeit](https://azure.microsoft.com/offers/ms-azr-0170p/).

### <a name="can-i-get-azure-for-students-again-next-year"></a>Az Azure diákoknak a jövő évben is beszerezhetők?

Köszönjük érdeklődését az Azure diákoknak! Örömmel vesszük, hogy Ön is használta. Jelenleg nem jelentjük be a megújítást. Látogasson vissza ide a frissítésekhez.

### <a name="what-are-subscriptions-and-how-do-they-relate-to-azure-for-students"></a>Mik azok az előfizetések, és hogyan kapcsolódnak az Azure diákoknak?

Az előfizetések hozzáférést biztosítanak az Azure-szolgáltatásokhoz. Az Azure diákoknak 12 hónapig $100 kreditet biztosítunk. Több mint 25 ingyenes szolgáltatáshoz biztosít hozzáférést, beleértve a számítási, hálózati, tárolási és adatbázis-szolgáltatásokat is. Az ezen időszak során felmerülő költségek levonhatók a kreditből. Ha az Azure-szolgáltatásokat a 12 hónapos időszak végén vagy a $100-kredit kimerítése után kívánja használni, attól függően, hogy az utólagos elszámolású előfizetésre van-e szükség.

### <a name="what-happens-with-my-azure-services-if-i-dont-upgrade"></a>Mi történik az Azure-szolgáltatásokkal, ha nem frissítek?

Ha úgy dönt, hogy nem frissíti a 12 hónapos időszakot, vagy az $100-kreditet kimerítette, attól függően, hogy melyik terméket telepítette, a rendszer leszereli, és nem fogja tudni elérni őket. Az ingyenes előfizetés végén 90 nap áll rendelkezésre az utólagos elszámolású előfizetésre való frissítéshez.

### <a name="how-do-i-know-how-much-of-the-100-credit-i-have-left"></a>Hogyan tudni, hogy mekkora az $100-os kredit?

A fennmaradó kreditet az [Azure szponzorálási portálon](https://www.microsoftazuresponsorships.com/)tekintheti meg.

### <a name="how-do-i-download-the-software-developer-tools"></a>Hogyan letölti a szoftverfejlesztői eszközöket?

Az Azure for Students-előfizetés hozzáférést biztosít bizonyos szoftverfejlesztői eszközökhöz. A szoftverfejlesztői eszközök eléréséhez és letöltéséhez aktuális, aktív Azure diákoknak szóló előfizetésre van szükség. Nyissa meg a [Education-központt](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software) , és töltse le a szoftverfejlesztői eszközöket az Azure diákoknak szóló előfizetés használatával.

### <a name="what-is-microsoft-learn"></a>Mi a Microsoft Learn?

A [Microsoft Learn](https://docs.microsoft.com/learn/) egy ingyenes, online tanulási platform, amely lehetővé teszi az Azure-technológiák saját tempójában való megismerését. A képzési útvonalak kombinálják a modulokat, amelyek lehetővé teszik az alapok használatának megkezdését, majd a valós kihívásokkal foglalkozó speciális metódusokra való áttérést.

### <a name="can-azure-for-students-be-used-for-production-or-only-for-development"></a>Használható-e az Azure diákoknak éles környezetben, vagy csak fejlesztési célokra?

Az Azure diákoknak biztosít hozzáférést az összes olyan Azure-termékhez, amely kifejezetten az oktatási és oktatási, nem kereskedelmi célú kutatás támogatására, illetve az ezekre a célokra szánt szoftveralkalmazások tervezésére, fejlesztésére, tesztelésére és bemutatására irányul.

### <a name="can-i-apply-any-of-my-100-credit-toward-azure-marketplace-offers"></a>Használhatom a $100 kreditet az Azure Marketplace-ajánlatokhoz?

Nem. Az Azure Marketplace-ajánlatokra nem alkalmazhat kreditet. Az Azure piactér számos partnere azonban ingyenes próbaverziókat vagy ingyenes csomagokat kínál a megoldásaihoz.

## <a name="azure-for-students-starter"></a>Azure for Students Starter

### <a name="what-is-azure-for-students-starter"></a>Mi az Azure diákoknak kezdő?

Az Azure diákoknak kezdőnek a felhőben való fejlesztéshez szükséges Azure-termékekkel kezdheti meg. Önnek nem kell fizetnie. Ez a kedvezmény a következő szolgáltatások ingyenes szintjéhez nyújt hozzáférést:

- Azure App Service
- Azure Functions
- Azure Notification Hubs
- Azure Database for MySQL
- Application Insights
- Azure DevOps Services (korábban Visual Studio Team Services)

Az Azure diákoknak kezdőnek az ellenőrzött tanulók díjmentesen, kötelezettségvállalás vagy időkorlát nélkül elérhetők. Tekintse meg az [Azure diákoknak kezdő ajánlatát](https://azure.microsoft.com/offers/ms-azr-0144p/) a részletes használati feltételekről.

### <a name="who-is-eligible-for-azure-for-students-starter"></a>Ki jogosult az Azure diákoknak kezdőknek?

Az Azure diákoknak kezdő számára csak olyan tanulók számára érhető el, akik megfelelnek az alábbi követelményeknek:

- Meg kell erősítenie, hogy 13 éves vagy idősebb, ha a Egyesült Államokban lakik.
- Meg kell erősítenie, hogy 16 éves vagy idősebb, ha a Egyesült Államok eltérő országban lakik.
- A tanulmányi állapotát a szervezet e-mail-címén keresztül kell ellenőrizni. A Shibboleth akkor is használhatja, ha a szervezete támogatja.

Ez az ajánlat nem használható tömeges nyílt online tanfolyamon (MOOC), illetve a nonprofit szervezetektől származó egyéb szakmai képzéseken.

Ez az ajánlat csak egy Azure diákoknak kezdő előfizetésre korlátozódik jogosult ügyfélen. Nem ruházhatók át, és nem kombinálható más ajánlatokkal, hacsak a Microsoft másként nem engedélyezte.

### <a name="will-i-have-to-pay-something-at-any-point-in-time"></a>Kell-e fizetnem valamilyen időpontban?

Az Azure diákoknak kezdő ajánlathoz nem szükséges hitelkártya. Az ajánlat hozzáférést biztosít az Azure-szolgáltatások korlátozott készletéhez. Azonban bármikor frissítheti az utólagos elszámolású előfizetést, hogy hozzáférjen az összes Azure-szolgáltatáshoz.

### <a name="how-do-i-download-the-software-developer-tools"></a>Hogyan letölti a szoftverfejlesztői eszközöket?

Az Azure for Students-előfizetés hozzáférést biztosít bizonyos szoftverfejlesztői eszközökhöz, amelyek ingyenesen letölthetők.

A szoftverfejlesztői eszközök eléréséhez aktuális, aktív Azure diákoknak szóló előfizetésre van szükség.

Ezt a szoftvert a [Education-központ](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software)is letöltheti.

### <a name="what-is-microsoft-learn"></a>Mi a Microsoft Learn?

A [Microsoft Learn](https://docs.microsoft.com/learn/) egy ingyenes, online tanulási platform, amely lehetővé teszi az Azure-technológiák saját tempójában való megismerését. A képzési útvonalak kombinálják a modulokat, amelyek lehetővé teszik az alapok használatának megkezdését, majd a valós kihívásokkal foglalkozó speciális metódusokra való áttérést.

## <a name="azure-academic-grant"></a>Azure Academic-támogatás

### <a name="how-do-i-start-using-my-azure-course-credits"></a>Hogyan megkezdeni az Azure-beli tanfolyam-kreditek használatát?

Az Azure-tanfolyamhoz tartozó kreditek új Microsoft Azure Academic Grant-előfizetés létrehozásával érhetők el. Kattintson az **aktiválás** gombra a szponzorálási jóváhagyási e-mailben.

Lehetősége van arra is, hogy egy meglévő előfizetést a Microsoft Azure szponzorálás ajánlatra Konvertáljon a kreditek eléréséhez. Az előfizetés konvertálásának részleteit a következő kérdésben találhatja meg.

### <a name="can-i-associate-my-course-credits-with-an-existing-subscription"></a>Hozzárendelhet egy meglévő előfizetéshez tartozó tanfolyami kreditet?

A tanfolyami krediteket egy meglévő előfizetéssel társíthatja az ajánlatra jogosult fiókhoz. Vegye fel a kapcsolatot az [Azure támogatási szolgálatával](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview) a tanfolyami kreditek hozzárendeléséhez.

### <a name="why-do-i-see-a-0-balance-in-the-azure-portal"></a>Miért látok $0 egyenleget a Azure Portal?

Ha az előfizetéshez tartozó részleteket a Azure Portalban tekinti meg, akkor a $0,00-as verzió jelenik meg. Az ajánlat 100%-os kedvezményt helyez el az összes szolgáltatásból. Ezért a portálon megtekintheti a havi használati időszak során felszámított díjat, amelynek $0,00-ig kell lennie.

Az egyenleg és a szponzorálási információk megtekintéséhez nyissa meg az [Azure szponzorálásait](https://www.microsoftazuresponsorships.com/balance) , és jelentkezzen be a fiókjába.

### <a name="can-i-apply-my-course-credits-to-an-existing-enterprise-agreement-ea"></a>Alkalmazhatom a tanfolyami krediteket egy meglévő Nagyvállalati Szerződésre (EA)?

A tanfolyami kreditek nem társíthatók olyan előfizetésekhez, amelyek egy Nagyvállalati Szerződés (EA) alá tartozó fiókban vannak.

A tanfolyami kreditek alkalmazásához létre kell hoznia egy új fiókot, amely a nagyvállalati szerződésen kívül esik, és ezt követően is jogosult.

A szponzorálási időszak lejárta után visszaállíthatja az előfizetést az EA-be.

> [!WARNING]
> Ha a szponzort a szponzorálás vége előtt rendeli hozzá az EA-hoz, a rendszer minden szponzorálási alapot megszakít. További információért tekintse meg az [Azure Sponsorship ajánlat](https://azure.microsoft.com/offers/ms-azr-0143p/) használati feltételeit.

### <a name="can-i-apply-my-course-credits-to-my-csp-subscription"></a>Alkalmazhatom a tanfolyami krediteket a saját CSP-előfizetésre?

Nem, a tanfolyami kreditek ajánlata nem társítható a felhőalapú megoldás-szolgáltatói (CSP) előfizetésekhez.

### <a name="can-i-move-my-course-credits-to-another-account"></a>Áthelyezhetem a tanfolyami krediteket egy másik fiókba?

Igen, a tanfolyami kreditek egy másik fiókra is áthelyezhetők. Ehhez forduljon az [Azure ügyfélszolgálatához](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview).

### <a name="are-azure-marketplace-applications-covered-by-course-credits"></a>Az Azure Marketplace-alkalmazások a tanfolyami kreditek hatálya alá esnek?

Nem, a harmadik féltől származó alkalmazásokat nem fedezi a tanfolyami kreditek. A rendszer a fiókhoz tartozó bankkártyára terheli. További információk:

- Tekintse meg a Marketplace-szolgáltatásokat az [Azure piactéren](https://azuremarketplace.microsoft.com/marketplace/).
- Tekintse meg az [Azure Sponsorship ajánlat](https://azure.microsoft.com/offers/ms-azr-0143p/) használati feltételeit.

### <a name="i-have-a-previous-balance-due-can-i-pay-it-off-with-my-course-credits"></a>Van egy korábbi esedékes egyenlegem, fizethetem a tanfolyami kreditekkel?

Az Azure-tanfolyami kreditek csak a Azure Sponsorship ajánlat aktiválásának idejére vonatkoznak. Az ajánlat kezdési dátuma előtt minden költségért Ön felelős.

### <a name="how-do-i-know-if-my-subscription-is-on-the-azure-course-credit-offer"></a>Hogyan tudni, hogy az előfizetésem az Azure Course Credit ajánlaton van-e?

Ha megtekint egy adott előfizetést a Azure Portal előfizetések paneljén, akkor az **ajánlat nevét** fogja látni a tulajdonságok egyike. Az ajánlat neve **Azure Sponsorship** , ha csatlakozik a tanfolyami kreditekhez. Ha **Azure Sponsorship** nem jelenik meg, forduljon az ügyfélszolgálathoz, és kérje át.

### <a name="how-do-i-download-software"></a>Hogyan szoftver letöltése?

Az Microsoft Azure fejlesztői eszközök az oktatási előfizetéshez hozzáférést biztosítanak bizonyos szoftverfejlesztői eszközökhöz. Ezek az eszközök ingyenesen letölthetők.

A szoftvert a [Education-központ szoftverek szakaszban](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software)töltheti le.

### <a name="how-do-we-distribute-software-to-our-students"></a>Hogyan terjeszthetők a szoftverek a tanulók számára?

Az előfizetőnek szóló Azure fejlesztői eszközökhöz az iskolája vagy intézménye hozzáférhet a Education-központ áruházhoz. A tanulók a [Education-központ áruházban](https://azureforeducation.microsoft.com/devtools)férhetnek hozzá a felhőalapú szolgáltatásaihoz, amely a [Azure Portal](https://portal.azure.com/).

A tanulók bejelentkeznek a Azure Portalba az iskolájuk (vagy az Azure dev Tools for Teaching) hitelesítő adataival. Ezután a tanulók megnyitják a Education-központ áruházat, és hozzáférnek a rendelkezésre álló szoftverek letöltéséhez.

### <a name="is-azure-dev-tools-for-teaching-available-internationally"></a>Elérhető-e az Azure fejlesztői eszközök a nemzetközi tanításhoz?

Igen, a több mint 140 országban/régióban érhető el, ahol az Azure kereskedelmi elérhetőséggel rendelkezik.

### <a name="which-languages-are-available-in-the-software-to-end-users"></a>Mely nyelvek érhetők el a szoftverben a végfelhasználók számára?

A Education-központ áruház a következő nyelveken érhető el: arab, Kínai (egyszerűsített), Kínai (hagyományos), dán, holland, angol, francia, német, héber, olasz, Japán, Koreai, portugál, Orosz, spanyol, svéd és török.

### <a name="if-our-students-download-software-through-the-azure-dev-tools-for-teaching-program-do-they-get-unlimited-use-of-the-software"></a>Ha a tanulók szoftvereket töltenek le az Azure dev Tools for Teaching program segítségével, korlátlanul használhatják a szoftvert?

Igen. A tanulók korlátlan számú szoftvert kapnak a tanulási és kutatási erőfeszítéseik elvégzéséhez.

### <a name="if-our-students-create-viable-apps-and-products-using-azure-dev-tools-for-teaching-software-can-they-sell-them-commercially"></a>Ha a diákok életképes alkalmazásokat és termékeket hoznak létre az Azure dev Tools for Teach Software segítségével, akkor kereskedelmi szempontból is értékesíthetik azokat?

Általában nem. A tanulók nem tudnak eladni alkalmazásokat és termékeket az Azure dev Tools for Teaching használatával. Az Azure dev Tools for Teaching és a Microsoft App Store csapatai közötti partnerségnek köszönhetően azonban a tanulók létrehozhatnak játékokat és alkalmazásokat a Windows áruházban való értékesítéshez.

### <a name="do-i-have-unlimited-use-of-the-software-through-the-azure-dev-tools-for-teaching-program"></a>Korlátlanul használhatom a szoftvert az Azure dev Tools for Teaching program használatával?

Igen. Ha egy oktatói tag regisztrálva van egy jóváhagyott tanfolyamon, jogosultak arra, hogy a nem kereskedelmi célú használatra az Azure fejlesztői eszközöket telepítsenek a saját számítógépére.

### <a name="how-do-i-access-my-visual-studio-enterprise-benefit"></a>Hogyan hozzáférni a Visual Studio Enterprise juttatáshoz?

Az előfizetéshez készült Azure fejlesztői eszközök rendszergazdájaként a Visual Studio Enterprise-előfizetését az [Azure dev Tools for Teaching Management portálon](https://azureforeducation.microsoft.com/account/Subscriptions)keresztül érheti el.

A jóváhagyást követően bejelentkezhet a [Visual Studio-portálra](https://my.visualstudio.com/) , és további előnyöket válthat ki.

### <a name="does-microsoft-azure-dev-tools-for-teaching-include-microsoft-office"></a>A Microsoft Azure fejlesztői eszközei közé tartozik a Microsoft Office?

Nem. A Microsoft Azure fejlesztői eszközeinek célja az, hogy részlegeket, oktatókat és tanulókat biztosítson a szoftverfejlesztés és a tesztelés konkrét kibővítéséhez szükséges eszközökkel. Ezért olyan technológiákat biztosítunk, mint a Windows Server, a Visual Studio .NET, a SQL Server és a Platform SDK.

### <a name="does-azure-dev-tools-for-teaching-include-azure-credit"></a>Az Azure fejlesztői eszközei közé tartozik az Azure Credit?

Nem, az oktatási előfizetéshez készült Microsoft Azure fejlesztői eszközök nem tartalmazzák az Azure-kreditet. De regisztrálhat az Azure diákoknak szóló ajánlatára, amely az Azure-szolgáltatások használatáért $100USD értékű Azure-kreditet biztosít. Látogasson [el a jövőbe az Azure diákoknak szóló felépítése!](https://aka.ms/student) további információért.

### <a name="do-students-need-an-office-365-or-active-directory-account-to-access-azure-dev-tools-for-teaching"></a>A tanulóknak Office 365-vagy Active Directory-fiókra van szükségük az Azure fejlesztői eszközök tanításhoz való hozzáféréséhez?

Nem. A diákoknak nincs szükségük Office 365-fiókra. Ha a tanulók hozzáférnek a Active Directory-fiókjához, ugyanazokat a hitelesítő adatokat használják a szoftverbe való bejelentkezéshez.

Ha a tanulók nem használják Active Directory, akkor létre kell hozniuk egy [Microsoft-fiókot](https://account.microsoft.com/account) (ha még nem rendelkezik ilyennel) ugyanazzal az e-mail-címmel, amelyet Ön biztosít.

### <a name="why-arent-my-sign-in-credentials-recognized-when-i-sign-in-to-azure-dev-tools-for-teaching"></a>Miért nem ismerhető fel a bejelentkezési hitelesítő adatok, amikor bejelentkeznek az Azure dev Tools for Teaching szolgáltatásba?

Győződjön meg arról, hogy az iskolai hitelesítő adataival való tanításhoz próbál bejelentkezni az Azure fejlesztői eszközeibe. Segíthet megnyitni egy privát böngészőablakot.

Ha továbbra sem tud bejelentkezni, forduljon az előfizetés rendszergazdájához. Az előfizetés-rendszergazda megkereséséhez [vegye fel velünk a kapcsolatot](https://aka.ms/adt4tsupport).

### <a name="how-do-i-find-my-subscriber-id"></a>Hogyan megkeresni az előfizetői azonosítót?

- **Amikor először regisztrálja a programot**: az előfizetői azonosító a kapott előfizetés-üdvözlő e-mailben szerepel.
- **Ha megújította az előfizetését**: az előfizetői azonosító az előfizetés-rendszergazda által fogadott megújítási e-mailben szerepel.

Az előfizetői azonosító a Visual Studio Subscription portálon is elérhető. Miután bejelentkezett, tekintse meg az **előfizetést** a **fiókom oldalon.**

Ha segítségre van szüksége az előfizetői azonosító megkereséséhez, [lépjen kapcsolatba velünk](https://azureforeducation.microsoft.com/institutions/Contact).

### <a name="are-we-automatically-enrolled-in-azure-dev-tools-for-teaching-if-we-receive-it-as-part-of-our-academic-volume-licensing-agreement"></a>A rendszer automatikusan regisztrálja az Azure fejlesztői eszközökön a tanítást, ha a tanulmányi mennyiségi licencszerződés részeként érkezik?

Nem, a Microsoft nem regisztrálja automatikusan, ha rendelkezik egy tanulmányi mennyiségi licencszerződéssel. A tanulmányi mennyiségi licencszerződés az alábbiakat tartalmazza:

- Beléptetés oktatási megoldásokra (EFS)
- Open Value előfizetési szerződés oktatási megoldásokhoz (OVS-ES)
- Kampusz szerződés
- Iskolai szerződés

Az Azure fejlesztői eszközökön regisztrálnia kell a megfelelő promóciós kódokat a tanulmányi mennyiségi licenchez kapott előfizetés üdvözlő e-mail-címének használatával.

Az előfizetést az érvényesség lejárta után is meg kell újítania. Nem újítja meg automatikusan.

Ha nem találja a promóciós kódját, [lépjen kapcsolatba velünk](https://azureforeducation.microsoft.com/institutions/Contact).

### <a name="how-and-when-do-we-renew-our-azure-dev-tools-for-teaching-subscription"></a>Hogyan és mikor újítjuk meg az Azure fejlesztői eszközöket oktatási előfizetésre?

60 nappal a tagság lejárta előtt az előfizetés megújításához meg kell kapnia az e-mail-emlékeztetőket. A megújítási e-mailben válassza a [megújítási hivatkozást](https://portal.azureforeducation.microsoft.com/).

Ha nem kapja meg ezeket az emlékeztető e-maileket, és aggódik, hogy előfizetése hamarosan lejár, [lépjen kapcsolatba velünk](https://aka.ms/adt4tsupport).

Másik lehetőségként nyissa meg az [Azure fejlesztői eszközök for Teaching-renew oldalt](https://portal.azureforeducation.microsoft.com/), és fejezze be a megújítási folyamatot.

Az előfizetés lejárati idejének vizsgálatához nyissa meg az [Azure dev Tools for Teaching Management Portalt](https://azureforeducation.microsoft.com/account/Subscriptions), és tekintse meg az **előfizetések**területen.

### <a name="what-if-i-need-more-help"></a>Mi a teendő, ha további segítségre van szükségem?

[Lépjen kapcsolatba velünk](https://azureforeducation.microsoft.com/institutions/Contact) az előfizetések támogatási oldalán, és keresse meg a régióját.

### <a name="where-is-the-azure-dev-tools-for-teaching-privacy-and-cookies-policy"></a>Hol található az Azure fejlesztői eszközök az adatvédelem és a cookie-k szabályzatának tanításához?

A jelen adatvédelmi nyilatkozat számos Microsoft-termékre vonatkozik, beleértve annak alkalmazásait, eszközeit, kiszolgálóit, szolgáltatásait, szoftvereit és webhelyeit. A termékre vonatkozó információkat is tartalmaz, és részletezi a szabályzatot a cookie-k használatához.

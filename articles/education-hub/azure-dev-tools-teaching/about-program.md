# <a name="what-is-azure-dev-tools-for-teaching"></a>Mi az az Azure Dev Tools for Teaching?

Az oktatási Microsoft Azure fejlesztői eszközök a Microsoft professzionális fejlesztői eszközeinek, szoftverének és szolgáltatásainak köszönhetően alacsony díjszabású előfizetésekkel rendelkeznek. A tanulók díjmentesen vehetik igénybe a fejlesztői eszközöket – az alkalmazások, a játékok és a webhelyek létrehozásához szükségesek, így üldözik álmaikat, meghozhatják a következő nagy áttörést a technológiában, vagy elkezdhetik a karrierjét.

Az előfizetéshez készült Azure dev Tools rendszergazdája az alábbiakat teszi lehetővé:

- A szoftvereket egy központi Online helyen kezelheti és érheti el, így nem kell webáruházat vagy belső webhelyet fenntartania.

- A tudományos felhasználók a szoftverekhez, az ingyenes képzésekhez, a tanulási anyagokhoz és az Azure Cloud platformhoz is hozzáférést biztosíthatnak ugyanazon az online portálon.

## <a name="program-details"></a>Program részletei

Az Azure Dev Tools for Teachinget a természettudományos képzésekhez terveztük. Minden olyan kurzus, amelynek a fő témája a természettudományok, a technológia, a mérnöki tudományok vagy a matematika, jogosult használni az Azure Dev Tools for Teachinget, amely az oktatóknak és a tanulóknak egyaránt segítséget jelent a hatékonyabb munkához. 

Az Microsoft Azure fejlesztői eszközök az oktatási előfizetéshez hozzáférést biztosítanak bizonyos szoftverfejlesztői eszközökhöz. Ezek az eszközök ingyenesen letölthetők. Ha egy oktatói tag regisztrálva van egy jóváhagyott tanfolyamon, Ön jogosult az Azure fejlesztői eszközök telepítésére a saját számítógépére nem kereskedelmi célú használatra.

Az Azure dev Tools for Teaching a több mint 140 országban/régióban érhető el, ahol az Azure kereskedelmileg elérhető. A Education-központ áruház a következő nyelveken érhető el: arab, Kínai (egyszerűsített), Kínai (hagyományos), dán, holland, angol, francia, német, héber, olasz, Japán, Koreai, portugál, Orosz, spanyol, svéd és török.

## <a name="student-benefits"></a>Tanulói előnyök

Minden szoftver egy olyan [Education-központon](https://azureforeducation.microsoft.com/devtools) keresztül érhető el, amely a tudományos felhasználók számára elérhető összes terméket és szolgáltatást felépíti a [Azure Portalba](https://portal.azure.com/). A tanulók a szoftverek eléréséhez egyszerűen bejelentkeznek a Azure Portalba ugyanazzal a hitelesítő adatokkal, amelyeket az Imagine-előfizetéséhez használnak.

A tanulóknak nem kell regisztrálniuk az Azure-előfizetésre, hogy hozzáférjenek a szoftverek előnyeihez. A Microsoft azonban a 100 USD értékű Azure-kreditet, valamint az ingyenes Azure-szolgáltatásokhoz való hozzáférést is lehetővé teszi a diákoknak: a tanulók az [Azure diákoknak szóló ajánlatra](azure-students-program.md) regisztrálhatnak, anélkül, hogy bankkártyát kellene igénybe venniük.

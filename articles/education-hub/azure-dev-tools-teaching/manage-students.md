# <a name="enrolling-students-in-azure-dev-tools-for-teaching"></a>Tanulók beléptetése az Azure fejlesztői eszközökbe oktatási célokra
Az Azure fejlesztői eszközök oktatási célokra való megvásárlása után az intézmény tartományában található összes diák automatikusan hozzáférhet az ajánlathoz. A vásárlási folyamat során a rendszer kérni fogja az előfizetés-rendszergazda e-mail-címének megadását. Az Egyetem által ellenőrzött tartomány feloldja a diákokat ugyanazzal az e-mail-tartománnyal, és lehetővé teszi számukra, hogy bejelentkezzenek, és beváltsák a szoftver letöltési előnyeit. Ha a tanulók még nem rendelkeznek Microsoft-fiókkal (MSA), vagy egy egyetemi Active Directory részét képezik, a rendszer kérni fogja, hogy hozzon létre egyet, amely megfelel az egyetemi e-mail-címének. Ha a tanulók a Education-központ elérésével kapcsolatos problémákba ütköznek, tekintse meg az alábbi hibaelhárítási útmutatót, vagy lépjen kapcsolatba velünk a következő címen: [https://aka.ms/studentsupport](https://aka.ms/studentsupport) .

## <a name="managing-access-for-students"></a>Tanulók hozzáférésének kezelése
Az Azure fejlesztői eszközökre való bejelentkezéshez használt tartomány alapján kezelheti a diákok hozzáférését. A [felügyeleti portál](https://azureforeducation.microsoft.com/en-us/account/Subscriptions) az Azure fejlesztői eszközök tanításához. Itt láthatja az összes előfizetését. Válassza ki azt az előfizetést, amelynek a hozzáférését szeretné kezelni.

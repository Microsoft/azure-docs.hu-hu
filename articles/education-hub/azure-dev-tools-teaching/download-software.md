# <a name="help-students-download-software-developer-tools-from-the-azure-education-hub"></a>Segítség a tanulók számára szoftverek fejlesztői eszközeinek letöltéséhez az Azure Education-központ

Az Azure dev Tools for Teaching program révén a tanulók az aktuális és aktív Azure for Students-előfizetés keretében ingyenesen tölthetik le a szoftveres fejlesztői eszközöket. Az Azure diákoknak biztosít hozzáférést az összes olyan Azure-termékhez, amely kifejezetten az oktatási és oktatási, nem kereskedelmi célú kutatás támogatására, illetve az ezekre a célokra szánt szoftveralkalmazások tervezésére, fejlesztésére, tesztelésére és bemutatására irányul. A tanulók az Azure Education-központból tölthetik le az eszközöket.

## <a name="instructions-for-students-how-to-download-software-developer-tools"></a>Diákoknak szóló útmutató: szoftverek fejlesztői eszközeinek letöltése

A következő eljárással töltheti le a szoftveres fejlesztői eszközöket a tanulók számára:

1. Nyisson meg egy böngészőt, és nyissa meg az [Azure Education-központ](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).
1. Jelentkezzen be az Azure diákoknak előfizetési hitelesítő adataival.
1. Válassza ki a Education-központ valamelyik szoftverfejlesztő eszközét a letöltés ablaktábla megnyitásához.
1. Olvassa el a szoftver adatait, és győződjön meg arról, hogy a szoftver kompatibilis a számítógép rendszerével.
1. Miután megerősítette, hogy a szoftver kompatibilis-e a rendszerrel, válassza a **Letöltés**lehetőséget.
1. A szoftver letöltéséhez és telepítéséhez kövesse a képernyőn megjelenő utasításokat.

   Az Azure diákoknak 12 hónapig $100 kreditet biztosítunk. Több mint 25 ingyenes szolgáltatáshoz biztosít hozzáférést, beleértve a számítási, hálózati, tárolási és adatbázis-szolgáltatásokat is. Az ezen időszak során felmerülő költségek levonhatók a kreditből.

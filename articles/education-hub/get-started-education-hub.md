# <a name="getting-started-with-azure-education-hub"></a>Első lépések az Azure Education-központ

A Education-központ első lépések oldal gyors hivatkozásokat biztosít a Education-központba való első beérkezéskor. Itt tájékozódhat arról, hogyan állíthatja be a tanfolyamot, megismerheti a különböző szolgáltatásokat Microsoft Learnon keresztül, vagy egyszerűen üzembe helyezheti első szolgáltatásait a gyors üzembe helyezési sablonok segítségével.

# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>32 bites operációs rendszerek támogatása az Azure-beli virtuális gépeken

A Microsoft Azure mostantól lehetővé teszi a felhasználók számára, hogy 32 bites Windows operációs rendszereiket átvigyék az Azure-ba. Mivel ezen operációs rendszerek némelyike már elérte az életfenntartó képességre vonatkozó szerződését, előfordulhat, hogy a Microsoft nem nyújt számukra további támogatást. A microsoft azure-alapú virtuális gépen (VM) futó Linux-alapú vagy Berkeley szoftverterjesztési (BSD) alapú operációs rendszerekhez nem érhető el támogatás.

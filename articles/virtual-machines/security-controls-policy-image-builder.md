# <a name="azure-policy-regulatory-compliance-controls-for-azure-image-builder"></a>Az Azure-rendszerkép-szerkesztő Azure Policy vonatkozó megfelelőségi szabályozása

A [Azure Policy szabályozási megfelelősége](../governance/policy/concepts/regulatory-compliance.md) biztosítja a Microsoft által létrehozott és felügyelt kezdeményezési definíciókat, amelyek a különböző megfelelőségi szabványokhoz kapcsolódó **megfelelőségi tartományokhoz** és **biztonsági ellenőrzésekhez** _készültek_. Ez a lap felsorolja az Azure-rendszerkép-készítő **megfelelőségi tartományait** és **biztonsági vezérlőit** . A **biztonsági vezérlők** beépített moduljait egyenként is hozzárendelheti ahhoz, hogy az Azure-erőforrások megfeleljenek az adott szabványnak.

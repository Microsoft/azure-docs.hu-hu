# <a name="workloads-on-azure-virtual-machines"></a>Számítási feladatok Azure-beli virtuális gépeken

Az Azure Virtual machines számos számítási feladatokat is támogatja. Minden számítási feladat az optimális kialakítás üzembe helyezéseket az Azure-beli virtuális gépek az arra vonatkozó útmutatás speciális is rendelkezik. Íme néhány speciális számítási feladatokhoz kínálhatunk útmutatást.


- [SAP](./sap/get-started.md) az Azure-ban az üzletmenet szempontjából kritikus fontosságú SAP számítási feladatok és a forgatókönyvek platformon futtatható egy skálázható, a szabályozásoknak megfelelően a nagyvállalatoknál bizonyított.
- [Oracle](./oracle/oracle-considerations.md) támogatja, Oracle DB 12.1 Standard és Enterprise kiadás virtuálisgép-lemezképek az Azure-ban futó Oracle Linux alapján. 
- [A nagyszámítógépes újratárolása](./mainframe-rehosting/overview.md) 



**Következő lépések**

Ismerje meg az Azure virtuális gépek számára az Azure virtuális gép gyors útmutatók a [Linux](../linux/quick-create-cli-nodejs.md) és [Windows](../windows/quick-create-powershell.md).

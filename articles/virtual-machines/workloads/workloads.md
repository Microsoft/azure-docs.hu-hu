# <a name="workloads-on-azure-virtual-machines"></a>Munkaterhelések az Azure Virtual Machines

Az Azure-beli virtuális gépek számos munkaterhelést támogatnak. Minden számítási feladat speciális útmutatást nyújt az Azure-beli virtuális gépeken üzemelő példányok optimális kialakításához. Íme néhány, a speciális számítási feladatokhoz használható útmutató.

- A [nagy teljesítményű számítástechnika](./hpc/overview.md) az Azure-ban CPU vagy GPU által optimalizált virtuális gépeket használ összetett, nagy számítási igényű problémák megoldásához.
- Az Azure-beli [SAP](./sap/get-started.md) az üzleti szempontból kritikus fontosságú SAP-munkaterheléseket és forgatókönyveket méretezhető, megfelelő és nagyvállalati működésre alkalmas platformon futtathatja.
- Az [Oracle](./oracle/oracle-considerations.md) támogatja a Oracle db 12,1 standard és Enterprise kiadásait az Azure-ban Oracle Linux-alapú virtuálisgép-rendszerképeken. 
- [Nagyszámítógépek újraüzemeltetése](./mainframe-rehosting/overview.md) 



**Következő lépések**

Ismerje meg az Azure-beli virtuális gépeket a [Linux](../linux/quick-create-cli.md) és a [Windows rendszerhez](../windows/quick-create-powershell.md)készült Azure VM rövid útmutatókkal.

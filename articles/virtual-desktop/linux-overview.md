# <a name="thin-client-support"></a>Vékony ügyfelek támogatása

A Windows rendszerű virtuális asztali erőforrásokat a saját partnerei által biztosított, a [webügyféllel](connect-web.md) vagy a következő támogatott ügyfelekkel érheti el a vékony ügyféleszközök számára. Számos partnerrel dolgozunk, hogy a támogatott Windowsos virtuális asztali ügyfeleket engedélyezzük más platformokon.

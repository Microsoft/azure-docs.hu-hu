# <a name="windows-virtual-desktop-partner-integrations"></a>Windows virtuális asztal partnerintegrációk

Ez a cikk a Windows virtuális asztal partnerek betűrendben sorolja fel.

## <a name="active-cypher"></a>Aktív titkosító csomagokkal kapcsolatban

Aktív titkosító csomagokkal kapcsolatban egy Windows virtuális asztal értéknövelt szolgáltatásait felhasználva szeretnék szolgáltató, amely programcsomagok informatikai az illetéktelen behatolásokat és az adatvesztés a vállalati védelme érdekében titkosítási technológiával. Aktív Cypher innovatív titkosítási technológia támogatja a teljes fájl védelmi stratégia Windows virtuális asztalok futtatását az Azure-ban.

Windows virtuális asztalok használata esetén a aktív Cypher kisegítő és felhasználói adatok mobilitási engedélyezése során nem szándékos adatveszteség megelőzése további képességeket biztosít. Az aktív Cypher meglévő csoport biztonsági szabályzatok és az Azure Information Protection engedélyeket is javítható, felhasználói fájlok titkosítása Windows virtuális asztalok használata során.

[Nyissa meg a partner webhelyére.](https://activecypher.com/)

## <a name="cloudjumper"></a>CloudJumper

CloudJumper programcsomagok szolgáltatást nyújtó vállalkozásoknak és a vállalati informatikai szoftverrel azok alkalmazásfüggőségeit üzembe helyezése és kezelése Windows virtuális asztali környezetek virtuális asztali Windows értéknövelt szolgáltatásait felhasználva szeretnék szolgáltató. CloudJumper szoftvert, az informatikai részleg felügyelete a Windows virtuális asztal központi telepítés minden rétegben. Számítási feladatok és alkalmazások automatikus, annak érdekében, hogy a felhasználók gyorsan férhessenek hozzá az Asztalukat bárhol, bármilyen eszközön.

CloudJumper a szoftvert, felhőalapú munkaterület Management Suite kiterjeszheti az Windows virtuális asztal egyszerűsítésével, üzembe helyezés és a folyamatban lévő felügyeleti feladatok az Azure-ban. Egy egyetlen panelen üvegből, az informatikai részleg üzembe helyezése, kezelése és optimalizálhatja a felhasználói munkaterületek infrastruktúráját. CloudJumper a egyszerű szkript elindítása motor integrálható az informatikai szolgáltatás platformok részt vevő virtuális asztali Windows-kiépítés feladatok automatizálására. Ezenkívül CloudJumper az API-kkal további bővíthetőség és integráció a vállalat más rendszerei, mint például a ServiceNow és a BMC készen áll.

[Nyissa meg a partner webhelyére.](https://cloudjumper.com/wvd/)

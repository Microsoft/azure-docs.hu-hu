# <a name="connect-with-the-web-client"></a>Kapcsolódás a webügyféllel

A webes ügyfél lehetővé teszi a Windows virtuális asztal erőforrásainak elérését egy webböngészőből a hosszadalmas telepítési folyamat nélkül.

>[!NOTE]
>A webes ügyfél jelenleg nem rendelkezik mobil operációs rendszer támogatással.

## <a name="supported-operating-systems-and-browsers"></a>Támogatott operációs rendszerek és böngészők

Bár minden HTML5-kompatibilis böngészőnek működnie kell, hivatalosan is támogatjuk a következő operációs rendszereket és böngészőket.

| Böngésző           | Támogatott operációs rendszer                     | Megjegyzések               |
|-------------------|----------------------------------|---------------------|
| Microsoft Edge    | Windows                          |                     |
| Internet Explorer | Windows                          |                     |
| Apple Safari      | macOS                            |                     |
| Mozilla Firefox   | Windows, macOS, Linux            | 55-ös vagy újabb verzió |
| Google Chrome     | Windows, macOS, Linux, Chrome OS |                     |

## <a name="access-remote-resources-feed"></a>Távoli erőforrás-csatorna elérése

A böngészőben keresse meg a [Windows Virtuális asztal webes ügyfelet,](https://rdweb.wvd.microsoft.com/webclient) és jelentkezzen be a felhasználói fiókjával.

>[!NOTE]
>Ha már bejelentkezett egy másik Azure Active Directory-fiókkal, mint amelyet a Windows virtuális asztalhoz kíván használni, jelentkezzen ki, vagy használjon privát böngészőablakot.

A bejelentkezés után most meg kell jelennie az erőforrások listájának. Az erőforrásokat úgy indíthatja el, hogy úgy választja ki őket, mintha egy normál alkalmazást szeretne a **Minden erőforrás** lapon.

## <a name="next-steps"></a>További lépések

Ha többet szeretne tudni a webes ügyfél használatáról, olvassa el az Első lépések a webes ügyféllel című [útmutatót.](https://go.microsoft.com/fwlink/?linkid=2120256)

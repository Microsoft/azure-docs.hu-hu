# <a name="discover-and-use-sample-notebooks-in-azure-notebooks-preview"></a>Mintajegyzetfüzetek felderítése és használata az Azure Notebookelőzetes verzióban

Az internet számos érdekes és hasznos Jupyter-jegyzetfüzetből áll, amelyeket könnyedén behozhat az Azure-jegyzetfüzetekbe. Ez a cikk a jelentős minták listáját tartalmazza.

[!INCLUDE [notebooks-status](../../includes/notebooks-status.md)]

Miután azonosította a feltárni kívánt jegyzetfüzetet, az alábbiakat érdemes megértenie:

- Ha egy jegyzetfüzet már az Azure Notebooks, használja a **Klónozás** gombot a projekt irányítópultján, hogy egy másolatot a projekt a saját fiókjában.
- Egyes jegyzetfüzetek futtatásához meghatározott csomagok futtatására van szükség. Bár a `!pip install` (Python) `install.packages` vagy (R) segítségével csomagokat telepíthet magából a jegyzetfüzetből, [konfigurálhatja az Azure Notebooks projektet a megfelelő környezettel.](configure-manage-azure-notebooks-projects.md)

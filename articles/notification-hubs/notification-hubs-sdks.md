# <a name="azure-notification-hubs-sdks"></a>Azure Notification Hubs SDKs

A Microsoft és harmadik felek közzététele SDK-k az Azure Notification hubs használatával. A Microsoft tárolóhelyekkel Android, iOS- és .NET cél platformonként mintaalkalmazások is tartalmazhatnak.

| SDK            | Közzétevő      |
| -------------- | -------------- |
| [.NET Server](https://github.com/Azure/azure-notificationhubs-dotnet) | Microsoft |
| [Android](https://github.com/Azure/azure-notificationhubs-android)  |Microsoft |
| [Apache Cordova](https://github.com/derek82511/cordova-azure-notification-hubs) | Külső |
| [iOS](https://github.com/Azure/azure-notificationhubs-ios) |Microsoft |
| [Java Server](https://github.com/Azure/azure-notificationhubs-java-backend) |Microsoft |
| [Nodejs](https://github.com/Azure/azure-sdk-for-node) | Microsoft |
| [React natív](https://github.com/CatalystCode/react-native-azurenotificationhub) | Külső |

> [!Note]
> Microsoft nem felelős a minőségét, hasznosságát vagy bármilyen külső megoldások támogathatóság.

Tudassa velünk, ha nincs semmilyen hiányzik.

# <a name="azure-notification-hubs-sdks"></a>Azure Notification Hubs SDK-k

A Microsoft és a harmadik felek SDK-kat tesznek közzé az Azure Notification Hubs. Az Android, iOS és .NET rendszerhez készült Microsoft-Tárházak az egyes célzott platformokra vonatkozó példákat is tartalmaznak.

| SDK            | Kiadó      |
| -------------- | -------------- |
| [.NET-kiszolgáló](https://github.com/Azure/azure-notificationhubs-dotnet) | Microsoft |
| [Android](https://github.com/Azure/azure-notificationhubs-android)  |Microsoft |
| [Apache Cordova](https://github.com/derek82511/cordova-azure-notification-hubs) | Harmadik féltől származó |
| [iOS](https://github.com/Azure/azure-notificationhubs-ios) |Microsoft |
| [Java-kiszolgáló](https://github.com/Azure/azure-notificationhubs-java-backend) |Microsoft |
| [NodeJS](https://github.com/Azure/azure-sdk-for-node) | Microsoft |
| [Natív reagálás](https://github.com/CatalystCode/react-native-azurenotificationhub) | Harmadik féltől származó |

> [!Note]
> A Microsoft nem felelős semmilyen harmadik féltől származó megoldás minőségéről, hasznosságáról vagy támogatásáról.

Kérjük, tudassa velünk, ha hiányzik a probléma.

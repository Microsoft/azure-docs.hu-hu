# <a name="azure-notification-hubs-sdks"></a>Az Azure Értesítési központok SDK-k

A Microsoft és a harmadik felek sdk-ket tesznek közzé az Azure Értesítési központokhoz. Az Android, iOS és .NET rendszerre készült Microsoft-adattárak az egyes célplatformokhoz is tartalmaznak mintaalkalmazásokat.

| SDK            | Közzétevő      |
| -------------- | -------------- |
| [.NET kiszolgáló](https://github.com/Azure/azure-notificationhubs-dotnet) | Microsoft |
| [Android](https://github.com/Azure/azure-notificationhubs-android)  |Microsoft |
| [Apache Cordova](https://github.com/derek82511/cordova-azure-notification-hubs) | Harmadik fél |
| [iOS](https://github.com/Azure/azure-notificationhubs-ios) |Microsoft |
| [Java szerver](https://github.com/Azure/azure-notificationhubs-java-backend) |Microsoft |
| [Nodejs](https://github.com/Azure/azure-sdk-for-node) | Microsoft |
| [Reagálni natív](https://github.com/CatalystCode/react-native-azurenotificationhub) | Harmadik fél |

> [!Note]
> A Microsoft nem vállal felelősséget a harmadik féltől származó megoldások minőségéért, hasznosságáért vagy támogathatóságáért.

Kérjük, tudassa velünk, ha van valami hiányzik.

# <a name="configure-android-device-messaging-adm-settings-for-a-notification-hub-in-the-azure-portal"></a>Egy értesítési központ Android Device Messaging (ADM) beállításainak konfigurálása az Azure Portalon
Ez a cikk bemutatja, hogyan az Azure notification hub Android Device Messaging (ADM) beállításainak konfigurálása az Azure portal használatával. 

## <a name="prerequisites"></a>Előfeltételek
Ha még nem hozott létre egy értesítési központ, hozzon létre egyet. További információkért lásd: [Azure notification hub létrehozása az Azure Portalon](create-notification-hub-portal.md). 

## <a name="configure-android-device-messaging"></a>Androidos eszközök üzenetküldése konfigurálása

Az alábbi eljárás biztosít egy értesítési központ Amazon Device Messaging (ADM) beállítások konfigurálásának lépései: 

1. Az Azure Portalon az a **értesítési központ** lapon jelölje be **Amazon (ADM)** a bal oldali menüben.
2. Adjon meg értéket a **ügyfél-azonosító** és **titkos Ügyfélkód**.
3. Kattintson a **Mentés** gombra.
    
   ![Képernyőfelvétel: az ADM-beállítások az Azure Portalon](./media/notification-hubs-kindle-get-started/notification-hub-adm-settings.png)

# <a name="sfctl"></a>sfctl
Parancsok kezelése a Service Fabric-fürtök és entitásokat. Ezen verziója kompatibilis 6.4 a Service Fabric-futtatókörnyezetben.

Az alábbi parancsok főnév-verb mintát. Alcsoportok további információt talál.

## <a name="subgroups"></a>Alcsoportok
|Alcsoport|Leírás|
| --- | --- |
| [Alkalmazás](service-fabric-sfctl-application.md) | Létrehozása, törlése és kezelheti az alkalmazások és alkalmazás-típusok. |
| [a Chaos](service-fabric-sfctl-chaos.md) | Indítása, leállítása és a chaos jelentés tesztelheti a szolgáltatást. |
| [cluster](service-fabric-sfctl-cluster.md) | Válassza ki, kezelése és üzemeltetése a Service Fabric-fürtök. |
| [Compose](service-fabric-sfctl-compose.md) | Létrehozása, törlése és a Docker Compose alkalmazások kezeléséhez. |
| [Tároló](service-fabric-sfctl-container.md) | Tároló futtatási kapcsolatos parancsok egy fürtcsomóponton. |
| [van](service-fabric-sfctl-is.md) | Lekérdezés, és parancsokat küldjön az infrastruktúra-szolgáltatás. |
| [háló](service-fabric-sfctl-mesh.md) | Törölje, és az alkalmazások Service Fabric-háló kezelése. |
| [Csomópont](service-fabric-sfctl-node.md) | A csomópontok, amely egy fürt kezelése. |
| [partition](service-fabric-sfctl-partition.md) | Lekérdezését, és bármely szolgáltatás partícióinak kezelését. |
| [A tulajdonság](service-fabric-sfctl-property.md) | A Service Fabric neve Store és a lekérdezés tulajdonságai. |
| [replica](service-fabric-sfctl-replica.md) | A szolgáltatás partícióinak tartozó replikák kezelése. |
| [rpm](service-fabric-sfctl-rpm.md) | Lekérdezés, és parancsokat küldjön a repair-kezelő szolgáltatás. |
| [SA-fürt](service-fabric-sfctl-sa-cluster.md) | Önálló Service Fabric-fürtök kezelése. |
| [service](service-fabric-sfctl-service.md) | Létrehozása, törlése és kezelése a service, a szolgáltatástípusok és a szolgáltatáscsomagok. |
| [Beállítások](service-fabric-sfctl-settings.md) | Konfigurálja helyi sfctl ezen példányának beállításait. |
| [store](service-fabric-sfctl-store.md) | Hajtsa végre a fürt lemezképtárolójába alapvető szolgáltatói műveletek. |

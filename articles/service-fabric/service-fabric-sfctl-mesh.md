# <a name="sfctl-mesh"></a>sfctl mesh
A Service Fabric Mesh-alkalmazások törlése és kezelése.

## <a name="subgroups"></a>Alcsoportok
|Alcsoport|Leírás|
| --- | --- |
| [App](service-fabric-sfctl-mesh-app.md) | Alkalmazás-erőforrások be- és törlése. |
| [kód-csomag-log](service-fabric-sfctl-mesh-code-package-log.md) | Az adott szolgáltatásreplika megadott kódcsomagtárolójának naplóinak beszereznie. |
| [Telepítési](service-fabric-sfctl-mesh-deployment.md) | Hozzon létre service fabric hálós erőforrásokat. |
| [Átjáró](service-fabric-sfctl-mesh-gateway.md) | Hálós átjáró-erőforrások be- és törlése. |
| [Hálózati](service-fabric-sfctl-mesh-network.md) | Hálóhálózati erőforrások be- és törlése. |
| [titkos kód](service-fabric-sfctl-mesh-secret.md) | Hálótitkos erőforrások be- és törlése. |
| [titkos érték](service-fabric-sfctl-mesh-secretvalue.md) | Háló titkos értékerőforrásainak be- és törlése. |
| [Szolgáltatás](service-fabric-sfctl-mesh-service.md) | Egy alkalmazás-erőforrás szolgáltatásrészleteinek és listázásának beszereznie. |
| [szolgáltatás-replika](service-fabric-sfctl-mesh-service-replica.md) | A replika részleteinek és egy adott szolgáltatás listájának listája egy alkalmazás-erőforrásban. |
| [Kötet](service-fabric-sfctl-mesh-volume.md) | Köteterőforrások beésése és törlése. |

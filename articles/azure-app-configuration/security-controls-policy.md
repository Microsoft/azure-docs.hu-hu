# <a name="azure-policy-regulatory-compliance-controls-for-azure-app-configuration"></a>Az Azure-alkalmazások konfigurációjának Azure Policy vonatkozó megfelelőségi szabályozása

A [Azure Policy szabályozási megfelelősége](../governance/policy/concepts/regulatory-compliance.md) biztosítja a Microsoft által létrehozott és felügyelt kezdeményezési definíciókat, amelyek a különböző megfelelőségi szabványokhoz kapcsolódó **megfelelőségi tartományokhoz** és **biztonsági ellenőrzésekhez** _készültek_. Ezen az oldalon a **megfelelőségi tartományok** és az Azure-alkalmazások konfigurációjának **biztonsági vezérlőinek** listája látható. A **biztonsági vezérlők** beépített moduljait egyenként is hozzárendelheti ahhoz, hogy az Azure-erőforrások megfeleljenek az adott szabványnak.

# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>Regisztráció az Azure Certified for IoT-portálon

Az Azure [Certified for IoT portál](https://aka.ms/ACFI) használatával:

- Fejezze be az IoT-eszköz minősítési folyamatát.
- IoT Plug and Play preview eszközképesség-modellek keresése. Ezekkel a modellekkel gyorsan hozhat létre IoT-kompatibilis eszközöket, és integrálhatja őket megoldásokkal.
- Eszközképességi modell közzététele a nyilvános tárházban.

## <a name="sign-in"></a>Bejelentkezés

_Microsoft-munkahelyi fiókjával_ jelentkezzen be az [Azure Certified for IoT portal](https://aka.ms/ACFI) portálra. A portál bejelentkezéskor ellenőrzi a Microsoft PartnerCenterben való tagságát:

- Ha szervezete már tagja a [Microsoft Partner Centernek,](https://docs.microsoft.com/partner-center/)hozzáférést kap a portál összes funkciójához.
- Ha szervezete nem tagja a [Microsoft Partner Centernek,](https://docs.microsoft.com/partner-center/)futtathat minősítési teszteket, és böngészhet az eszközképességi modell tárházaiközött. Az eszközt nem teheti közzé az IoT-tanúsítvánnyal rendelkező eszközkatalógusban.
- Ha megjelenik egy értesítési üzenet arról, hogy a Microsoft Partner Network-azonosítója hiányzik, [hozzon létre egy Partnerközpont-fiókot.](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account) A fiók jóváhagyása után visszatérhet a portálra, és közzéteheti az eszközt. További információt a [Partnerközpont gyakori kérdések című témakörben talál.](https://support.microsoft.com/help/4340639/partner-center-account-faqs)

Mielőtt közzéadná az eszközt az IoT-tanúsítvánnyal rendelkező eszközkatalógusban, olvassa el és fogadja el a programszerződést. A szerződés az Azure Certified for IoT portal beállítások lapján található.

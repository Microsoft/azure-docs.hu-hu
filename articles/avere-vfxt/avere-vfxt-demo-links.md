# <a name="avere-vfxt-demo-projects"></a>Avere vFXT demo projektek

A mintaoktató anyagok a [GitHubon](https://github.com/Azure/Avere)találhatók. Ezek a kis projektek bemutatják az Avere vFXT azure-hoz való kulcsfontosságú funkcióit és használati eseteit.

## <a name="video-rendering"></a>Videó renderelése

* [Renderelés az Azure Batch és az Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/maya_azure_batch_avere_vfxt_demo.md) használatával – Egy 60 perces projekt, amely bemutatja, hogyan használható az Autodesk Maya az Azure Batch és az Avere vFXT-fürt egy animációs film létrehozásához

* [Miért érdemes az Avere vFXT-t renderelésre használni?](https://github.com/Azure/Avere/blob/master/docs/why_avere_for_rendering.md) - Bemutató, amely összehasonlítja a renderelési időket a hálózathoz csatlakoztatott tárolóból Avere vFXT fürttel vagy anélkül

## <a name="high-performance-computing"></a>Nagy teljesítményű számítástechnika

* [Gyakorlati tanácsok az Azure Virtual Machine (VM) rendszerindítási idejének javításához](https://github.com/Azure/Avere/blob/master/docs/azure_vm_provision_best_practices.md) – Reprodukálható teszt, amely az Avere vFXT for Azure segítségével javítja a rendszerindítási időket, amikor gyorsan felpörgettöbb több ezer számítási ügyfelet

## <a name="vfxt-performance"></a>vFXT teljesítmény

* [A vFXT teljesítményének mérése a vdbench segítségével](https://github.com/Azure/Avere/blob/master/docs/vdbench.md) – Alapvető tesztbeállítás a virtuális és közepes méretű számítási feladatok létrehozásához a vFXT memória- és lemezalrendszerek teszteléséhez

## <a name="client-setup"></a>Ügyfél beállítása

* [Windows 10 workstation for Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/windows_10_avere_vfxt_mounted_workstation.md) - Bemutatja, hogyan kell beállítani egy Windows munkaállomást, és csatlakoztatni egy Avere vFXT fürthöz

# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>AI-alapú felhőszolgáltatás az Azure Cognitive Search segítségével
[Az Azure Cognitive Search](https://azure.microsoft.com/services/search/) egy szolgáltatásként kereshető felhőalapú megoldás, amely a fejlesztők számára API-kat és eszközöket biztosít a webes, mobil- és nagyvállalati alkalmazások ban található privát, heterogén tartalmakhoz való gazdag keresési élmény hozzáadásához. Az egyéni kód meghívja az adatok betöltését (indexelését) az index létrehozásához és betöltéséhez. A másik oldalon az alkalmazáskód lekérdezési kérelmeket ad ki, és kezeli a válaszokat. A keresési élmény az Ügyfélben az Azure Cognitive Search funkcióinak használatával van definiálva, és a lekérdezés végrehajtása egy megőrzött indexen keresztül történik, amelyet ön hoz létre, birtokol és tárol a szolgáltatásban.

## <a name="azure-cognitive-search-features"></a>Az Azure Cognitive Search szolgáltatásai
- **Szabadkézi szövegkeresés**: Teljes szöveges keresést, egyszerű lekérdezésszintaxist és Lucene lekérdezésszintaxist biztosít.
- **Egyszerű pontozási profilok:** Modell relevancia függvényében értékek a dokumentumokban magukat.
- **Geosearch**: A keresési eredmény fizikai helyhez való közelsége alapján vizsgálja az adatokat.
- **Felhasználói élmény funkciói**: Automatikus kiegészítés, keresési javaslatok, rendezés és lapozás.
- **Kognitív keresés:** Alkalmazható egy indexelési folyamat kinyerése szöveges információkat a nyers tartalom kép és szöveg elemzés.
- **Tudástároló**: Az indexelés során létrehozott gazdagításokat menti.
- **Adatforrások:** Kinyerheti a kereshető tartalmat az elsődleges adattárakban, amikor az Azure SQL Database, az Azure Cosmos DB és az Azure Blob storage csatlakozik.
- **Adatok importálása varázsló**: Az indexelők konfigurálása. 
- **Indextervező**: Indexet feltűnik. 
- **Kereséskezelő**: Teszteli a lekérdezéseket, és finomítja a pontozási profilokat.

Részletes leírást talál az Azure Cognitive Search [szolgáltatásairól.](/azure/search/search-what-is-azure-search#feature-descriptions)

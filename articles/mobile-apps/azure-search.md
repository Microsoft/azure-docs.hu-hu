# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>AI-alapú felhőalapú szolgáltatás az Azure Cognitive Search
Az [Azure Cognitive Search](https://azure.microsoft.com/services/search/) egy felhőalapú keresési megoldás, amely lehetővé teszi a fejlesztők számára API-k és eszközök számára, hogy a webes, mobil-és vállalati alkalmazásokban gazdag keresési élményt adjanak hozzá. Az egyéni kód meghívja az adatfeldolgozást (indexelést) egy index létrehozásához és betöltéséhez. A másik oldalon az alkalmazás kódja lekérdezi a kérelmeket, és kezeli a válaszokat. A keresési élmény az Azure Cognitive Search funkcióinak használatával van meghatározva az ügyfélen, és a lekérdezés végrehajtása a szolgáltatásban létrehozott, saját és tárolt megőrzött indexen keresztül történik.

## <a name="azure-cognitive-search-features"></a>Azure Cognitive Search-funkciók
- **Szabad formátumú szöveges keresés** a teljes szöveges kereséssel, az egyszerű lekérdezési szintaxissal és a Lucene lekérdezési szintaxissal.
- **Egyszerű pontozási profilok** a dokumentumokban lévő értékek függvényében.
- A **geo-Search** lehetővé teszi a felhasználók számára, hogy a keresési eredmények közelsége alapján egy fizikai helyre tárja fel az adatelemzést.
- A felhasználó az automatikus kiegészítéssel, a keresési javaslatokkal, a rendezéssel és a lapozással kapcsolatos **funkciókat** kínál.
- A képek és a szövegek **megismerésére szolgáló kognitív keresés** alkalmazható egy indexelési folyamatra a szöveges információk nyers tartalomból való kinyeréséhez.
- A **Tudásbázisban** az indexelés során létrehozott dúsítások menthetők.
- Csatlakozhat a Azure SQL Databasehoz, Azure Cosmos DBhoz vagy az Azure Blob Storage-hoz **adatforrásként** a kereshető tartalom kinyeréséhez az elsődleges adattárakban.
- Az **adatimportálás varázsló** segítségével konfigurálhatja az indexelő és az indexelő **tervezőt** az indexek kiépítéséhez, valamint a **keresési tallózóhoz** a lekérdezések teszteléséhez és a pontozási profilok pontosításához.

Részletes leírást talál az Azure Cognitive Search [szolgáltatásairól](/azure/search/search-what-is-azure-search#feature-descriptions).

# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Content Delivery Network az Azure-ban az Azure Content Delivery Network
Az [Azure Content Delivery Network](https://azure.microsoft.com/services/cdn/) globális megoldást kínál a fejlesztők számára a nagy sávszélességű tartalmak, például videók, képek vagy statikus fájlok gyors kézbesítésére a felhasználók számára a tartalom több száz, a világ különböző pontjain való gyorsítótárazásával. Content Delivery Network a dinamikus tartalmat is felgyorsíthatja, amely nem gyorsítótárazható, csökkentheti a betöltési időt, sávszélességet takaríthat meg, és növelheti a rugalmasságot. Ezek az előnyök a webhelyek és a mobileszközök fejlesztésére, valamint a streaming média, a játékok, a belső vezérlőprogram frissítései vagy a IoT-végpontok kódolására és terjesztésére használhatók.

## <a name="azure-content-delivery-network-features"></a>Azure Content Delivery Network-funkciók
  - Zökkenőmentesen dolgozhat olyan Azure-szolgáltatásokkal, mint a web Applications, a Media Services, a Storage és a Cloud Services.
  - Egyszerűen kezelheti a forgalmi tüskéket és a nagy terheléseket.
  - A Akamai és a Verizon integrálásával kialakíthatja és testre szabhatja a Content Delivery stratégiát.
  - Dinamikus hely gyorsításának használata.
  - Szabályozza Content Delivery Network gyorsítótárazási viselkedését a gyorsítótárazási szabályokkal.
  - HTTPS-alapú egyéni tartomány támogatásának beszerzése.
  - Az Azure Diagnostics naplóiban megtekintheti az alapvető elemzéseket, és exportálhatja őket az Azure Blob Storage-ba, az Azure Event Hubsba vagy Azure Monitorba.
  - A teljesítmény javítása a fájlok tömörítésével.
  - Az geofiltering használatával korlátozza a Content Delivery Network a tartalmat ország vagy régió szerint.

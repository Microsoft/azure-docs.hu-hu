# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Content Delivery Network az Azure-ban Azure CDN szolgáltatással
A [Azure CDN](https://azure.microsoft.com/services/cdn/) globális megoldást kínál a fejlesztők számára a nagy sávszélességű tartalmak, például videók, képek vagy statikus fájlok gyors kézbesítésére a felhasználók számára azáltal, hogy a tartalmakat több száz jelenléti pontra (POP) gyorsítótárazzák a világ minden részén. Azure CDN a dinamikus tartalmat is felgyorsíthatja, amely nem gyorsítótárazható, és lehetővé teszi a betöltési idő csökkentését, a sávszélesség mentését és a rugalmasság növelését, akár webhelyeket, akár mobil alkalmazásokat fejleszt vagy kezel média, játék szoftver, belső vezérlőprogram frissítései vagy IoT-végpontok.

## <a name="azure-cdn-features"></a>Az Azure CDN szolgáltatásai
  - **Zökkenőmentesen** együttműködik az Azure-szolgáltatásokkal, beleértve a webalkalmazásokat, a Media Servicesokat, a tárolást és a Cloud Services.
  - Egyszerűen **kezelheti a forgalmi tüskéket és a nagy terheléseket**.
  - A **Akamai és a Verizon integrációja** a Content Delivery-stratégia összeállításához és testreszabásához.
  - Dinamikus hely gyorsulása.
  - Szabályozza Azure CDN gyorsítótárazási viselkedését a **gyorsítótárazási szabályokkal**.
  - HTTPS- **alapú egyéni tartomány támogatása**.
  - **Azure Diagnostics-naplók** az alapvető elemzések megtekintéséhez és exportálásához Blob Storage, Event Hubs vagy Azure monitor.
  - A teljesítmény javítása a **fájlok tömörítésével**.
  - **Geo-szűrés** a Azure CDN tartalom ország/régió szerinti korlátozásához.

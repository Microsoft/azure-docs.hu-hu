# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Tartalomkézbesítési hálózat az Azure-ban az Azure Content Delivery Network hálózattal
[Az Azure Content Delivery Network](https://azure.microsoft.com/services/cdn/) globális megoldást kínál a fejlesztőknek a nagy sávszélességű tartalmak, például videók, képek vagy statikus fájlok gyors továbbítására a felhasználók számára azáltal, hogy a tartalmaikat világszerte több száz jelenléti ponthoz gyorsítótárasztja. A Content Delivery Network felgyorsíthatja a dinamikus tartalmat is, amely nem gyorsítótárazható, hogy csökkentse a betöltési időt, sávszélességet takarítson meg és növelje a válaszidőt. Ezeket az előnyöket akár webhelyek, akár mobilalkalmazások fejlesztése vagy kezelése, akár streaming média, játékszoftver, firmware-frissítések vagy IoT-végpontok kódolása és terjesztése teszi.

## <a name="azure-content-delivery-network-features"></a>Az Azure Content Delivery Network szolgáltatásai
  - Zökkenőmentesen dolgozhat az Azure-szolgáltatásokkal, például a webes alkalmazásokkal, a médiaszolgáltatásokkal, a tárolással és a felhőszolgáltatásokkal.
  - Könnyen kezelhető a forgalmi csúcsok és a nagy terhelések.
  - Integrálja az Akamai-val és a Verizonnal a tartalomkézbesítési stratégia kialakításához és személyre szabásához.
  - Dinamikus helygyorsítás használata.
  - A Tartalomkézbesítési hálózat gyorsítótárazási viselkedésének szabályozása gyorsítótárazási szabályokkal.
  - Szerezzen be HTTPS egyéni tartománytámogatást.
  - Tekintse meg az Azure diagnosztikai naplók at az alapvető elemzések megtekintéséhez és exportálásához az Azure Blob storage, az Azure Event Hubs vagy az Azure Monitor.
  - A fájlok tömörítésével javíthatja a teljesítményt.
  - A geoszűrés segítségével ország vagy régió szerint korlátozhatja a Tartalomkézbesítési hálózat tartalmát.

# <a name="azure-policy-regulatory-compliance-controls-for-azure-automation"></a>Azure Policy megfelelőségi vezérlői a Azure Automation

[A megfelelőségi Azure Policy](../governance/policy/concepts/regulatory-compliance.md) a Microsoft által létrehozott és felügyelt kezdeményezési definíciókat (más néven  beépítetteket) biztosít a különböző megfelelőségi szabványokhoz kapcsolódó megfelelőségi tartományokhoz és biztonsági vezérlőkhöz.  Ezen az oldalon a megfelelőségi **tartományok** és a biztonsági **vezérlők** Azure Automation. A beépített biztonsági vezérlőket egyenként  is hozzárendelheti, hogy az Azure-erőforrások megfeleljenek az adott szabványnak.

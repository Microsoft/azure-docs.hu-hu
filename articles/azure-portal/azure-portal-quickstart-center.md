# <a name="get-started-with-the-azure-quickstart-center"></a>Ismerkedés az Azure gyorsútmutatóval

Az Azure Quickstart Center egy interaktív élmény az Azure Portalon, amely bárki számára elérhető, aki javítani szeretné az Azure-ral kapcsolatos ismereteit. Az Azure-ban új szervezetek számára ez a leggyorsabb módja a felhőkörnyezet beszervezésének és beállításának.

## <a name="overview"></a>Áttekintés

Az Azure Quickstart Center három lehetőséget kínál az első lépésekhez:

* **Beállítási útmutatók:** Az informatikai rendszergazda és a felhőtervező számára tervezett útmutatóink bemutatják az Azure bevezetésének legfontosabb fogalmait. A strukturált lépések segítségével a Microsoft ajánlott eljárásait is alkalmazhatja a tanulás sal. Az áttelepítési útmutató segítségével felmérheti a készültséget, és megtervezheti az Azure-ra való áttérést.

* **Projekt indítása:** Ha készen áll egy erőforrás létrehozására, ebben a szakaszban további tudnivalókat olvashat a lehetőségekről, mielőtt elkötelezné magát egy szolgáltatási lehetőség mellett. További információkkal tudhat meg a szolgáltatásról és arról, hogy miért érdemes használni, feltárni a költségeket és azonosítani az előfeltételeket. Miután a választás, akkor közvetlenül létrehozni.

* **Online tanulás**: Az Azure Quickstart Center ezen szakasza a Microsoft Learn ingyenes bevezető tanfolyami moduljait emeli ki. Válasszon ki egy csempét egy tanfolyam elindításához, és tudjon meg többet a felhőbeli fogalmakról és az erőforrások kezeléséről az Azure-ban.

## <a name="how-to-use-azure-quickstart-center"></a>Az Azure Gyorsútmutató használata

1. Jelentkezzen be az [Azure Portalra.](https://portal.azure.com)

1. Válassza az Azure Portal menü **Minden szolgáltatás** lehetőséget.

1. Válassza **az Általános** > **rövidútmutató stanéven.**

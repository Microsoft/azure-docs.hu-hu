# <a name="keyboard-shortcuts-in-the-azure-portal"></a>Billentyűparancsok az Azure Portalon

Ez a cikk felsorolja az Azure Portalon mindenütt használható billentyűparancsokat. Előfordulhat, hogy az egyes szolgáltatások saját billentyűparancsokkal rendelkeznek.

## <a name="actions"></a>Műveletek

|A művelet menete |Nyomja le |
| --- | --- |
|Erőforrás létrehozása|G+N|
|**Az összes szolgáltatás megnyitása**|G+B|
|Erőforrások, szolgáltatások és dokumentumok keresése|G+/|
|Erőforrás menüelemeinek keresése|CTRL+/ |
|A kijelölt bal oldali oldalsáv-elem feljebb lépése |ALT+Shift+Fel nyíl|
|A kijelölt bal oldali oldalsávelem lefelé mozgatása |ALT+Shift+Le nyíl|

## <a name="navigation"></a>Navigáció

|Ehhez a navigációhoz |Nyomja le |
| --- | --- |
|Fókusz áthelyezése a parancssávra |G+, |
|Fókusz váltása a fejléc és a bal oldalsáv között | G+. |

## <a name="go-to"></a>Ugrás ide

|Ugrás erre a helyre |Nyomja le |
| --- | --- |
|Ugrás az **irányítópultra** |G+D |
|Ugrás a **Minden erőforrásra**|G+A |
|Ugrás **az Erőforráscsoportokra**|G+R |
|A bal oldali oldalsáv elem megnyitása ebben a pozícióban |G+szám|

# <a name="keyboard-shortcuts-in-the-azure-portal"></a>Billentyűparancsok az Azure Portalon

Ez a cikk a Azure Portalban mindenhol működő billentyűparancsokat sorolja fel. Előfordulhat, hogy az egyes szolgáltatások saját billentyűparancsokkal rendelkeznek.

## <a name="actions"></a>Műveletek

|A művelet menete |Nyomja le |
| --- | --- |
|Erőforrás létrehozása|G + N|
|**Az összes szolgáltatás** megnyitása|G + B|
|Erőforrások, szolgáltatások és dokumentumok keresése|G +/|
|Erőforrás-menüelemek keresése|CTRL +/ |
|A kijelölt bal oldalsáv-elem mozgatása |ALT + SHIFT + felfelé mutató nyíl|
|A kijelölt bal oldalsáv-elem mozgatása lefelé |ALT + SHIFT + lefelé mutató nyíl|

## <a name="navigation"></a>Navigáció

|A navigáció elvégzése |Nyomja le |
| --- | --- |
|Fókusz áthelyezése a parancssáv fölé |G +, |
|Fókusz váltása a fejléc és a bal oldali oldalsáv között | G +. |

## <a name="go-to"></a>Ugrás ide

|Ugrás erre a helyre |Nyomja le |
| --- | --- |
|Ugrás az **irányítópultra** |G + D |
|Ugrás az **összes erőforrásra**|G + A |
|Erőforrás- **csoportok** keresése|G + R |
|A bal oldali oldalsáv-elem megnyitása ezen a pozíción |G + szám|

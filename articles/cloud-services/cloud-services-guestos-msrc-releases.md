# <a name="azure-guest-os"></a>Azure-beli vendég operációs rendszer
Az alábbi táblázatok az Azure Microsoft Biztonsági reagálási központ (MSRC) frissítéseit mutatják be. Ebben a cikkben annak megállapításához, hogy egy adott frissítés vonatkozik-e az Ön által használt vendég operációs rendszerre. A frissítések mindig az adott [családra][family-explain] vonatkozóan mindig frissülnek, amelybe bevezették őket.

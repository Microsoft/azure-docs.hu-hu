# <a name="microsoft-trust-center"></a>Microsoft adatvédelmi központ
Az Azure.com webhelyen az Azure biztonsági információk helyet biztosít a szükséges információkat, megtervezéséhez, tervezési, központi telepítése, konfigurálása, és biztonságosan kezelheti a felhőalapú megoldások. A Microsoft Trust Center az adatokat kell lennie abban, hogy az Azure platform, amelyen futtatja a szolgáltatások biztonságos is van.

Tudjuk, hogy az alkalmazások és adatok bízza az Azure-ba, amikor fog kérdése van. Hol található azt? Férhet hozzá? Milyen módon a Microsoft a védelmét? Hogyan is, győződjön meg arról, hogy a Microsoft milyen ugyanakkor állapotát?

És válaszokat. Mivel az adatok, dönt, hogy kinek van hozzáférése, és, működjön együtt velünk, dönthet arról, hogy helyétől. Az adatok védelme érdekében, hogy használjuk állapota-a-a legújabb biztonsági technológia és világszínvonalú titkosítás. A megfelelőségi egymástól függetlenül ellenőrzi, és számos szinteken transzparens vagyunk – a hogyan kezeljük a kódban, a biztonság az ügyféladatok jogi igények.

Itt látható a Microsoft Trust Center címen található:

* [Biztonsági](https://aka.ms/tcsecurity) – ismerje meg, hogy a Microsoft Cloud-szolgáltatások biztonságosak.
* [Adatvédelmi](https://aka.ms/tcprivacy) – megismerheti, hogy a Microsoft biztosítja, hogy az adatok a Microsoft felhőbeli adatvédelmi.
* [Megfelelőségi](https://aka.ms/tccompliance) – Fedezze fel, hogyan a Microsoft segít a szervezeteknek felel meg a gyűjtemény és az egyének adatok használatát szabályozó országos, regionális és iparág-specifikus követelményeknek.
* [Átlátszóság](https://aka.ms/tctransparency) – hogyan Microsoft úgy véli, hogy Ön határozhatja meg az adatok a felhőben és a Microsoft miként biztosítja, hogy a lehető legnagyobb mértékben kapcsolatos adatok kezelésének módját.
* [Termékek és szolgáltatások](https://aka.ms/tcproductsservices) – tekintse meg az összes Microsoft Cloud-termékek és szolgáltatások egy helyen
* [Szolgáltatásmegbízhatósági portálon](https://aka.ms/tcservicetrportal) – ajánlott eljárások szerezze be a független naplózási jelentések Microsoft felhőszolgáltatásra, kockázatfelmérések, biztonsági másolatait, és a kapcsolódó anyagokat.
* [What's New](https://aka.ms/tcwhatsnew) – ismerje meg, what's new in Microsoft Cloud megbízhatóság
* [Erőforrások](https://aka.ms/tcresources) – tanulmányokat, videókat és a Microsoft megbízható felhőben esettanulmányok vizsgálata

A [Microsoft Trust Center](https://www.microsoft.com/trustcenter) kell tudni, hogy mire biztonságossá tétele a Microsoft Cloud rendelkezik.

# <a name="microsoft-security-response-center"></a>Microsoft Security Response Center

A Microsoft Security Response Center (MSRC) a rendszer a világ biztonsági szakértői által vezetett. Azok azonosítása, figyelése, válaszol, és oldja meg a biztonsági incidensek és a helyszíni és felhőbeli biztonsági résekről, az óra körül, az év egyes napjaira.

Mellett folyamatos feladatát a MSRC csapat egy Ön számára elérhető erőforrások száma rendelkezik, így megismerheti, hogyan teheti biztonságossá az Azure-objektumok és a központi telepítések hatékonyabban.

## <a name="the-msrc-blog"></a>A MSRC Blog

A [MSRC blog](https://blogs.technet.microsoft.com/msrc/) nyissa meg a legfrissebb hírek a MSRC mire felhőbeli fenyegetések elleni védelem érdekében, hogy az a hely.

## <a name="white-papers"></a>Tanulmányok

A MSRC számos közzétett [tanulmányok](https://technet.microsoft.com/library/bb969102.aspx) adott segítenek megérteni, hogy mit tesznek azt, és a következőképpen tehetik. Az egyes védelmének biztosítása a Microsoft cloud betekintést nyújtson, és hogyan biztonsági ugyanezzel a konfigurációval azonban hasznos információkat tartalmazzák.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Biztonsági kutató Engagement és kiteljesíti programok

A MSRC támogatja az együttműködés és a biztonsági kutatók globálisan azt mutatja be a Microsoft termék biztonsági kapcsolatokkal.

A Microsoft kiteljesíti programok kell fizetnie a kutatók:

- Új kiaknázása technikák
- Védelmi vonalként ötleteit, amely az új biztonsági rések csökkentése
- A Microsoft a kritikus biztonsági rések azonosítása a helyszíni és felhőbeli szoftvert.

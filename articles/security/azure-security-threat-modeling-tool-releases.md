# <a name="threat-modeling-tool-releases"></a>Threat Modeling eszköz kiadására

A Microsoft Threat Modeling Tool az általánosan elérhető egy szabad [kattintson-letöltési](https://aka.ms/threatmodelingtool) Windows alkalmazás. Ez a kézbesítési mechanizmus lehetővé teszi számunkra, hogy küldje le a legújabb javítások és hibajavítások az ügyfelek számára minden alkalommal, amikor megnyitják az eszközt, így könnyebben karbantartásáról és használatáról.

## <a name="system-requirements"></a>Rendszerkövetelmények

- Támogatott operációs rendszerek
  - Microsoft Windows 10
- További követelmények
  - Internetkapcsolat szükséges az eszköz, valamint a sablonok a frissítések fogadásához.

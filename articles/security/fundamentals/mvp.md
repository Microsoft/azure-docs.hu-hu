# <a name="azure-security-mvp-program"></a>Azure Security MVP-program

A Microsoft legértékesebb szakemberei (MVP-EK) olyan közösségi vezetők, akik példaértékű elkötelezettséget tanúsítanak a másoknak való segítségnyújtáshoz. A MVP lehetővé teszi mások számára a Microsoft-technológiákkal való lehető leghatékonyabban nyújtott tapasztalatát. A Közösséggel és a Microsofttal közösen megosztják a szenvedélyét, a valós ismereteiket és a technikai szakértelmét.

A Microsoft Azure most felismeri az Azure Security különleges szakértelmével rendelkező közösségi szakértőket. A Microsoft MVP az MVP-t az Azure biztonsági hozzájárulási területén Microsoft Azure is odaítélheti.

![Microsoft MVP](./media/mvp/azure-security-mvp-fig1.png)

Nincs olyan teljesítményteszt, amely az MVP-nek valóra vált. Ez részben azért van, mert a technológia és életciklusa alapján változik. A feltételek némelyike az alábbiakat tartalmazza:

- A jelölteknek az online fórumokon, például a Microsoft Answers, a TechNet és az MSDN szolgáltatásban való hozzájárulásainak következményei
- Wikik és online tartalmak
- Konferenciák és felhasználói csoportok
- Podcastok, webhelyek, blogok és közösségi média
- Cikkek és könyvek.

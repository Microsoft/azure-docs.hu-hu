# <a name="azure-security-overview"></a>Az Azure biztonsági szolgáltatásainak áttekintése

Az Azure biztonsági kérdéseire vonatkozó technikai válaszokat a Azure.com Microsoft Azure biztonsági információs webhelyén találja. Ha érdekli az Azure megfelelősége, és hogy az Azure Hogyan védi a saját infrastruktúráját, látogasson el a [Microsoft adatvédelmi](https://microsoft.com/en-us/trustcenter/default.aspx)központba.

Tudjuk, hogy a biztonság az egyik a felhőben, és mennyire fontos, hogy pontos és kellő időben tájékozódjon az Azure biztonságáról. Az Azure alkalmazásokhoz és szolgáltatásokhoz való használatának egyik legjobb oka az, hogy kihasználja az elérhető biztonsági eszközök és képességek széles körét. Ezek az eszközök és képességek segítik a biztonságos megoldások létrehozását a biztonságos Azure platformon.

Ez a lap és ezen a webhelyen az új Azure-szolgáltatások online állapotba kerülnek, és frissülnek a meglévő szolgáltatások. Biztonsági ajánlatainkat folyamatosan fejlesztjük, és itt megosztjuk a fejlesztések technikai részleteit. Javasoljuk, hogy rendszeresen térjen vissza erre a webhelyre, és tekintse meg az újdonságokat.

## <a name="using-the-azure-security-information-site"></a>Az Azure biztonsági információs webhely használata

A lap bal oldalán megtekintheti a biztonsági témakörök kategóriáinak gyűjteményét. Kattintson vagy koppintson a témakörre a kategória cikkeinek megtekintéséhez.

Minden kategóriához tartozik egy áttekintő cikk. Az áttekintő cikk célja, hogy összefoglalást nyújtson arról, hogy mit kínál az Azure az adott kategória biztonsága szempontjából. Az áttekintő cikkek segítenek megismerni az Azure-ban elérhető termékeket, szolgáltatásokat és szolgáltatásokat, amelyek segítségével javíthatja az üzemelő példányok biztonságát. Ezek az áttekintő cikkek a technikai tartalomra mutató hivatkozásokat tartalmaznak, így többet tudhat meg a témakörökről és azok megvalósításáról.

Az Azure-beli biztonsági információs hely jelenleg a következő kategóriákat kezeli:

* További források
* Hálózati biztonság
* Tárbiztonság
* Virtuális gépek biztonsága
* Endpoint Protection
* Biztonságkezelés és megfigyelés
* Identitáskezelés
* PaaS szolgáltatások
* Iparág-specifikus
* Architektúra

> [!NOTE]
> Az Azure biztonsági információi oldalon található összes cikk megjelenik az oldal bal oldalán lévő navigációban. Ha meg szeretné tekinteni az összes cikket, kattintson vagy koppintson a továbbiak elemre a Kategóriák lista alján.
>
>

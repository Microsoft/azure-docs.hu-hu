# <a name="microsoft-security-response-center"></a>Microsoft Security Response Center

A Microsoft Security Response Center (MSRC) a világ egyik legtapasztaltabb biztonsági szakértője. A biztonsági incidenseket, a helyszíni és a Felhőbeli biztonsági réseket az év minden napján azonosítják, figyelik, válaszolják meg és oldják meg.

A folyamatos munkája mellett a MSRC csapata számos erőforrást biztosít Önnek, így megismerheti, hogyan védheti meg az Azure-eszközöket és-üzembe helyezéseket hatékonyabban.

## <a name="the-msrc-blog"></a>A MSRC blogja

A [MSRC blog](https://blogs.technet.microsoft.com/msrc/) az a hely, ahol megtekintheti a legfrissebb híreket arról, hogy mit tesz a MSRC a Felhőbeli fenyegetések elleni védelem érdekében.

## <a name="white-papers"></a>Tanulmányok

A MSRC számos olyan tanulmányt tett [](https://technet.microsoft.com/library/bb969102.aspx) közzé, amelyek segítenek megérteni, hogy mit csinálnak, és hogyan használják. Egyesek betekintést nyújtanak a Microsoft-felhő biztonságossá tételére, és hasznos információkkal szolgálnak az azonos biztonsági konfigurációk alkalmazásáról.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Biztonsági kutató engagement és Bounty programok

A MSRC a Microsoft termékbiztonság érdekében globálisan támogatja a biztonsági kutatókkal való együttműködést és kapcsolatokat.

A Microsoft Bounty-programok a következőknek fizetnek kutatókat:

- Új kiaknázási technikák
- Védelmi ötletek, amelyek az új biztonsági réseket használják
- Kritikus biztonsági rések azonosítása a Microsoft helyszíni és felhőalapú szoftverében.

## <a name="next-steps"></a>További lépések

További információ ezekről a programokról: [MSRC bug Bounty](https://technet.microsoft.com/security/dn425036) oldal és a [MSRC blog](https://blogs.technet.microsoft.com/msrc/).

Ha többet szeretne megtudni a MSRC, tekintse meg a [MSRC kezdőlapját](https://technet.microsoft.com/library/dn440717.aspx).

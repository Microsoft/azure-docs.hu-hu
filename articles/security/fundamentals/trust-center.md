# <a name="microsoft-trust-center"></a>Microsoft adatvédelmi központ
A Azure.com Azure Security információs webhelye biztosítja a felhőalapú megoldások biztonságos tervezéséhez, megtervezéséhez, üzembe helyezéséhez, konfigurálásához és kezeléséhez szükséges információkat. A Microsoft adatvédelmi központtal arról is tájékozódhat, hogy az Azure platform, amelyen a szolgáltatásokat futtatja, biztonságban kell lennie.

Tudjuk, hogy ha az alkalmazásokat és az adatait az Azure-ba bízza, kérdései lesznek. Hol található? Ki férhet hozzá? Mit tesz a Microsoft a védelemhez? Hogyan tudja ellenőrizni, hogy a Microsoft mit mond?

És választ kaptunk. Mivel az Ön adatai, Ön dönti el, hogy ki fér hozzá, és Ön dolgozik velünk, hogy eldöntse, hol található. Az adatai védelme érdekében a legkorszerűbb biztonsági technológiát és világszínvonalú titkosítást használjuk. A megfelelőségi adatok egymástól függetlenek, és számos szinten transzparensek vagyunk, attól, hogy hogyan kezeljük az ügyféladatokat a kód biztonságára vonatkozó jogi igényekkel.

A Microsoft adatvédelmi központban találja a következőt:

* [Biztonság](https://aka.ms/tcsecurity) – megtudhatja, hogyan biztosítható az összes Microsoft Cloud szolgáltatás.
* [Adatvédelem](https://aka.ms/tcprivacy) – Ismerje meg, hogy a Microsoft miként gondoskodik adatai védelméről a Microsoft-felhőben.
* [Megfelelőség](https://aka.ms/tccompliance) – Ismerje meg, hogy a Microsoft hogyan segíti a szervezeteknek az egyéni adatgyűjtést és-felhasználást szabályozó nemzeti, regionális és iparági követelményeknek való megfelelést.
* [Átláthatóság](https://aka.ms/tctransparency) – megtekintheti, hogyan kezeli a Microsoft az adatait a felhőben, és hogyan segíti a Microsoft a lehető legnagyobb mértékben az adatkezelési folyamatokat.
* [Termékek és szolgáltatások](https://aka.ms/tcproductsservices) – egy helyen tekintheti meg az összes Microsoft Cloud terméket és szolgáltatást
* [Szolgáltatás-megbízhatósági portál](https://aka.ms/tcservicetrportal) – a Microsoft Cloud Services, a kockázatértékelés, az ajánlott biztonsági eljárások és a kapcsolódó anyagok független naplózási jelentéseinek másolatait szerezheti be.
* [Újdonságok – Ismerje](https://aka.ms/tcwhatsnew) meg az Microsoft Cloud Trust újdonságait
* [Erőforrások](https://aka.ms/tcresources) – tanulmányok, videók és esettanulmányok kivizsgálása a Microsoft megbízható felhőben

A [Microsoft adatvédelmi](https://www.microsoft.com/trustcenter) központjának tisztában kell lennie azzal, hogy mit teszünk a Microsoft Cloud biztonságossá tételéhez.

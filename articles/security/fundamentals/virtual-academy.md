# <a name="azure-security-courses-from-microsoft-virtual-academy"></a>A Microsoft Virtual Academy Azure Biztonsági kurzusai

Ide tartozik a biztonsági fejlesztési és üzembe helyezési eljárásokkal kapcsolatos lecke.

[Gyakori feladatok Linux rendszeren az Azure-ban](https://mva.microsoft.com/en-us/training-courses/common-tasks-for-linux-on-azure-16191?l=J0Hvb7qJC_1204668937)

[A felhő védelme](https://mva.microsoft.com/en-us/training-courses/secure-the-cloud-14037?l=lQIkkst0B_5300115881)

Ismerje meg az árnyékok felderítéséhez, az alkalmazás-hozzáférés kezeléséhez és a gyanús tevékenységek figyeléséhez használt technológiákat a speciális biztonsági jelentéskészítés, a felhasználói viselkedés elemzései, a naplózás és a riasztások segítségével.

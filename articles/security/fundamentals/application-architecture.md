# <a name="application-architecture-on-azure"></a>Alkalmazásarchitektúra az Azure-on
A Microsoft Azure felhőalapú megoldásai biztonságossá tételéhez elengedhetetlen, hogy egy erős építészeti alap legyen. Az építészek, tervezők és kivitelezők minden előnyt élveznek az alkalmazások és szolgáltatások architektúrájának hatékony ismeretében. Ez az alapvető ismeretek segítségével megismerheti a felhőalapú megoldások összes összetevőjét, és egyszerűbbé teheti a biztonság integrálását a tervezés és a megvalósítás minden aspektusára.

Az alábbi lépéseket követve segítséget nyújtunk az építészeti vizsgálatokhoz és kialakításokhoz:

* Építészeti tervek
* Felhő-és nagyvállalati szimbólum beállítása
* 3D terv Visio-sablon

## <a name="architectural-blueprints"></a>Építészeti tervek
A Microsoft olyan magas szintű [építészeti tervrajzokat](https://aka.ms/azblueprints) tesz közzé, amelyek bemutatják, hogyan hozhat létre bizonyos típusú rendszereket a Microsoft-termékekkel.
Minden tervezet tartalmaz egy:

* Lapos 2D Visio 2003-alapú fájl, amelyet letöltheti és módosíthatja
* Színes 3D perspektíva PDF-fájl a tervnek a kevésbé technikai hallgatósághoz való bevezetéséhez
* Videó, amely végigvezeti a 3D-s verzión

## <a name="cloud-and-enterprise-symbol-set"></a>Felhő-és nagyvállalati szimbólum beállítása
[Tekintse meg a Visio és a szimbólumok betanítása videót](https://aka.ms/CnESymbolsVideo) , majd [töltse le a Cloud and Enterprise szimbólumot](https://aka.ms/CnESymbols) úgy, hogy segítsen létrehozni az Azure-t, a Windows servert és a SQL Servert leíró műszaki anyagokat. Használhatja az architektúra-diagramok, a tananyagok, a bemutatók, az adatlapok, a infographics, a tanulmányi és a harmadik féltől származó könyvek szimbólumait, ha a könyv a Microsoft termékeinek használatára vonatot használ. Ezeket azonban nem a felhasználói felületeken való használatra szántuk.

## <a name="3d-blueprint-visio-template"></a>3D terv Visio-sablon
A [Microsoft Architecture tervezetek](https://aka.ms/azblueprints) 3D-verziói eredetileg egy nem Microsoft-eszközön lettek létrehozva. Egy új Visio 2013 (és újabb) sablon, amely 2015 augusztus 5-én, a [EDX.org-on elosztott Microsoft Architecture minősítési kurzus](https://docs.microsoft.com/azure/architecture/)részeként lett elküldve.

A sablon a tanfolyamon kívül is elérhető.

* Először tekintse meg [a betanítási videót](https://aka.ms/3dBlueprintTemplateVideo) , hogy tudja, mit tehet
* A [Microsoft 3D Blueprint Visio-sablon](https://aka.ms/3DBlueprintTemplate) letöltése
* A 3D sablonnal használható [Felhőbeli és vállalati szimbólumok](https://docs.microsoft.com/azure/architecture/) letöltése

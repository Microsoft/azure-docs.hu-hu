# <a name="azure-network-security"></a>Azure hálózati biztonság
## <a name="abstract"></a>Absztrakt
Az Azure hálózati szolgáltatásai a tervezéssel maximalizálják a rugalmasságot, a rendelkezésre állást, a rugalmasságot, a biztonságot és az integritást. Ez a tanulmány részletesen ismerteti az Azure hálózatkezelési funkcióit. Azt is leírja, hogyan használhatják az ügyfelek az Azure natív biztonsági funkcióit az információs eszközök védelmére.
A tanulmány célközönsége az alábbiakat tartalmazza:
- Technikai vezetők, hálózati rendszergazdák és fejlesztők, akik az Azure-ban elérhető és támogatott biztonsági megoldásokat keresnek.
- A KKV-k és az üzleti folyamatok vezetői, akik magas szintű áttekintést szeretnének a nyilvános Azure-felhőben a hálózati biztonsággal kapcsolatos Azure-technológiákról és-szolgáltatásokról.

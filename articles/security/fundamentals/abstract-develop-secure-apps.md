# <a name="develop-secure-cloud-applications-on-azure"></a>Biztonságos felhőalapú alkalmazások fejlesztése az Azure-ban
## <a name="abstract"></a>Absztrakt
Ez a tanulmány általános útmutatót tartalmaz a biztonsági kérdésekről és ellenőrzésekről, amelyeket figyelembe kell venni a szoftverfejlesztés életciklusának minden fázisában a Felhőbeli alkalmazások fejlesztésekor. Ezen fogalmak bevezetése a termék kiadása előtt több biztonságos szoftver kiépítésében is segít. Az ebben a dokumentumban bemutatott javaslatok az Azure Security és az ügyfelek tapasztalataiból származnak.

Ez a tanulmány az Azure-megoldások fejlesztésére és üzembe helyezésére szolgáló minden szinten a szoftverfejlesztők, fejlesztők és tesztelők számára készült erőforrás.

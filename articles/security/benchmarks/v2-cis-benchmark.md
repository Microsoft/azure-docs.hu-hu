# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Az Azure Security benchmark v2 és a CIS Microsoft Azure alapítványok Teljesítménytesztének leképezése

A CIS Microsoft Azure alapítványok Teljesítményteszte az Internet Security Center által biztosított biztonsági útmutatás az Azure biztonságos alapkonfigurációjának létrehozásához. A teljesítményteszt hatóköre az Azure-felhő bevezetésekor a biztonság alapszintű szintjének meghatározása. Ez a teljesítményteszt az [Azure Security teljesítményteszt v2](overview.md)-vel van összhangban.

Egy leképezést biztosítunk a [CIS Microsoft Azure founds teljesítményteszthez a legújabb verziójú v 1.3.0](https://www.cisecurity.org/benchmark/azure/) és az [Azure Security benchmark v2](overview.md)-ben. Ez segít megérteni, hogy a két viszonyítási pontra vonatkozó javaslatok hogyan illeszkednek egymáshoz. A teljes leképezést [innen](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)töltheti le.

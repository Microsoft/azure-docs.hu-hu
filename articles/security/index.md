# <a name="azure-security-documentation"></a>Az Azure Security dokumentációja

A biztonság minden szempontból az Azure integrált része. Az Azure a globális biztonsági intelligenciából, az ügyfelek által használható kifinomult vezérlőkből és a biztonság szempontjából megerősített infrastruktúrákból származó egyedi biztonsági előnyöket kínál. Ez a hatékony kombináció védelmet biztosít az alkalmazások és az adatok számára, támogatja a megfelelőség felé tett lépéseket, és költséghatékony biztonsági megoldást biztosít a különböző méretű vállalatok számára.

# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>A fenyegetésmodellezési eszköz frissített kiadása: 7.1.51023.1 - 11.1.2018

Amint azt a GA kiadási megjegyzések ben eredetileg [megjegyeztük,](threat-modeling-tool-releases-71509112.md)kiadtunk egy frissítést (7.1.51023.1) a Microsoft Threat Modeling Tool számára, amely megköveteli az előzetes verzió felhasználóitól (preview ügyfelek < 7.1.50911.2) a támogatott GA kiadásra való frissítéshez. Ez a kiadás nem tartalmaz új funkciókat vagy javításokat.

- Az előzetes verzió felhasználói az ügyfél megnyitásakor automatikusan letöltik a frissítést. Ha úgy dönt, hogy nem telepíti az új frissítést, az eszköz előzetes verziója bezárul.
- Az eszköz GA verziójának felhasználóinak el kell dönteniük, hogy szeretnének-e frissíteni.
- Az eszköz 2016-os verziójának felhasználói nem érintettek.

## <a name="feature-changes"></a>Funkcióváltozások

- None

## <a name="system-requirements"></a>Rendszerkövetelmények

- Támogatott operációs rendszerek
  - Microsoft Windows 10
- .NET verzió szükséges
  - .NET 3.5.2
- További követelmények
  - Az eszköz és a sablonok frissítéseinek fogadásához internetkapcsolat szükséges.

# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory tanulmányok 

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

A tanulmányok lehetővé teszik az Azure Data Factory mélyebb szintű felfedezését. Ez a cikk az Azure Data Factory elérhető tanulmányainak listáját tartalmazza.

| **Whitepaper** | **Leírás** |
| --- | --- |
|[Azure Data Factory – Adatintegráció a felhőben](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Ez a dokumentum bemutatja, hogy az Azure Data Factory hogyan teheti lehetővé egy modern adattárház felépítését, lehetővé teszi a fejlett elemzések intelligens SaaS-alkalmazások vezetését, és az SQL Server Integrations Services-csomagok Azure-ba való kiemelését.|
|[Adatáttelepítés a helyszíni relációs adattárból az Azure-ba az Azure Data Factory használatával](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | Ez a dokumentum a meglévő helyszíni relációs adattárraktárból (például Netezza, Oracle, Teradata, SQL server) az Azure-ba (például blob storage vagy Azure Data Lake Storage) az Azure Data Factory használatával történő áttelepítésének összetettségével foglalkozik. A kihívások és az ajánlott eljárások a rugalmasság, a teljesítmény, a méretezhetőség, a felügyelet és a biztonság köréépülnek az Azure Data Factory által az Azure Data Factory azure-ba irányuló big data betöltési úthoz. |
|[SAP-adatintegráció az Azure Data Factory használatával](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | Ez a dokumentum összefoglalja az Azure Data Factory jelenlegi támogatását az SAP-adatintegrációhoz, beleértve a célforgatókönyvet, az SAP-összekötő beállításait és a különböző követelmények összehasonlítását, valamint a Data Factory minden SAP-összekötőjének bevezetését. | 
|[Azure Data Factory: SSIS a felhőben](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| Ez a dokumentum azt, hogy miért szeretné áttelepíteni a meglévő SSIS-számítási feladatokat az Azure Data Factoryba, és foglalkozni a gyakori szempontokés aggályok. Ezután bemutatjuk az Azure-SSIS ir létrehozásának technikai részleteit, majd bemutatjuk, hogyan töltheti fel, futtathatja ki és figyelheti csomagjait az Azure Data Factory-n keresztül olyan eszközökkel, amelyeket valószínűleg ismer, például az SQL Server Management Studio (SSMS). |
|[Azure-adatok gyári – átadási paraméterek](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| Ez a cikk bemutatja, hogyan adja át a paramétereket a folyamat és a tevékenység, valamint a tevékenységek között. |
|[Azure-adatgyár –DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| Ez a tanulmány ismerteti az Azure Data Factory folyamatos integrációjával és üzembe helyezésével kapcsolatos gyakorlati tanácsokat. |

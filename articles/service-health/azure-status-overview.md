# <a name="azure-status-overview"></a>Az Azure állapota áttekintése

[Az Azure-állapot](https://status.azure.com/status/) az Azure-szolgáltatások és -régiók állapotának globális áttekintését biztosítja. Az Azure-állapottal információkat kaphat a szolgáltatás rendelkezésre állásáról. Az Azure-állapot mindenki számára elérhető a szolgáltatás állapotáról beszámoló összes szolgáltatás, valamint a széles körű hatással járó incidensek megtekintéséhez. Ha azonban Ön jelenleg Azure-felhasználó, javasoljuk, hogy használja a személyre szabott élményt az [Azure Service Health szolgáltatásban.](https://aka.ms/azureservicehealth) Az Azure Service Health tartalmazza az összes kimaradást, a közelgő tervezett karbantartási tevékenységeket és a szolgáltatási tanácsokat.

![Az Azure állapota oldal](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Az Azure állapotfrissítései

Az Azure-állapotlap valós időben frissül, ahogy az Azure-szolgáltatások állapota megváltozik. Ha nyitva hagyja az Azure-állapotlapot, szabályozhatja, hogy az oldal milyen sebességgel frissüljön az új adatokkal. A lap tetején láthatja az oldal legutóbbi frissítésének idejét.

![Az Azure állapotfrissítése](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Az Azure állapotelőzményei

Bár az Azure-állapotoldal mindig a legfrissebb állapotinformációkat jeleníti meg, a régebbi eseményeket az [Azure állapotelőzmények lapján](https://status.azure.com/status/history/)tekintheti meg.

A 2019 novemberében közzétett eseményektől kezdve az állapotelőzmények lap a múltban legfeljebb 365 napos eseményeket jelenít meg, beleértve az Azure előzetes kiváltó okát, mérséklését és következő lépéseit.

## <a name="rss-feed"></a>RSS-hírcsatorna

Az Azure-állapot [rss-hírcsatornát](https://status.azure.com/status/feed/) is biztosít az Azure-szolgáltatások állapotában végrehajtott változásokról, amelyekre előfizethet.

# <a name="azure-status-overview"></a>Az Azure állapotának áttekintése

Az Azure-beli [állapot](https://status.azure.com/status/) globális áttekintést nyújt az Azure-szolgáltatások és-régiók állapotáról. Az Azure status használatával információt kaphat a szolgáltatás elérhetőségéről. Az Azure-beli állapot mindenki számára elérhető, hogy megtekintse az összes olyan szolgáltatást, amely jelentést készít a szolgáltatás állapotáról, valamint a széles körű hatással járó incidensekről. Ha Ön jelenleg egy Azure-felhasználó, javasoljuk, hogy használja az [Azure Service Health](https://aka.ms/azureservicehealth)személyre szabott élményt. Azure Service Health tartalmazza az összes kimaradást, a közelgő tervezett karbantartási tevékenységeket és a szolgáltatási tanácsadókat.

![Az Azure állapota oldal](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Azure-állapot frissítései

Az Azure-állapot lap valós időben frissül az Azure-szolgáltatások állapotának változása miatt. Ha megnyitotta az Azure Status (állapot) lapot, beállíthatja, hogy a lap milyen sebességgel frissüljön fel az új adattal. A lap tetején láthatja, hogy mikor frissítették utoljára a lapot.

![Azure-állapot frissítése](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Az Azure állapotelőzményei

Amíg az Azure Status (állapot) lapon mindig a legfrissebb állapotadatok láthatók, a régebbi események az [Azure status History (Azure status előzmények) lapon](https://status.azure.com/status/history/)tekinthetők meg.

A múltban akár 90 napig is megtekintheti az eseményeket, beleértve az előzetes kiváltó okot, a mérséklést és az Azure-ra vonatkozó további lépéseket.

## <a name="rss-feed"></a>RSS-hírcsatorna

Az Azure-beli állapot [egy RSS-hírcsatornát](https://status.azure.com/status/feed/) is biztosít az Azure-szolgáltatások azon állapotának változásairól, amelyekre előfizethet.

# <a name="migration-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Áttelepítési megoldások az Azure VMware megoldású virtuális gépekhez (VM)

Az Azure VMware megoldás használatának egyik leggyakoribb felhasználási esete az adatközpont-kiürítés.  Lehetővé teszi, hogy továbbra is maximalizálja VMware-beruházásait, mivel az Azure VMware-megoldás mindig naprakész marad. Emellett a saját számítási feladatait a natív Azure-szolgáltatások teljes választékával növelheti.  A folyamat első kulcsfontosságú lépése a régi VMware-alapú környezet áttelepítése az Azure VMware megoldásba.

A migrációs partnereink piacvezető migrációs megoldásokkal rendelkeznek a VMware-alapú környezetekben. Világszerte ügyfeleink ezeket a megoldásokat használták az Azure-ba és az Azure VMware-megoldásba való áttelepítésre.

Nem szükséges a VMware HCX áttelepítési eszközként használni, ami azt jelenti, hogy fizikai számítási feladatokat is áttelepíthet az Azure VMware megoldásba. Emellett az Azure VMware-megoldási környezetbe való Migrálás nem igényel ExpressRoute-csatlakozást, ha az nem érhető el a forrás környezetében.  A Migrálás több helyen is elvégezhető, ha úgy dönt, hogy több Azure-régióban is üzemelteti a munkaterheléseket.

További információ ezekről a megoldásokról: [RiverMeadow](https://www.rivermeadow.com/migrating-to-vmware-on-azure).

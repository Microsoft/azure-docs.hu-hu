# <a name="azure-sql-database-features"></a>Az Azure SQL adatbázis-szolgáltatásai

Az Azure SQL Database szolgáltatás közös kódbázist oszt meg az SQL Server legújabb stabil verziójával. A legtöbb szabványos SQL-nyelv, lekérdezésfeldolgozás és adatbázis-kezelési szolgáltatások azonosak az SQL Server és az Azure SQL Database. Az SQL Server ben és az Azure SQL Database összes ízében gyakori funkciók a következők:
  - Nyelvi jellemzők - [A folyamatnyelvi kulcsszavak](https://docs.microsoft.com/sql/t-sql/language-elements/control-of-flow), [kurzorok](https://docs.microsoft.com/sql/t-sql/language-elements/cursors-transact-sql), [adattípusok](https://docs.microsoft.com/sql/t-sql/data-types/data-types-transact-sql), [DML-utasítások](https://docs.microsoft.com/sql/t-sql/queries/queries), [predikátumok](https://docs.microsoft.com/sql/t-sql/queries/predicates), [sorszámok](https://docs.microsoft.com/sql/relational-databases/sequence-numbers/sequence-numbers), tárolt [eljárások](https://docs.microsoft.com/sql/relational-databases/stored-procedures/stored-procedures-database-engine)és [változók vezérlése](https://docs.microsoft.com/sql/t-sql/language-elements/variables-transact-sql).
  - Adatbázis-szolgáltatások - [Automatikus hangolás (tervkényszerítés)](https://docs.microsoft.com/sql/relational-databases/automatic-tuning/automatic-tuning), [Változáskövetés](https://docs.microsoft.com/sql/relational-databases/track-changes/about-change-tracking-sql-server), [Adatbázis-rendezés](https://docs.microsoft.com/sql/relational-databases/collations/set-or-change-the-database-collation), [Tartalmazott adatbázisok](https://docs.microsoft.com/sql/relational-databases/databases/contained-databases), [Tartalmazott felhasználók](https://docs.microsoft.com/sql/relational-databases/security/contained-database-users-making-your-database-portable), Adattömörítés , [Adatbázis-konfigurációs beállítások](https://docs.microsoft.com/sql/t-sql/statements/alter-database-scoped-configuration-transact-sql), Online [indexműveletek](https://docs.microsoft.com/sql/relational-databases/indexes/perform-index-operations-online), [Particionálás](https://docs.microsoft.com/sql/relational-databases/partitions/partitioned-tables-and-indexes)és [Időbeli táblák](https://docs.microsoft.com/sql/relational-databases/tables/temporal-tables) ([lásd az első lépések útmutatót](sql-database-temporal-tables.md)). [Data compression](https://docs.microsoft.com/sql/relational-databases/data-compression/data-compression)
  - Biztonsági funkciók – [Alkalmazásszerepkörök](https://docs.microsoft.com/sql/relational-databases/security/authentication-access/application-roles), [Dinamikus adatmaszkolás](https://docs.microsoft.com/sql/relational-databases/security/dynamic-data-masking) [(lásd az első lépések útmutatóját](sql-database-dynamic-data-masking-get-started.md)), [a Sorszintű biztonság](https://docs.microsoft.com/sql/relational-databases/security/row-level-security)és a Fenyegetésészlelés című témakört az egyes adatbázisok és rugalmas készletek és a felügyelt példányok első lépésekhez vezető [útmutatói.](sql-database-threat-detection.md) [managed instance](sql-database-managed-instance-threat-detection.md)
  - Többmodelles képességek - [Grafikonfeldolgozás](https://docs.microsoft.com/sql/relational-databases/graphs/sql-graph-overview), [JSON-adatok](https://docs.microsoft.com/sql/relational-databases/json/json-data-sql-server) ([lásd az első lépések útmutatóját](sql-database-json-features.md)), [OPENXML](https://docs.microsoft.com/sql/t-sql/functions/openxml-transact-sql), [Térbeli](https://docs.microsoft.com/sql/relational-databases/spatial/spatial-data-sql-server), [OPENJSON](https://docs.microsoft.com/sql/t-sql/functions/openjson-transact-sql)és [XML-indexek](https://docs.microsoft.com/sql/t-sql/statements/create-xml-index-transact-sql).

Az Azure SQL Database kezeli az adatbázisokat, és garantálja azok magas rendelkezésre állását. Egyes funkciók, amelyek hatással lehetnek a magas rendelkezésre állású, vagy nem használható a PaaS-világban korlátozott funkciókkal rendelkeznek az Azure SQL Database. Emellett egyes adatbázis-szolgáltatások típusától függ az Azure SQL Database, amely a létrehozott. Ezeket a szolgáltatásokat az alábbi táblázatok ismertetik. Az Azure SQL Database segítségével létrehozhat egy adatbázist egy [felügyelt példány](sql-database-managed-instance.md)részeként, egyetlen adatbázisként vagy egy rugalmas készlet részeként. Ha további részletekre van szüksége a különbségekről, megtalálhatja őket az [egyes adatbázis és rugalmas készletek vagy](sql-database-transact-sql-information.md) a felügyelt példány külön [oldalain.](sql-database-managed-instance-transact-sql-information.md)

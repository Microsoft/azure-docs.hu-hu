# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>StorSimple a Cloud Solutions szolgáltatói programhoz: gyakori kérdések

A következő kérdések és válaszok, amelyek StorSimple-partnerként működhetnek a StorSimple virtuális tömbök üzembe helyezése vagy kezelése során Azure Portalban.

A kérdések és válaszok a következő kategóriákba vannak rendezve:

 - [A StorSimple és a Cloud Solutions-szolgáltató program (CSP) modellje](#storsimple-and-csp-model)
 - [Virtuális tömb üzembe helyezése és kezelése partnerként](#deploy-and-manage-storsimple-as-a-partner)

## <a name="storsimple-and-csp-model"></a>StorSimple és CSP modell

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Mit jelent a StorSimple a Cloud Solutions Provider (CSP) program részeként?
 A StorSimple for CSP lehetővé teszi, hogy a CSP-partnereink eladják StorSimple virtuális tömböket ügyfeleiknek. Ez a modell lehetővé teszi, hogy a CSP-partnerek a Microsoft felhőalapú szolgáltatásainak közvetlen kiépítés, Számlázás és támogatás mellett a teljes körű ügyfél-életciklust is ellássák. 

További információkért keresse fel a [StorSimple a CSP programban című részt](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-microsoft-365-and-enterprise-mobility-suite-ems"></a>A StorSimple használaton alapul, vagy a Microsoft 365 és a nagyvállalati mobilitási csomag (EMS) alapján?
A StorSimple egy használaton alapuló szolgáltatás. Azok az ügyfelek, akik a CSP útvonalon keresztül szerzik be a StorSimple virtuális tömböt, a virtuális tömb és az Azure használati díját kell megfizetniük, külön sorban, a CSP-számlán. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-microsoft-365-and-the-enterprise-mobility-suite"></a>A StorSimple üzleti modellje hasonló más szolgáltatásokhoz, például a Microsoft 365 és a nagyvállalati mobilitási csomaghoz? 
Igen. A modell egy nagykereskedelmi kedvezmény modell, ugyanúgy, mint az Azure más, használati alapú szolgáltatásai. A StorSimple nagykereskedelmi kedvezménye hasonló a többi Azure-szolgáltatáshoz. Ez a modell lehetővé teszi a CSP-partnereink számára, hogy a StorSimple Virtual Array használatával több és több Azure-beli felhasználást is értékesítsenek.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Mely StorSimple SKU-i érhető el a Launch?
Ugyanaz a SKU, amely elérhető az önálló StorSimple-szolgáltatásokkal. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Van-e különbség a Microsoft által az EA és a StorSimple által közvetlenül megvásárolt StorSimple a CSP részeként?
Nem. A CSP alatt az ügyfelek a CSP-partner által kínált egyéb szolgáltatásokat is vásárolhatják, a StorSimple Virtual array (SVA) szolgáltatást pedig egy számlán keresztül. Ne feledje, hogy az StorSimple 8000 sorozatú fizikai eszköz beszerzéséhez az ügyfeleknek továbbra is az EA-útvonalon kell átesniük. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>Milyen piacokon érhető el a StorSimple a CSP számára?
A CSP StorSimple virtuális tömbje minden olyan piacon elérhető, ahol a StorSimple jelenleg elérhető. További információkért keresse fel azon [régiók listáját, ahol a StorSimple elérhető](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Milyen típusú megoldásokat tud biztosítani a partner a StorSimple és a CSP-vel? 
Több megoldás létezik:

- A partnerek eladják a StorSimple virtuális tömböt. 
- A partnerek a StorSimple és más Azure-szolgáltatások vagy harmadik féltől származó szoftverek köré épülő megoldásokat is képesek biztosítani. További információkért tekintse át a [StorSimple körüli lehetséges megoldásokat](./storsimple-disaster-recovery-using-azure-site-recovery.md).
- Az új üzleti modelleket a felügyelt szolgáltatások nyújtásával is felfedezhetik.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Mik a StorSimple CSP-partnerek számára elérhető ösztönzők? Van-e jogosultsága a CSP programokra vonatkozó ösztönzőknek?
Az ösztönzőkkel kapcsolatos információkért keresse fel a [CSP programra vonatkozó ösztönzőket](https://partner.microsoft.com/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>StorSimple üzembe helyezése és kezelése partnerként

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Hogyan kezelhetem a StorSimple virtuális tömböt a CSP-ben? 
A StorSimple virtuális Array-előfizetéseket adhat hozzá az ügyfelek számlájához a partner centeren keresztül. Emellett a Azure Portal használatával is hozzáadhat felhasználókat az előfizetésekhez.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>A Azure Portal módszer ugyanaz, mint a CSP egyéb szolgáltatásai? 
Igen. A CSP-hez készült StorSimple Virtual Array használatával a partnerek számára a legjobb módszer a Azure Portal eléréséhez, [Ha az](https://partnercenter.microsoft.com/) ügyfeleket és előfizetéseket kezelhetik. A StorSimple-előfizetések kezeléséhez a partnernek vagy az ügyfélnek (a megadott engedélyektől függően) be kell jelentkeznie a Azure Portalba. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>A Microsoft egy új portált szállít a StorSimple for CSP szolgáltatáshoz? 
Nem. A StorSimple virtuális tömb felügyelhető a CSP-ben a Azure Portalon keresztül.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Létrehoztam egy önálló StorSimple-előfizetést az ügyfelem számára. Az eszköz felügyeletéhez az Azure Portalt kell használom? 
Igen. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>A Microsoft-fiók csapata kompenzálja a StorSimple a CSP-n keresztüli értékesítésének eredményeképpen?
Igen. A végfelhasználók számára a partner Sales Executive és a Customer Account csapatának kompenzálása a CSP-eladások esetében történik. A partnereknek el kell végezniük a különböző értékek és ajánlatok értékesítését.

## <a name="support-for-storsimple-and-csp"></a>A StorSimple és a CSP támogatása

### <a name="are-there-any-forums-to-get-additional-support-for-csp-partners-to-deploy-and-manage-storsimple"></a>Vannak olyan fórumok, amelyek további támogatást kapnak a CSP-partnerek számára a StorSimple üzembe helyezéséhez és kezeléséhez?
A StorSimple a CSP-fórumon találhatja meg az egyes gyakran feltett kérdésekre adott válaszokat. Először csatlakoztatnia kell az [Azure Advisors Yammer-csoportot](https://www.yammer.com/azureadvisors) . Ezután keresse meg és csatlakozzon a Group- **StorSimple partner Advisors** szolgáltatáshoz.

### <a name="how-does-the-support-work-for-storsimple-for-csp"></a>Hogyan működik a támogatás a StorSimple for CSP esetében? 
A CSP-ben a StorSimple támogatási modellje megegyezik a CSP-ben lévő többi Azure-szolgáltatással. További információkért keresse fel a [CSP ügyfélszolgálatát](/partner-center/csp-overview).

A CSP-StorSimple kapcsolatos további információkért nyissa meg a következőt:

- [Microsoft Cloud megoldás-szolgáltatói program](https://partner.microsoft.com/Membership)
- [Partnerközpont](https://partnercenter.microsoft.com/)

# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>StorSimple for Cloud Solutions Provider Program: Gyakori kérdések


## <a name="overview"></a>Áttekintés
Az alábbiakban olyan kérdéseket és válaszokat, amelyek storSimple-partnerként rendelkezhetnek, amikor egy StorSimple virtuális tömböt telepít vagy kezel az Azure Portalon.

A kérdések és válaszok a következő kategóriákba sorolhatók:

 - [StorSimple és Cloud Solutions Provider Program (CSP) modell](#storsimple-and-csp-model)
 - [Virtuális tömb telepítése és kezelése partnerként](#deploy-and-manage-storsimple-as-a-partner)
 - StorSimple partnerek támogatása a kriptaszámára

## <a name="storsimple-and-csp-model"></a>StorSimple és CSP modell

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Mit jelent a StorSimple a Cloud Solutions Provider (CSP) program részeként?
 A StorSimple for CSP lehetővé teszi a CSP-partnerek számára, hogy újraértékesítsék ügyfeleiket a StorSimple Virtual Array számára. Ez a modell lehetővé teszi a kripta-partnerek számára, hogy a Microsoft felhőszolgáltatásainak közvetlen kiépítésével, számlázásával és támogatásával birtokolják a teljes körű ügyféléletciklust. 

További információ: [StorSimple in CSP program](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>A StorSimple használatalapú vagy olyan üléslicencekön alapul, mint az Office 365 és az Enterprise Mobility Suite (EMS)?
A StorSimple egy használatalapú szolgáltatás. Azok az ügyfelek, akik a CsP-útvonalon keresztül szerzik be a StorSimple virtuális tömböt, a virtuális tömb és az Azure használatáért fizetnek, a kripta-számlán külön sorelemként számlázva. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>A StorSimple üzleti modell hasonló más szolgáltatások, például az Office 365 és az Enterprise Mobility Suite modelléhez? 
Igen. A modell egy nagykereskedelmi kedvezményes modell, csakúgy, mint az Azure-ban más használatalapú szolgáltatások. A StorSimple nagykereskedelmi kedvezménye hasonló más Azure-szolgáltatásokhoz. Ez a modell lehetőséget biztosít kriptapartnereink számára, hogy többet értékesítsenek és az Azure-felhasználást a StorSimple virtuális tömbön keresztül hajtsák.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Mely StorSimple skus-ok érhetők el a megjelenéskor?
Ugyanazok a sk-ek, amelyek az önálló StorSimple-szolgáltatásokkal érhetők el. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Van-e különbség a StorSimple közvetlenül vásárolt a Microsoft az EA és a StorSimple részeként CSP?
Nem. A csp alatt az ügyfelek a CSP-partner által kínált egyéb szolgáltatásokat is megvásárolhatják a StorSimple virtual array (SVA) mellett egy számla alatt. Vegye figyelembe, hogy a StorSimple 8000 sorozatú fizikai eszköz beszerzéséhez az ügyfeleknek továbbra is át kell mennie az EA útvonalon. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>Mely piacokon érhető el a StorSimple a kriptaindításkor?
A StorSimple virtuális tömb a CSP-hez minden olyan piacon elérhető, ahol a StorSimple ma elérhető. További információt a [StorSimple számára elérhető régiók listájában talál.](https://azure.microsoft.com/regions/services/)

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Milyen megoldásokat tud egy partner a StorSimple és a CSP segítségével? 
Több megoldás is létezik:

- A partnerek viszonteladhatják a StorSimple virtuális tömböt. 
- A partnerek a StorSimple és más Azure-szolgáltatások vagy harmadik féltől származó szoftverek köré épített megoldásokat kínálhatnak. További információkért tekintse át a [StorSimple környékén létező lehetséges megoldásokat.](https://aka.ms/storsimple-build-solutions)
- A felügyelt szolgáltatások nyújtásával új üzleti modelleket is felfedezhetnek.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Milyen ösztönzők érhetők el a StorSimple csp-partnerek számára? Jogosult vagyok-e a CSP-programok ösztönzésére?
Az ösztönzőkkel kapcsolatos információkért látogasson el a [CSP-program ösztönzőire.](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details)


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>A StorSimple üzembe helyezése és kezelése partnerként

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Hogyan felügyelhetem a StorSimple virtuális tömböt a CSP-ben? 
A StorSimple Virtual Array-előfizetéseket a Partnerközponton keresztül veheti fel az ügyfelek fiókjához. Emellett az Azure Portal használatával felhasználókat adhat hozzá az előfizetésekhez.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Az Azure Portal-megközelítés megegyezik a kriptaszolgáltatások kal? 
Igen. A StorSimple virtuális tömb csp, a legjobb módja a partnerek számára az Azure Portal eléréséhez kell a [Partner-központ,](https://partnercenter.microsoft.com/) ahol kezelhetik az ügyfelek és előfizetések. A StorSimple-előfizetések kezeléséhez a partnernek vagy az ügyfélnek (a megadott engedélyektől függően) be kell jelentkeznie az Azure Portalra. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>A Microsoft egy új portált szállít a StorSimple for CSP számára? 
Nem. A StorSimple virtuális tömb a CSP-ben az Azure Portalon keresztül felügyelhető lesz.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Önálló StorSimple-előfizetést létesítettem az ügyfelem számára. Az eszköz felügyeletéhez az Azure Portalon kell használnom? 
Igen. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>A Microsoft-fiók csapata kompenzálva van a StorSimple csp-n keresztül történő értékesítéséből?
Igen. A partner értékesítési vezetője és a végfelhasználó ügyfélszámla-csapata kárpótolja a csp-értékesítéseket. A partnerektől elvárjuk, hogy vezető szerepet vállaljanak differenciált értékük és ajánlataik értékesítésében.

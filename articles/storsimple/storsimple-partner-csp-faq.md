# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>A StorSimple felhőalapú megoldások szolgáltató program: Gyakori kérdések


## <a name="overview"></a>Áttekintés
Az alábbiakban a kérdések és válaszok, lehetséges, hogy a StorSimple-partnerként üzembe helyezésekor vagy kezelése a StorSimple Virtual Array az Azure Portalon.

Kérdések és válaszok a következő kategóriákba vannak rendezve:

 - [A StorSimple és a Felhőbeli szolgáltató Program (CSP) modell](#storsimple-and-csp-model)
 - [Üzembe helyezés és kezelés a virtuális tömb partnerként](#deploy-and-manage-storsimple-as-a-partner)
 - StorSimple partnerek CSP-hez támogatása

## <a name="storsimple-and-csp-model"></a>A StorSimple és a CSP-modell

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Mit jelent szeretné, hogy a StorSimple a Felhőbeli szolgáltató (CSP) program keretében?
 StorSimple CSP-hez lehetővé teszi, hogy a CSP-partnerek továbbértékesítését StorSimple Virtual Array az ügyfeleknek. Ez a modell lehetővé teszi, hogy a CSP-partnerek az ügyféléletciklus végpontok közötti közvetlen kiépítése, számlázás és a támogatás a Microsoft felhőalapú szolgáltatások a. 

További információért ugorjon [StorSimple CSP program](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>Az egy használatalapú vagy munkaállomás-licencek Office 365-höz hasonló alapján a StorSimple és a nagyvállalati mobilitási csomag (EMS)?
A StorSimple szolgáltatás használatalapú. Ügyfelek, akik be kell szereznie a StorSimple Virtual Array keresztül a CSP-útvonal virtuális tömb és az Azure külön sor-elemek a CSP számlán számít használatáért kell fizetnie. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>A StorSimple üzleti modell hasonló a más szolgáltatásokkal, például az Office 365 és a nagyvállalati mobilitási csomaggal van? 
Igen. A modell hasonlóan más használatalapú Azure-szolgáltatások nagykereskedelmi kedvezményt modell használható. A storsimple-höz készült nagykereskedelmi kedvezményt hasonlít, amely más Azure-szolgáltatások. Ez a modell további értékesítésére, és a meghajtó Azure consumption keresztül a StorSimple Virtual Array lehetőséget kínál a CSP-partnereink.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Melyik StorSimple termékváltozatok indításkor érhetők el?
Az azonos termékváltozat, amely a különálló StorSimple szolgáltatások érhetők el. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>A StorSimple a Microsoft nagyvállalati szerződés feltételei szerint közvetlenül megvásárolt és a StorSimple közötti különbség CSP részeként van?
Nem. A CSP ügyfelek is vásárolhatnak más szolgáltatások, amelyeket a CSP-partnerek és a StorSimple virtuális tömb (SVA) egyetlen számlán. Vegye figyelembe, hogy beszerzése a StorSimple 8000 sorozatú fizikai eszköz, az ügyfelek továbbra is szeretné halad át a nagyvállalati szerződéssel rendelkező útvonalat. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>Amely markets érhető el a StorSimple CSP-hez indításakor?
A StorSimple Virtual Array CSP-hez érhető el minden olyan piacon, ahol StorSimple érhető el még ma. További információkért látogasson el a [régióban, ahol a StorSimple elérhető listája](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Milyen típusú megoldások továbbítható egy partnert a StorSimple és a CSP? 
Van több megoldást:

- A partnerek által is az a StorSimple Virtual Array. 
- A partnerek a StorSimple és egyéb Azure-szolgáltatások vagy harmadik féltől származó szoftverek kerülő megoldásokat biztosíthat. További információkért tekintse át néhány lehetséges [létező körül a StorSimple megoldások](https://aka.ms/storsimple-build-solutions).
- Akkor is felfedezhetik új üzleti modellek juttatni a felügyelt szolgáltatások.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Mik azok a StorSimple CSP-partnerek számára elérhető ösztönzők? Vagyok. jogosult vagyok bármely CSP program ösztönzők?
Ösztönzők információkért látogasson el [CSP program ösztönzők ](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>Üzembe helyezése és kezelése a StorSimple-partnerként

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Hogyan felügyelhető a StorSimple Virtual Array in CSP? 
A StorSimple Virtual Array előfizetéseket is hozzáadhat az ügyfelek fiókra a partnerközpontban. Emellett használhatja az Azure Portalon vehet fel felhasználókat az előfizetéseket.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Az Azure Portalon megközelítést ugyanaz, mint más szolgáltatásokat CSP-hez? 
Igen. A StorSimple Virtual Array CSP-hez, a legjobb módszer a partnerek számára érhető el az Azure portal használatával kell lennie a [Partnerközpont](https://partnercenter.microsoft.com/) ahol felügyelheti az ügyfeleket és előfizetéseket. A StorSimple-előfizetések kezelése, a partner vagy az ügyfél (függően jogosultságaitól) kell jelentkezzen be az Azure Portalon. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>A Microsoft szállít új portál storsimple CSP-hez? 
Nem. Fogja tudni kezelni a CSP a StorSimple Virtual Array az Azure Portalon keresztül.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>A StorSimple előfizetési saját ügyfél önálló ellátta I. Rendelkezik az Azure portal használata az eszköz felügyeletéhez? 
Igen. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>A Microsoft Account team kompenzována eredményeként értékesítési storsimple CSP programban?
Igen. A partneri értékesítési vezetői és a felhasználói fiókok ügyfélszolgálata a végfelhasználó ügyfél számára a CSP-értékesítések bude kompenzována. A partnerek vehet igénybe az érdeklődő az eladási azok egyedi érték és ajánlatokat.

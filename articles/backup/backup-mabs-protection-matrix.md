## <a name="azure-expressroute-support"></a>Az Azure ExpressRoute támogatása

Az Azure ExpressRoute-on keresztül is biztonsági másolatot kaphat az adatokról nyilvános társviszony-létesítéssel (a régi áramkörök számára elérhető) és a Microsoft-társviszony-létesítéssel. A privát társviszony-létesítés biztonsági mentése nem támogatott.

Nyilvános társviszony-létesítés: Hozzáférés biztosítása a következő tartományokhoz/címekhez:

* `http://www.msftncsi.com/ncsi.txt`
* `microsoft.com`
* `.WindowsAzure.com`
* `.microsoftonline.com`
* `.windows.net`

A Microsoft társviszony-létesítésével válassza ki a következő szolgáltatásokat/régiókat és a megfelelő közösségi értékeket:

* Azure Active Directory (12076:5060)
* Microsoft Azure-régió (a Recovery Services-tároló helyének megfelelően)
* Azure Storage (a Recovery Services-tároló helyének megfelelően)

További információt az [ExpressRoute útválasztási követelményei című témakörben talál.](https://docs.microsoft.com/azure/expressroute/expressroute-routing)

>[!NOTE]
>A nyilvános társviszony-létesítés elavult az új áramkörök esetében.

## <a name="cluster-support"></a>Fürttámogatás

Az Azure Backup Server a következő fürtözött alkalmazásokban képes megvédeni az adatokat:

* Fájlkiszolgálók

* SQL Server

* Hyper-V – Ha egy Hyper-V fürt et egy kicsinyített MABS védelmi ügynökkel véd, nem adhat hozzá másodlagos védelmet a védett Hyper-V számítási feladatokhoz.

    Ha a Hyper-V rendszert Windows Server 2008 R2 rendszeren futtatja, telepítse a KB [975354](https://support.microsoft.com/kb/975354)frissítésben leírt frissítést.
    Ha a Hyper-V rendszert fürtkonfigurációban futtatja Windows Server 2008 R2 rendszeren, telepítse az SP2 szervizcsomagot és a KB [971394.](https://support.microsoft.com/kb/971394)

* Exchange Server – Az Azure Backup Server képes megvédeni a nem megosztott lemezfürtöket a támogatott Exchange Server-verziók (fürtfolyamatos replikáció) esetén, és képes megvédeni a helyi folyamatos replikációra konfigurált Exchange Server kiszolgálót is.

* SQL Server – Az Azure Backup Server nem támogatja a fürt által megosztott köteteken (CSV) tárolt SQL Server-adatbázisok biztonsági mentését.

Az Azure Backup Server képes megvédeni a fürt számítási feladatok, amelyek ugyanabban a tartományban található, mint a MABS-kiszolgáló, és egy gyermek vagy megbízható tartományban. Ha nem megbízható tartományokban vagy munkacsoportokban szeretné védeni az adatforrásokat, használja az NTLM-et vagy a tanúsítványhitelesítést egyetlen kiszolgálóhoz, vagy csak egy fürt höz.

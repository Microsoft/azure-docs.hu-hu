# <a name="azure-policy-built-in-definitions-for-azure-attestation"></a>Az Azure-igazolás beépített definíciói Azure Policy

Ez az oldal a [Azure Policy](../governance/policy/overview.md) beépített szabályzat-definícióinak indexe az Azure-igazoláshoz. További Azure Policy a más szolgáltatásokhoz készült beépített [definíciók: Azure Policy beépített definíciók](../governance/policy/samples/built-in-policies.md).

Az egyes beépített szabályzat-definíciók neve a Azure Portal házirend-definíciójának nevére hivatkozik. A **verzió** oszlopban található hivatkozás használatával megtekintheti a forrást a [Azure Policy GitHub](https://github.com/Azure/azure-policy)-tárházban.

# <a name="azure-data-science-virtual-machine-release-notes"></a>Az Azure Data Science virtuális gép kiadási megjegyzések

Ebben a cikkben az Azure Data Science virtuális gép kiadásai. A teljes listát az eszközök szerepelnek, valamint a verziószámok, nézd meg [ezt az oldalt](./tools-included.md).

Az ismert hibák és kerülő megoldások [listájában](reference-known-issues.md) olvashat.

## <a name="2020-02-24"></a>2020-02-24

Az [Ubuntu 18.04](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.ubuntu-1804?tab=Overview) és [a Windows 2019-es](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.dsvm-win-2019?tab=Overview) képek adatelemzési virtuálisgép-lemezképei már elérhetők.

- Power BI Desktop 2.73.55xx
- Firefox 69.0.2
- PyCharm 19,2,3
- RStudio 1.2.50xx

### <a name="default-browser-for-windows-updated"></a>Frissített alapértelmezett böngésző a Windows rendszerhez

Korábban az alapértelmezett böngésző az Internet Explorer programra volt állítva. A rendszer most arra kéri a felhasználókat, hogy az első bejelentkezéskor válasszanak ki egy alapértelmezett böngészőt.

## <a name="what-can-i-find-in-the-gallery"></a>Mi található a katalógusban?
Az Azure AI-katalógusban az erőforrásokat, használhatja a saját elemzési megoldásokat fejleszthet különböző tartalmazza.

* **[Kísérletek](gallery-experiments.md)**  – a katalógus tartalmazza a kísérletek fejlesztettek ki az Azure Machine Learning Studio számos. Ezek a gyors proof-of-concept kísérletek, amelyek bemutatják a machine learning-technika, az összetett gépi tanulási problémák teljes által fejlesztett megoldásokat adott tartomány.
* **[A Jupyter Notebooks](gallery-jupyter-notebooks.md)**  – Jupyter notebookok közé tartozik a kód, adatvizualizációkat, és dokumentáció egy interaktív vászonra.
  A katalógus-notebookjait adja meg, oktatóanyagok és részletes magyarázatát az fejlett gépi tanulási technikák és megoldásokat.

## <a name="contribute-to-the-gallery"></a>A katalógus bővítése
Amikor a katalógus Közösség egy tagja válnak, a bejelentkezéshez. Ez lehetővé teszi, hogy a saját katalóguselemek hozzájárul, hogy mások is kihasználhatják a megismert megoldások.

Ahogy mások bővítményeként a hozzájárulását a katalógusban, kövesse a megtekintések és a hozzájárulását a letöltések:

![Ez az elem megoszthatja ismerősök felvételének engedélyezése](./media/gallery-how-to-use-contribute-publish/view-and-download-counts.png)

Jelentkezzen be egy hozzászólás eszközre, például a Disqus, és a megjegyzések vonatkozó értesítések fogadása a hozzájárulások.

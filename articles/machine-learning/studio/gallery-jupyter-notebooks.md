# <a name="discover-jupyter-notebooks-in-the-azure-ai-gallery"></a>Az Azure AI katalógus Jupyter-Notebookjait felderítése

**[A Jupyter Notebooks](https://gallery.cortanaintelligence.com/notebooks)**  a [Azure AI-katalógusban](http://gallery.cortanaintelligence.com) vegye fel a kódot, adatvizualizációk és dokumentációja egy egyszerű, interaktív vászonra.
A katalógus-notebookjait adja meg, oktatóanyagok és részletes magyarázatát az fejlett gépi tanulási technikák és megoldásokat.

## <a name="discover"></a>Ismertetők
  Tallózás a katalógusban notebookokhoz, nyissa meg a [katalógus](http://gallery.cortanaintelligence.com) kattintson **notebookok** a katalógus kezdőlap tetején.

 A **[Jupyter notebookok](https://gallery.cortanaintelligence.com/notebooks)** lap a legnépszerűbb jegyzetfüzet listáját jeleníti meg.
Kattintson a **összes** jegyzetfüzetek megtekintése.
Ezen a lapon tallózhat a katalógusban található összes jegyzetfüzet. Is kereshet és felső belépés keresőkifejezéseket a bal oldalon a szűrési feltételeket kiválasztásával.

 Kattintson bármely notebook számára, hogy nyissa meg a notebook részleteit megjelenítő oldalon, és további információ. Ezen az oldalon megjegyzés, visszajelzés, vagy tegyen fel kérdéseket a megjegyzések szakaszának keresztül. Akkor is megoszthatja azt vagy munkatársai a megosztás képességeivel Linkedinen vagy Twitteren barátaival. Írhat is e-mailt a notebook számára, hogy más felhasználók számára a lap megtekintéséhez meghívása mutató hivatkozást.

![Ez az elem megoszthatja ismerősök felvételének engedélyezése](./media/gallery-how-to-use-contribute-publish/share-links.png)

![A saját Megjegyzések hozzáadása](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="download"></a>Letöltés
A katalógus egy példányát minden olyan jegyzetfüzet letöltheti a Machine Learning Studio munkaterületre.

### <a name="download-from-the-gallery"></a>Töltse le a katalógusból
A katalógus egy jegyzetfüzetet másolatát töltheti le:

1. Nyissa meg a notebook részleteit megjelenítő oldalon a katalógusban
2. Kattintson a **Megnyitás a Studióban**
3. Válassza ki a régiót és a használni kívánt munkaterületet
   
    ![Jupyter notebook megnyitása a katalógusból](./media/gallery-jupyter-notebooks/open-notebook-from-gallery.png)

Amikor rákattint **Megnyitás a Studióban**, a notebook betölti a Machine Learning Studio munkaterületét, és (Ha Ön még nem tette Studio, meg kell adnia jelentkezzen be Microsoft-fiókjához tartozó notebookról is előtt megnyitása másolja a munkaterületre). Annak a notebook később kattintva **notebookok** Studio kezdőlapjának bal oldalán.

### <a name="download-in-machine-learning-studio"></a>Töltse le a Machine Learning Studióban
Is megnyithatja az azonos katalógus notebookok bármelyikét a Machine Learning Studióban használata közben:

1. A Machine Learning Studióban, kattintson a **+ új**
2. Válassza ki **Notebook** -jegyzetfüzeteket a katalógusból listája jelenik meg, hogy közül választhat, vagy a keresőmező segítségével konkrét jegyzetfüzet annak
3. Mutasson az egérmutatóval egy jegyzetfüzetet, és kattintson a **Notebook megnyitása**
   
    ![Nyissa meg a katalógus Notebookban, Machine Learning Studio belül](./media/gallery-jupyter-notebooks/open-notebook-from-studio.png)

A notebook másolatának letöltése és a Jupyter notebookok szakaszban Machine Learning Studio munkaterületét megnyitva.
A notebook listázva lesz, a jegyzetfüzet, valamint a **notebookok** Studio lapján (a a [Studio kezdőlap](https://studio.azureml.net/), kattintson a **notebookok** a bal oldali).

# <a name="azure-machine-learning-curated-environments"></a>Azure Machine Learning kurátori környezetek

Ez a cikk a Azure Machine Learning-ben található kurátori környezeteket sorolja fel. A Azure Machine Learning által biztosított, és a munkaterületen alapértelmezés szerint elérhetővé tett környezetek. Azokat a gyorsítótárazott Docker-rendszerképek teszik lehetővé, amelyek a Azure Machine Learning SDK legújabb verzióját használják, csökkentik a Futtatás előkészítési költségeit, és lehetővé teszik a gyorsabb üzembe helyezési időt. Ezekkel a környezetekkel gyorsan megkezdheti a különböző gépi tanulási keretrendszerek használatát.

> [!NOTE]
> Ez a lista 2020 decemberében frissül. A Python SDK-val a környezetek legnaprakészebb listáját és azok függőségeit veheti igénybe. További információ a [környezetekről szóló cikkben](./how-to-use-environments.md#use-a-curated-environment)található.

# <a name="cheat-sheet-for-an-automated-data-pipeline-for-azure-machine-learning-predictions"></a>Gyorsreferencia az Azure Machine Learning-előrejelzések automatikus folyamatához
A **Microsoft Azure Machine Learning automatizált adatfolyamat-csalási lap** segítségével navigálhat a technológia segítségével, hogy az adatokat a Machine Learning webszolgáltatás, ahol a prediktív elemzési modell által pontozható.

Attól függően, hogy az adatok a helyszínen, a felhőben, vagy a valós idejű streamelés, különböző mechanizmusok állnak rendelkezésre az adatok áthelyezése a webszolgáltatás végpontpontozási.
Ez a csalólap végigvezeti a meghozandó döntéseken, és olyan cikkekre mutató hivatkozásokat kínál, amelyek segíthetnek a megoldás kidolgozásában.

## <a name="download-the-machine-learning-automated-data-pipeline-cheat-sheet"></a>A Machine Learning automatizált adatfolyamat-csalási lapjának letöltése
Miután letöltötte a puskát, kinyomtathatja a bulvárlap méretben (11 x 17 in.).

Töltse le a cheat sheet itt: ** [Microsoft Azure Machine Learning automatizált adatfolyamat cheat sheet](https://download.microsoft.com/download/C/C/7/CC726F8B-2E6F-4C20-9B6F-AFBEE8253023/microsoft-machine-learning-operationalization-cheat-sheet_v1.pdf)**

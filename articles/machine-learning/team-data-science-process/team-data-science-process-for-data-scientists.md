# <a name="team-data-science-process-for-data-scientists"></a>Csoportos adatelemzési folyamat adatszakértők számára

Ez a cikk nyújt útmutatást célok megvalósításához átfogó tudományos megoldások az Azure-technológiák általában használt körét. Végigvezeti:

- az elemzési számítási feladatok ismertetése 
- a csoportos adatelemzési folyamat használata
- Azure Machine Learning segítségével
- az adatátviteli és tárolási alapjai
- data source dokumentáció
- eszközök használatával az adatok feldolgozása

Ezen oktatóanyagok kapcsolódó a csoportos adatelemzési folyamat (TDSP) és a Microsoft és a nyílt forráskódú szoftverek és a eszközkészlettel, amely hasznos lehet a envisioning, végrehajtása és jut a kiváló adatelemzési megoldások.

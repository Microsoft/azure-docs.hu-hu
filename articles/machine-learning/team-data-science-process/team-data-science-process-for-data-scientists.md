# <a name="team-data-science-process-for-data-scientists"></a>Csapatadat-elemzési folyamat adatszakértők számára

Ez a cikk útmutatást nyújt olyan célkitűzések hez, amelyek általában az Azure-technológiákkal átfogó adatelemzési megoldások megvalósításához használatosak. Ön végigvezeti:

- elemzési számítási feladatok megismerése
- a csapatadat-elemzési folyamat használatával
- az Azure Machine Learning használatával
- az adattovábbítás és -tárolás alapjai
- adatforrás-dokumentáció biztosítása
- eszközök használata elemzési feldolgozáshoz

Ezek a képzési anyagok a Team Data Science Process (TDSP) és a Microsoft és a nyílt forráskódú szoftverek és eszközkészletek, amelyek hasznosak az adatelemzési megoldások megálmodozásához, végrehajtásához és biztosításához.

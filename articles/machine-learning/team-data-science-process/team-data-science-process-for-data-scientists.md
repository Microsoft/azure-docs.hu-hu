# <a name="team-data-science-process-for-data-scientists"></a>Csoportos adatelemzési folyamat adatszakértők számára

Ez a cikk az Azure-technológiákkal kapcsolatos átfogó adatelemzési megoldások megvalósítására szolgáló célkitűzésekhez nyújt útmutatást. Ön vezérli a következőket:

- elemzési számítási feladatok ismertetése
- a csoportos adatelemzési folyamat használata
- Azure Machine Learning használata
- az adatátviteli és-tárolási alapjai
- adatforrás-dokumentáció megadása
- eszközök használata az elemzések feldolgozásához

Ezek a tananyagok a csoportos adatelemzési folyamat (TDSP) és a Microsoft, valamint a nyílt forráskódú szoftverek és eszközkészletek esetében hasznosak, amelyek az adatelemzési megoldások tervezéséhez, végrehajtásához és megvalósításához nyújtanak segítséget.

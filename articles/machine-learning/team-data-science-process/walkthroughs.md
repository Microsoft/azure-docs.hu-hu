# <a name="walkthroughs-executing-the-team-data-science-process"></a>A csapatadat-elemzési folyamatot végrehajtó forgatókönyvek

Ezek **az átfogó forgatókönyvek** bemutatják a lépéseket a csapat adatelemzési folyamat az egyes forgatókönyvek. Bemutatják, hogyan kombinálhatja a felhőalapú, a helyszíni eszközöket és a szolgáltatásokat egy **intelligens alkalmazás**munkafolyamatává. A forgatókönyvek az általuk használt **platform** szerint vannak csoportosítva.


## <a name="walkthrough-descriptions"></a>Forgatókönyv leírásai

Az alábbiakban röviden ismertetjük, hogy ezek a forgatókönyv-példák mit nyújtanak a megfelelő platformokon:


- [HDInsight Spark-forgatókönyvek a PySpark és a Scala használatával](walkthroughs-spark.md) Ezek a forgatókönyvek a PySpark és a Scala egy Azure Spark-fürtön prediktív elemzéseket használ.
- [HDInsight-hadoop-forgatókönyvek a Hive használatával](walkthroughs-hdinsight-hadoop.md) Ezek a forgatókönyvek a Hive egy HDInsight Hadoop-fürttel prediktív elemzéssel.
- [Az Azure Data-tó forgatókönyvei az U-SQL használatával](walkthroughs-azure-data-lake.md) Ezek a forgatókönyvek az U-SQL és az Azure Data Lake segítségével prediktív elemzéseket végeznek.
- [SQL Server](walkthroughs-sql-server.md) Ezek a forgatókönyvek az SQL Server, az SQL Server R Services és az SQL Server Python Services segítségével végeznek prediktív elemzést.
- [SQL adattárház](walkthroughs-sql-data-warehouse.md) Ezek a forgatókönyvek az SQL Data Warehouse segítségével prediktív elemzéseket végeznek. 



## <a name="next-steps"></a>További lépések

A csapatadat-elemzési folyamat életciklusának témakörét a [Csapatadat-elemzési folyamat életciklusa című témakörben közöld.](lifecycle.md) Ez az életciklus ismerteti azokat a lépéseket, az elejétől a végéig, amelyeket a projektek általában a végrehajtáskor követnek.

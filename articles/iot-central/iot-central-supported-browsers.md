# <a name="supported-browsers-for-azure-iot-central"></a>Az Azure IoT Central támogatott böngészők

*Ez a cikk operátorokra, fejlesztőkre és rendszergazdákra vonatkozik.*

Az Azure IoT Central érhetők el modern asztali számítógépek, táblagépek és böngészők között. A következő cikk ismerteti a támogatott böngészők listáját. 

Azt javasoljuk, hogy a legfrissebb böngészőben, amely kompatibilis az operációs rendszert használ. A következő böngészők támogatottak:
- A Microsoft Edge (legújabb verzió)
- Safari (csak Mac, legújabb verzió)
- Chrome (legújabb verzió)
- Firefox (legújabb verzió)

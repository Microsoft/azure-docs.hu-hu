# <a name="supported-browsers-for-azure-iot-central"></a>Az Azure IoT Central támogatott böngészők

*Ez a cikk operátorokra, fejlesztőkre és rendszergazdákra vonatkozik.*

Az Azure IoT Central a legtöbb modern asztali számítógépek, táblagépek és a böngészők érhetők el. A következő cikkben ismerteti a támogatott böngészők és a szükséges kapcsolati listáját.  

## <a name="supported-browsers"></a>Támogatott böngészők
Azt javasoljuk, hogy a legfrissebb böngészőben, amely kompatibilis az operációs rendszert használ. A következő böngészők támogatottak:
- A Microsoft Edge (legújabb verzió)
- Safari (csak Mac, legújabb verzió)
- Chrome (legújabb verzió)
- Firefox (legújabb verzió)

## <a name="required-protocols"></a>Szükséges protokollok
Az Azure IoT-központ szükséges, hogy a hálózat a HTTPS és a WebSocket protokoll támogatja a kimenő hálózati kapcsolatot.

# <a name="azure-policy-regulatory-compliance-controls-for-azure-kubernetes-service-aks"></a>Az Azure Kubernetes Service (ak) megfelelőségi szabályozásának Azure Policy

A [Azure Policy szabályozási megfelelősége](../governance/policy/concepts/regulatory-compliance.md) a Microsoft általlétrehozott és kezelt kezdeményezési definíciókat, valamint a különböző megfelelőségi szabványokhoz kapcsolódó megfelelőségi tartományokat és biztonsági ellenőrzéseket biztosít. Ez az oldal az Azure Kubernetes szolgáltatás (ak) megfelelőségi tartományait és biztonsági vezérlőit sorolja fel.

A **biztonsági vezérlők** beépített moduljait egyenként is hozzárendelheti ahhoz, hogy az Azure-erőforrások megfeleljenek az adott szabványnak.

# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Elemzési adatok megtekintése az Azure Web App-erőforrások különböző metrikák

![Web Apps szimbólum](./media/log-analytics-azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Közösségi támogatás az Azure Web App Analytics megoldás át lett helyezve.  
>- A megoldás már nem érhető el az Azure Marketplace-ről, de telepíthető [Azure gyorsindítási sablonok](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) a Közösség által támogatott ahol.
>- Ügyfelek, akik már telepítették a megoldás továbbra is használhatja azt a változások.  
>- A Microsoft javasolja, hogy [Application Insights](../application-insights/app-insights-overview.md) a webalkalmazások figyeléséhez.

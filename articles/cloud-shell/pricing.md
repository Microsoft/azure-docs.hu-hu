# <a name="pricing"></a>Díjszabás
A Bash in Cloud Shell és a Cloud Shell PowerShell az alábbi információk nak vannak alávetve.

## <a name="compute-cost"></a>Számítási költség
Az Azure Cloud Shell az Azure által ingyenesen biztosított gépen fut, de a használatához Azure-fájlmegosztásszükséges. 

## <a name="storage-cost"></a>Tárolási költség
A Cloud Shell új vagy meglévő Azure Files-megosztást igényel a fájlok munkamenetek közötti megőrzéséhez. A tárolás rendszeres költségekkel jár.

Az Azure Files költségeiről itt [olvashat részletesen.](https://azure.microsoft.com/pricing/details/storage/files/)

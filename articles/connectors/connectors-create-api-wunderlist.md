# <a name="monitor-and-manage-wunderlist-by-using-azure-logic-apps"></a>Wunderlist figyelése és kezelése Azure Logic Apps használatával

2020. május 6-án a [wunderlist szolgáltatás leállt](https://www.wunderlist.com/) , és felkérte az ügyfeleket, hogy [migrálják adataikat, és megkezdik a Microsoft használatát a művelet](https://www.wunderlist.com/blog/goodbye-from-wunderlist/)elvégzésére. Az elavult [wunderlist-összekötő](https://docs.microsoft.com/connectors/wunderlist/)lecserélése a [Microsoft to do Connector](https://docs.microsoft.com/connectors/todo/) használatával végezhető el.

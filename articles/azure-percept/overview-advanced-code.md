# <a name="advanced-development-with-azure-percept"></a>Fejlett fejlesztés az Azure Percept
Az Azure Percept a szoftverfejlesztők és az adatszakértők speciális kód-munkafolyamatokat használhatnak a mesterséges intelligencia életciklusának kezeléséhez. Egyre bővülő nyílt forráskódú kódtár használatával mintákat használhatnak a mesterséges intelligencia-fejlesztési utazás megkezdéséhez és a gyártásra kész megoldások létrehozásához.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Ismerkedés a fejlett fejlesztői oktatóanyagokkal

Ez a [Jupyter-jegyzetfüzet](https://github.com/microsoft/Project-Santa-Cruz-Private-Preview/blob/main/Sample-Scripts-and-Notebooks/Official/Machine%20Learning%20Notebooks/Transferlearningusing_SSDLiteV2%20Model.ipynb) elvégzi az adatátvitelt, és egy előre betanított TensorFlow modellt (MobileNetSSDV2Lite) használ a Python AzureML, amely a tálak észlelésére szolgáló egyéni adatkészlettel rendelkezik. Két oktatóanyagi lehetőség áll rendelkezésre, amely végigvezeti Önt a notebookon való együttműködésen, és üzembe helyezi a Bowl-detektort a projekt Santa Cruz fejlesztői készlet:

- [Felhőalapú fejlesztés](./advanced-development-cloud.md): ebben az oktatóanyagban a notebookot a [Azure Machine learning-portálon](https://ml.azure.com) fogja futtatni egy távoli számítási példánnyal.

- [Helyi fejlesztés](./advanced-development-local.md): ebben az oktatóanyagban helyileg fogja futtatni a jegyzetfüzetet a vs Code-ban egy távoli számítási példánnyal.

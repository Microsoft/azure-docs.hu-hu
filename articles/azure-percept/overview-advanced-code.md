# <a name="advanced-development-with-azure-percept"></a>Fejlett fejlesztés az Azure Percept

Az Azure Percept segítségével a szoftverfejlesztők és az adatszakértők speciális kód-munkafolyamatokat használhatnak a mesterséges intelligencia életciklusának kezeléséhez. Egyre bővülő nyílt forráskódú kódtár használatával mintákat használhatnak a mesterséges intelligencia-fejlesztési utazás megkezdéséhez és a termelésre kész megoldások létrehozásához.

## <a name="get-started-with-advanced-development"></a>Ismerkedés a fejlett fejlesztéssel

Tekintse meg az [Azure PERCEPT DK Advanced Development githubot](https://github.com/microsoft/azure-percept-advanced-development) , amely naprakész útmutatást, oktatóanyagokat és példákat tartalmaz a következőkhöz:

- Egyéni AI-modell üzembe helyezése az Azure Percept DK-ben
- Támogatott modell frissítése az adatátviteli tanulással
- És még sok minden más

## <a name="next-steps"></a>Következő lépések

További információ az elérhető [Azure PERCEPT AI-modellekről](./overview-ai-models.md). Ha ezek a modellek egyike sem felel meg az igényeinek, használja a speciális kódot, hogy saját modellt vagy számítógépes jövőképet használjon a Percept DK-be. Ha olyan hozzájárulással rendelkezik, amelyet úgy gondol, hogy segítene másoknak, nyugodtan nyisson meg egy pull-kérést.

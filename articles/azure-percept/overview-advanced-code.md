# <a name="advanced-development-with-azure-percept"></a>Fejlett fejlesztés az Azure Percept
Az Azure Percept a szoftverfejlesztők és az adatszakértők speciális kód-munkafolyamatokat használhatnak a mesterséges intelligencia életciklusának kezeléséhez. Egyre bővülő nyílt forráskódú kódtár használatával mintákat használhatnak a mesterséges intelligencia-fejlesztési utazás megkezdéséhez és a gyártásra kész megoldások létrehozásához.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Ismerkedés a fejlett fejlesztői oktatóanyagokkal

Ismerje meg az összes elérhető [Azure PERCEPT AI-modellt](./overview-ai-models.md).

Tekintse meg az [Azure PERCEPT DK Advanced Development githubot](https://github.com/microsoft/azure-percept-advanced-development) , amely naprakész útmutatást, oktatóanyagokat és példákat tartalmaz a következőkhöz:

* Egyéni AI-modell üzembe helyezése az eszközön
* Modell frissítése már támogatott az adatátviteli tanulással
* És még sok minden más

## <a name="next-steps"></a>Következő lépések

Ismerje meg az összes elérhető [Azure PERCEPT AI-modellt](./overview-ai-models.md). Ha ezek a modellek egyike sem felel meg az igényeinek, nyugodtan használhatja a speciális kód használatát, hogy saját modellt vagy számítógép-jövőképi folyamatot hozzon létre a Percept DK-be, és ha úgy érzi, hogy segítene mások számára, nyisson meg egy lekéréses kérelmet.

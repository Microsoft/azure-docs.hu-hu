# <a name="the-azure-percept-dk-update-experience"></a>Az Azure Percept DK frissítési élménye

Az Azure Percept DK használatával két lehetőség közül választhat a fejlesztői csomag operációs rendszerének és belső vezérlőprogramja (OTA) vagy USB-n keresztül. Az OTA frissítése egyszerűen megtarthatja az eszközök naprakészen tartását, míg az USB-frissítés jó megoldás, ha az OTA nem lehetséges, vagy ha az eszköz gyári beállításait szeretné visszaállítani. Annak biztosítása érdekében, hogy kihasználja az Ön számára legmegfelelőbb frissítési módszert, összeállítottunk egy gyűjteményt, amely segítséget nyújt Önnek. 

- [Azure-IoT Hub beállítása az Azure Percept DK-re való üzembe helyezéshez](./how-to-set-up-over-the-air-updates.md)
- [Az Azure Percept DK frissítése a levegőben](./how-to-update-over-the-air.md)
- [Az Azure Percept DK frissítése USB-kapcsolaton keresztül](./how-to-update-via-usb.md)

# <a name="azure-percept-dk-update-experience-overview"></a>Az Azure Percept DK frissítési élményének áttekintése

Az Azure Percept DK használatával a fejlesztői csomag operációs rendszerét és a belső vezérlőprogramot (OTA) vagy USB-n keresztül is frissítheti. Az OTA frissítése egyszerűen megtarthatja az eszközök naprakészen tartását IoT Hub szolgáltatás [eszközének frissítésével](https://docs.microsoft.com/azure/iot-hub-device-update/) . Az USB-frissítések olyan felhasználók számára érhetők el, akik nem tudnak az OTA-frissítéseket használni, vagy ha az eszköz gyári alaphelyzetbe állítására van szükség. Az Azure Percept DK-eszközök frissítéseinek megismeréséhez tekintse meg az alábbi útmutatókat:

- [Azure-IoT Hub beállítása az Azure Percept DK over-Air (OTA) frissítéseinek üzembe helyezéséhez](./how-to-set-up-over-the-air-updates.md)
- [Azure Percept-alapú DK frissítése over-the-Air (OTA)](./how-to-update-over-the-air.md)
- [Az Azure Percept DK frissítése USB-kapcsolaton keresztül](./how-to-update-via-usb.md)

# <a name="azure-percept-dk-8020-integration-overview"></a>Az Azure Percept DK 80/20 integrációjának áttekintése

Az Azure Percept DK és a audio kiegészítő szolgáltatás a [80/20 T-slot alumínium-építési rendszer](https://8020.net/)integrálására lett tervezve.

## <a name="8020-features"></a>80/20 funkciók

Az Azure Percept DK Carrier Board, az Azure Percept látási eszköz és az Azure Percept audio kiegészítő szolgáltatás beépített 80/20 1010-összekötőket biztosít, amelyek lehetővé teszik az 80/20 Rails nélküli, végtelen csatlakoztatási konfigurációkat. Ez az integráció lehetővé teszi, hogy az ügyfelek és a megoldások építői könnyebben kibővítsék a fogalmakat éles környezetekben.

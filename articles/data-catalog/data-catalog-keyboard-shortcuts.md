# <a name="keyboard-shortcuts-for-azure-data-catalog"></a>Az Azure Data Catalog billentyűparancsai

## <a name="keyboard-shortcuts-for-the-data-catalog-data-source-registration-tool"></a>Az Adatkatalógus adatforrás-regisztrációs eszközének billentyűparancsai

### <a name="general-keyboard-shortcuts"></a>Általános billentyűparancsok
| Művelet | Nyomja le |
| --- | --- |
| Tab ult a lap minden vezérlőelemére |Tab |
| Válasszon egy lehetőséget |Enter vagy Szóköz |
| Lap teljes méretűre állítása |WIN+UP |
| Lap visszaállítása/kis méretre állítása |WIN+LE |
| Oldalmenü megnyitása |ALT+SZÓKÖZ |

### <a name="authentication-page"></a>Hitelesítési lap
| Művelet | Nyomja le |
| --- | --- |
| Bejelentkezés a szolgáltatásba |ALT+S |

### <a name="data-source-selection-page"></a>Adatforrás-kijelölési lap
| Művelet | Nyomja le |
| --- | --- |
| Ugrás a következő oldalra (ha engedélyezve van) |ALT+N |
| A kijelölt szöveg módosítása, ha a fókusz egy csempén van |BALRA, FEL, JOBBRA VAGY LE NYÍL |

### <a name="data-source-connection-page"></a>Adatforrás-kapcsolat lapja
| Művelet | Nyomja le |
| --- | --- |
| Ugrás az előző oldalra (ha engedélyezve van) |ALT+P |
| Válassza a Csatlakozás gombot. |ALT+C |

### <a name="connection-context-page"></a>Kapcsolati környezet lapja
| Művelet | Nyomja le |
| --- | --- |
| A Regisztráció gomb (ha engedélyezve van) |ALT+R |
| Az előnézet teljében jelölőnégyzet (ha elérhető) |ALT+P |
| Adatprofil belefoglalása jelölőnégyzet (ha elérhető) |ALT+D |
| Navigálás az egyes listaelemekre |FEL vagy LE NYÍL |
| Kiszolgálóhierarchia, fanézet csomóscsomópontjának kibontása |JOBBRA NYÍL |
| A regisztrált **elérhető objektumok** vagy objektumok kijelölt **elemeinek áthelyezése** |Szóközt |

### <a name="publish-progress-page"></a>Folyamatjelző lap közzététele
| Művelet | Nyomja le |
| --- | --- |
| Mégse gomb |ALT+C |
| További objektumok regisztrálása gomb |ALT+R |
| Portál megtekintése gomb |ALT+V |

### <a name="publish-progress-page"></a>Folyamatjelző lap közzététele
| Művelet | Nyomja le |
| --- | --- |
| Mentés gomb (ha engedélyezve van) |ALT+S |
| Mégse gomb |ALT+C |

## <a name="keyboard-shortcuts-for-the-data-catalog-portal"></a>Az adatkatalógus-portál billentyűparancsai
| Művelet | Nyomja le |
| --- | --- |
| Navigálás |TAB és SHIFT+TAB |
| Kattintson egy elemre |SZÓKÖZ vagy ENTER |
| Részletezés szakaszba |SZÓKÖZ vagy ENTER |
| Kilépés egy szakaszból |Esc |
| Eszköz rögzítése/rögzítésének feloldása |ALT+P |
| A felderítési oldalon, ha egy eszköz fókuszban van, válassza ki az eszközt |SZÓKÖZ vagy ENTER |
| Eszköz hozzáadása/eltávolítása a többszörös kijelölésből |CTRL+SPACE vagy ENTER |
| A keresési egyezések menü be- és bekapcsolása |ALT+S |
| **Megnyitás a menüben** |ALT+O |
| Fedezze fel a tárolót, ha az eszköz egy tároló |ALT+L |

# <a name="scenario-hdp-upgrade-causes-no-data-in-apache-phoenix-views-in-azure-hdinsight"></a>Forgatókönyv: A HDP-frissítés nem okoz adatokat az Apache Phoenix nézetekben az Azure HDInsightban

Ez a cikk az Azure HDInsight-fürtökkel való kommunikáció során felmerülő problémák hibaelhárítási lépéseit és lehetséges megoldásait ismerteti.

## <a name="issue"></a>Probléma

Az Apache Phoenix nézet nem tartalmaz dátumot a HDP 2.4-ről a HDP 2.5-re való frissítés után.

## <a name="cause"></a>Ok

A nézetek indextáblája (a nézet összes indexe egyetlen fizikai Apache HBase táblában van tárolva) a frissítés során csonkul

## <a name="resolution"></a>Megoldás:

A frissítés után dobja el és hozza létre újra az összes nézetindexet.

# <a name="open-source-big-data-applications-that-work-with-azure-data-lake-storage-gen1"></a>Az Azure Data Lake Storage Gen1 szolgáltatással dolgozó nyílt forráskódú Big Data-alkalmazások

[!INCLUDE [data-lake-storage-gen1-rename-note.md](../../includes/data-lake-storage-gen1-rename-note.md)]

Ez a cikk az Azure Data Lake Storage Gen1-del együttműködő nyílt forráskódú big data-alkalmazásokat sorolja fel. Az alábbi táblázatban szereplő alkalmazások esetében csak a felsorolt terjesztéssel elérhető verziók támogatottak. Ha tudni szeretné, hogy ezek az alkalmazások milyen verziók érhetők el a HDInsight segítségével, olvassa el a [HDInsight-összetevő-verziószámozás című témakört.](../hdinsight/hdinsight-component-versioning.md)

# <a name="azure-spatial-anchors-support-options"></a>Az Azure Spatial Anchors támogatási lehetőségei

## <a name="open-a-tech-support-ticket"></a>Technikai támogatási jegy megnyitása

Technikai támogatási jegy megnyitása az Azure Portal on Azure Spatial Anchors szolgáltatásban:

1. Az [Azure Portal](https://azure.microsoft.com/account/) megnyitásával válassza ki a súgó ikont a felső menüsorban, majd kattintson a **Súgó + támogatás** gombra.

![Az Azure Portal súgója és támogatása](./media/spatial-anchor-support.png)

1. A Súgó + támogatás lapon válassza a **+ Új támogatási kérelem**lehetőséget .

![Az Azure Portal új támogatási kérelme](./media/spatial-anchor-support2.png)

1. A támogatási jegymezők kitöltésekor:

- Problématípus: Technikai
- Szolgáltatás: Térbeli horgonyok

![Az Azure Portal támogatási jegymezői](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Közösségi támogatás

A felhasználói közösség támogatása, kérjük, olvassa el [az Azure Spatial Anchors on Stack overflow.](https://stackoverflow.com/questions/tagged/azure-spatial-anchors)

## <a name="provide-content-feedback"></a>Tartalomvisszajelzést ad

Az egyes tartalmi cikk alján lehetőség van egy GitHub-probléma megnyitására és az Azure Spatial Anchor dokumentációs tartalmára vonatkozó visszajelzések megnyitására.

## <a name="provide-product-feedback"></a>Termékkel kapcsolatos visszajelzés küldése

Ha visszajelzést szeretne adni, megoszthatja az Azure Spatial Anchors szolgáltatással kapcsolatos ötletét vagy javaslatát, vagy szavazhat a mások által benyújtott ötletekről, látogasson el az [Azure Spatial Anchors visszajelzési fórumára.](https://feedback.azure.com/forums/919252-azure-spatial-anchors)

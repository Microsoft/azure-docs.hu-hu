# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Az Azure-webalkalmazás-erőforrások mérőszámait elemző adatok megtekintése

![Web Apps szimbólum](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Az Azure Web App Analytics megoldás a közösségi támogatásba került.  
>- A megoldás már nem érhető el az Azure piactéren, de telepíthető az [Azure Gyorsindítás sablonjaiból](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) , ahol a Közösség támogatja.
>- Azok a felhasználók, akik már telepítették a megoldást, továbbra is használhatják a módosításokat.  
>- A Microsoft azt javasolja, hogy a webalkalmazások figyeléséhez használja a [Application Insights](../../azure-monitor/app/app-insights-overview.md) .

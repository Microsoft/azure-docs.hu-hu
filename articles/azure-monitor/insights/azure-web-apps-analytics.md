# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Az Azure Web App-erőforrások összes metrikája esetén a metrikák elemzési adatainak megtekintése

![Webalkalmazások szimbóluma](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Az Azure Web App Analytics-megoldás átkerült a közösségi támogatásra.  
>- A megoldás már nem érhető el az Azure Marketplace-en, de telepíthető az [Azure Quickstart-sablonokból,](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) ahol a közösség támogatja.
>- Azok a felhasználók, akik már telepítették a megoldást, továbbra is használhatják a módosításokat.  
>- A Microsoft azt javasolja, hogy az [Application Insights](../../azure-monitor/app/app-insights-overview.md) segítségével figyelje a webalkalmazásokat.

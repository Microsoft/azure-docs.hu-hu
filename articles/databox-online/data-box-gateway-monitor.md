# <a name="monitor-your-azure-data-box-gateway"></a>Az Azure Data Box átjáró monitorozása

Ez a cikk ismerteti az Azure Data Box Gateway figyelése. Az eszköz figyeléséhez, használhatja az Azure portal vagy a helyi webes felületén. Az Azure portal használatával eszköz események megtekintése, konfigurálása és a riasztások kezelése és metrikákat tekinthet meg.

Ebben a cikkben az alábbiakkal ismerkedhet meg:

> [!div class="checklist"]
> * Eszköz-események megtekintése és az ezekkel kapcsolatos riasztások
> * Az eszköz a kapacitás és a tranzakciós metrikáinak megtekintése
> * Konfigurálhatja és kezelheti a riasztásokat

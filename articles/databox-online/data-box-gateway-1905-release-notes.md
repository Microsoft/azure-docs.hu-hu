# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Az Azure Data Box Edge és az Azure Data Box átjáró 1905 kibocsátási megjegyzései

## <a name="overview"></a>Áttekintés

A következő kiadási megjegyzések a kritikus fontosságú megoldatlan problémák és az Azure Data Box Edge és az Azure Data Box Gateway 1905 kiadás megoldott problémák azonosításához.

A kibocsátási megjegyzésekben folyamatosan frissülnek, és ahogy ismertté kritikus problémák adódnak. A Data Box Edge/Data Box Gateway, üzembe helyezése előtt alaposan tekintse át a kibocsátási megjegyzésekben található információkat.

Ez a kiadás felel meg a szoftver verziójával:

- **Data Box átjáró 1905 (1.6.887.626)**
- **Data Box Edge 1905 (1.6.887.626)**

> [!NOTE]
> Frissítés 1905 csak a Data Box peremeszközökre általánosan elérhető verziójú szoftver is alkalmazható.

## <a name="whats-new"></a>Újdonságok

- **Mező Programmable Gate Array (FPGA) Technológiáról naplózási fejlesztései** – ebben a kiadásban hajtottunk végre naplózási és riasztási fejlesztések kapcsolatos FPGA. Ez a a Data Box Edge szükséges frissítés, ha az Edge-számítási szolgáltatás használata az FPGA. További információkért lásd: hogyan [alakíthat át adatokat a Data Box Edge Edge számítási](data-box-edge-deploy-configure-compute-advanced.md).

## <a name="known-issues-in-ga-release"></a>Ismert problémák a GA kiadás

Nincsenek új problémák ebben a kiadásban tartozó kiadási. A kiadás minden feljegyzett problémákkal rendelkezik veszi át a korábbi kiadásokban. Ismert problémák listája, lépjen a [ismert problémák a GA kiadás](data-box-gateway-release-notes.md#known-issues-in-ga-release).

# <a name="monitor-your-azure-data-box-edge"></a>Az Azure Data Box Edge figyelése

Ez a cikk ismerteti, hogyan figyelheti az Azure Data Box Edge. Az eszköz figyeléséhez használhatja az Azure Portalon vagy a helyi webes felhasználói felületet. Az Azure Portal használatával megtekintheti az eszközeseményeket, konfigurálhatja és kezelheti a riasztásokat, és megtekintheti a metrikákat. A fizikai eszköz helyi webes felhasználói felületén megtekintheti a különböző eszközösszetevők hardverállapotát.

Ebben a cikkben az alábbiakkal ismerkedhet meg:

> [!div class="checklist"]
> * Eszközesemények és a megfelelő riasztások megtekintése
> * Eszközösszetevők hardverállapotának megtekintése
> * Az eszköz kapacitás- és tranzakciós mutatóinak megtekintése
> * Riasztások konfigurálása és kezelése

## <a name="view-hardware-status"></a>Hardverállapot ának megtekintése

Az eszközösszetevők hardverállapotának megtekintéséhez tegye a helyi webes felhasználói felület alábbi lépéseit.

1. Csatlakozzon az eszköz helyi webes felhasználói felületéhez.
2. Nyissa meg **a Karbantartás > hardver állapotát.** Megtekintheti a különböző eszközösszetevők állapotát.

    ![Hardverállapot ának megtekintése](media/data-box-edge-monitor/view-hardware-status.png)

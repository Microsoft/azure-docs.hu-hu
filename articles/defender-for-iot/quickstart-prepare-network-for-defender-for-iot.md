# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Rövid útmutató: a hálózat előkészítése az Azure Defender számára a IoT üzembe helyezéséhez

## <a name="choose-your-hardware"></a>Válassza ki a hardvert
A IoT készült Azure Defender számos tanúsított készüléket támogat, többek között a következőket:
- Hitelesített előre konfigurált berendezések, amelyeken már telepítve van a szoftver
- Nem konfigurált hitelesített készülékek, amelyeken az érzékelő szoftverét letöltheti és telepítheti
- Helyszíni felügyeleti konzol berendezésének specifikációi. A helyszíni felügyeleti konzol nem érhető el előre konfigurált készülékként. 

További információ: a [hardver specifikációjának útmutatója](https://aka.ms/AzureDefenderforIoTBareMetalAppliance)


## <a name="install-components"></a>Összetevők telepítése
A hardveres összetevők beszerzése és a szoftvercsomag letöltése után az Azure Defender for IoT portálon telepítheti az összetevő szoftvereit. További információ: [telepítési útmutató](https://aka.ms/AzureDefenderforIoTInstallSensorISO).

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Érzékelők beléptetése az Azure Defender for IoT portálra 
 Ez a cikk áttekintést nyújt az Azure Defender IoT-érzékelőkhöz és a helyszíni felügyeleti konzolhoz való üzembe helyezéséről, előkészítéséről és frissítési folyamatairól. Ezt a folyamatot az üdvözli az Azure Defender IoT oldalon és a további lapokon az alábbi utasításokat követve hajthatja végre.

![Képernyőkép az Azure Kezdőlap nézetéről](media/updates/image4.png)

A Kezdőlap megnyitása:

1. Jelentkezzen be az Azure-fiókjába, és keressen rá az Azure Defender for IoT kifejezésre.

1. Az **érzékelő üzembe helyezéséhez** a következőkre van szükség: előre konfigurált hardver vásárlása a nyílról, vagy a saját eszköz vásárlása és az érzékelő szoftver telepítése.
1. **A helyszíni felügyeleti konzol üzembe helyezéséhez** saját eszközt kell vásárolnia, és telepítenie kell a helyszíni felügyeleti konzol szoftverét. További tudnivalók.




## <a name="network-setup"></a>Hálózati beállítás
Miután telepítette a Defendert a IoT rendszerösszetevőkhöz, beállíthatja a hálózati berendezéseket az optimális hálózati figyeléshez. További információt a [hálózati telepítési útmutatóban](https://aka.ms/AzureDefenderForIoTNetworkSetup)talál.

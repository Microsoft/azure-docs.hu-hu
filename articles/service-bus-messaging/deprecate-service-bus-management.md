# <a name="deprecation-of-azure-service-manager-support-for-azure-service-bus-relay-and-event-hubs"></a>Az Azure-beli Service Manager a Azure Service Bus, a Relay és a Event Hubs

Resource Manager új generációs felhőalapú infrastruktúra-verem teljes mértékben lecseréli a "klasszikus" Azure Szolgáltatáskezelési modellt (klasszikus üzembe helyezési modell). Ennek eredményeképpen a klasszikus üzembe helyezési modell REST API-ját, valamint a Service Bus, a Relay és a Event Hubs támogatása 2021. november 1-jén ki leszvezetve. Ezt az elalasztást először a [Microsoft](https://techcommunity.microsoft.com/t5/Service-Bus-blog/Deprecating-Service-Management-support-for-Azure-Service-Bus/ba-p/370909)technikai közösségének bejelentésében jelentettük be, de nemrég úgy döntöttünk, hogy az eredeti bejelentéstől újabb két évvel meghosszabbítjuk az elalasztás idejét. Az egyszerű azonosítás érdekében ezek az API-k az `management.core.windows.net` URI-jukban vannak. Tekintse meg az alábbi táblázatot az elavult API-k listájáért és azok Azure Resource Manager API-verziójukért.

A Service Bus, Relay és Event Hubs folytatásához 2021. október 31-Resource Manager át a Resource Manager-re. Javasoljuk minden olyan ügyfelünknek, akik még mindig régi API-kat használnak a váltáshoz, hogy hamarosan kihasználják az Resource Manager további előnyeit, beleértve az erőforrás-csoportosítást, a címkéket, az egyszerűsített üzembe helyezési és felügyeleti folyamatot, valamint az Azure szerepköralapú hozzáférés-vezérlés (Azure RBAC) használatával való, aprólékos hozzáférés-vezérlést.

További információt a Azure Resource Manager és az Azure Service Manager a [TechNet blogján talál.](https://blogs.technet.microsoft.com/meamcs/2016/12/22/difference-between-azure-service-manager-and-azure-resource-manager/)

A Azure Service Bus, a Relay és Event Hubs API-k Service Manager és Resource Manager api-król az REST API nyújt további információt:

- [Azure Service Bus](/rest/api/servicebus/)
- [Azure Event Hubs](/rest/api/eventhub/)
- [Azure Relay](/rest/api/relay/)

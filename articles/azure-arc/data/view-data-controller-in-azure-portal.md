# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Az Azure arc adatkezelő erőforrásának megtekintése Azure Portal

Miután elvégezte az első [metrikák vagy naplók feltöltését az Azure](upload-metrics-and-logs-to-azure-monitor.md) -ba vagy a [feltöltött használatot](view-billing-data-in-azure.md), láthatja, hogy a Azure Portal az Azure-ban létrehozott Azure arc-adatkezelő erőforrás.

Nyissa meg a Azure Portal a speciális URL-cím használatával:  [https://portal.azure.com](https://portal.azure.com) .

Keresse meg az adatvezérlőt név szerint a keresősáv alatt, és kattintson rá.

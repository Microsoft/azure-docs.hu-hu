# <a name="azure-policy-built-in-definitions-for-azure-arc-enabled-kubernetes"></a>Az Azure arc-kompatibilis Kubernetes beépített definíciói Azure Policy

Ez az oldal [Azure Policy](../../governance/policy/overview.md) beépített szabályzat-definíciók indexe az Azure arc-kompatibilis Kubernetes. További Azure Policy a más szolgáltatásokhoz készült beépített [definíciók: Azure Policy beépített definíciók](../../governance/policy/samples/built-in-policies.md).

Az egyes beépített szabályzat-definíciók neve a Azure Portal házirend-definíciójának nevére hivatkozik. A **verzió** oszlopban található hivatkozás használatával megtekintheti a forrást a [Azure Policy GitHub](https://github.com/Azure/azure-policy)-tárházban.

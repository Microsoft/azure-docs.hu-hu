# <a name="azure-policy-regulatory-compliance-controls-for-azure-arc-for-servers"></a>Az Azure arc kiszolgálókon való megfelelőségének szabályozása Azure Policy

A [Azure Policy szabályozási megfelelősége](../../governance/policy/concepts/regulatory-compliance.md) biztosítja a Microsoft által létrehozott és felügyelt kezdeményezési definíciókat, amelyek a különböző megfelelőségi szabványokhoz kapcsolódó **megfelelőségi tartományokhoz** és **biztonsági ellenőrzésekhez** _készültek_. Ez a lap felsorolja az Azure arc-kompatibilis kiszolgálók **megfelelőségi tartományait** és **biztonsági vezérlőit** (előzetes verzió). A **biztonsági vezérlők** beépített moduljait egyenként is hozzárendelheti ahhoz, hogy az Azure-erőforrások megfeleljenek az adott szabványnak.

# <a name="enabling-synapse-workspace-features-on-a-dedicated-sql-pool-formerly-sql-dw-in-your-region"></a>A Synapse-munkaterület funkcióinak engedélyezése egy dedikált SQL-készleten (korábban SQL DW) a régióban.

## <a name="regions-not-supported"></a>A régiók nem támogatottak 
Az alábbi táblázat összefoglalja azokat a régiókat, ahol a Synapse-munkaterület funkcióinak engedélyezése egy meglévő dedikált SQL-készleten (korábban SQL DW) nem támogatott. Tekintse meg újra, hogy mikor válik elérhetővé a régiója.

\* Azt jelzi, hogy egy adott ütemezés szerint a régió jelenleg nem érhető el.

| Region | Synapse-munkaterület rendelkezésre állása |
|:--- |:--- |:--- |
| Ausztrália 1. középső középső országa |\* |
| Ausztrália 2. középső régiója |\* |
| Kína 2. keleti régiója |\* |
| Kína 2. északi régiója |\* |
| Dél-India |\*|
| Nyugat-India |\*|
| Dél-Korea déli régiója |\* |
| Nyugat-Svájc |\* |
| Egyesült Arab Emírségek északi részén |\*|
 
 
> [!NOTE]
> A fenti régiólista csak olyan régiókra vonatkozik, ahol a Synapse-munkaterület funkcióinak engedélyezése jelenleg nem érhető el.

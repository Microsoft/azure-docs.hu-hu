# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Támogatott nyelvi és futásidejű verziók az Apache Sparkhoz és a függő összetevőkhöz  

Az Apache Spark az Azure Synapse Analytics szolgáltatás támogatja a különböző futtatási idők és a szolgáltatások ez a dokumentum felsorolja a verziókat.

## <a name="spark-version"></a>Spark-verzió

A Spark 2.4 a jelenleg támogatott verzió.

## <a name="operating-system-version"></a>Az operációs rendszer verziója

Az Apache Spark az Azure Synapse-ban az Ubuntu 16.04-es verzióján fut.

## <a name="java-version"></a>Java verzió

Az Apache Spark az Azure Synapse-ban tartalmazza az 1.8.0_222-es Java-verziót

## <a name="scala-version"></a>Scala verzió

Az Apache Spark az Azure Synapse-ban tartalmazza a Scala 2.11.12-t

## <a name="net-core-version"></a>.NET Core verzió

Az Apache Spark az Azure Synapse-ban tartalmazza a .NET Core 3.0-s

## <a name="net-support-for-apache-spark"></a>Az Apache Spark .NET támogatása

Az Azure Synapse .NET-támogatást tartalmaz az Apache Spark 0.5.0-hoz

## <a name="delta-lake"></a>Delta Lake

A Synapse Analytics támogatja a Delta Lake 0.5.0-t

## <a name="python-version"></a>Python-verzió

Az Apache Spark az Azure Synapse-ban a Python 3.6.1-es verziójának telepítését tartalmazza

# <a name="azure-policy-regulatory-compliance-controls-for-azure-key-vault"></a>Azure Policy szabályozási megfelelőségi szabályozása Azure Key Vault

A [Azure Policy szabályozási megfelelősége](../governance/policy/concepts/regulatory-compliance.md) biztosítja a Microsoft által létrehozott és felügyelt kezdeményezési definíciókat, amelyek a különböző megfelelőségi szabványokhoz kapcsolódó **megfelelőségi tartományokhoz** és **biztonsági ellenőrzésekhez** _készültek_. Ez az oldal a Azure Key Vault **megfelelőségi tartományait** és **biztonsági vezérlőit** sorolja fel. A **biztonsági vezérlők** beépített moduljait egyenként is hozzárendelheti ahhoz, hogy az Azure-erőforrások megfeleljenek az adott szabványnak.

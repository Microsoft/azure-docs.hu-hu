# <a name="azure-policy-built-in-definitions-for-azure-container-instances"></a>Azure Policy beépített definíciók a Azure Container Instances

Ez az oldal a Azure Policy [beépített](../governance/policy/overview.md) szabályzatdefiníciók indexe Azure Container Instances. Az egyéb Azure Policy további beépített szolgáltatásaival kapcsolatban lásd a Azure Policy [definíciókat.](../governance/policy/samples/built-in-policies.md)

Az egyes beépített szabályzatdefiníciók neve a szabályzatdefinícióra mutató hivatkozásokat tartalmaz a Azure Portal. A Verzió oszlopban található **hivatkozással** megtekintheti a forrást a [GitHub Azure Policy tárban.](https://github.com/Azure/azure-policy)

# <a name="azure-resource-manager-templates-for-management-features"></a>Azure Resource Manager-sablonok felügyeleti funkciókhoz

Az alábbi táblázat az Azure Resource Manager-sablonokra mutató hivatkozásokat tartalmaz az Erőforrás-kezelő által biztosított funkciókhoz.

| | |
|-|-|
|**Szerepkör-hozzárendelések**||
| [Szerepkör hozzárendelése erőforráscsoporthoz](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-resourcegroup)| Beépített szerepkört rendel egy felhasználóhoz egy meglévő erőforráscsoporthoz. |
| [Szerepkör hozzárendelése meglévő virtuális géphez](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-virtualmachine)| Egy meglévő virtuális gép beépített szerepkört rendel egy felhasználóhoz. |
| [Szerepkör hozzárendelése több virtuális géphez](https://github.com/Azure/azure-quickstart-templates/tree/master/201-rbac-builtinrole-multipleVMs)| Beépített szerepkörökhozzárendelése a felhasználókhoz egynél több virtuális géphez. |
| [Szerepkör hozzárendelése az Azure-előfizetéshez](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/subscription-role-assigment)| Szerepkörhozzárendelés egy felhasználóhoz egy Azure-előfizetéshez. |
|**Szerepkör-definíció**||
| [Egyéni szerepkör-definíció létrehozása](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-role-def)| Új szerepkör-definíciót hoz létre egy Azure-előfizetésben. |
|**Erőforrás zárolása**||
| [Erőforráscsoport zárolása](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-rg-lock-role-assignment)| Létrehoz egy erőforráscsoportot, és **DoNotDelete** zárolást alkalmaz az erőforráscsoportra. A közreműködői szerepkört egy felhasználóhoz rendeli. |

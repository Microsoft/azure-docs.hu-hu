# <a name="azure-cli-samples-for-event-grid"></a>Azure CLI-minták az Event Gridhez

A következő táblázat az Event Griddel kapcsolatos Azure CLI-példákra mutató hivatkozásokat tartalmaz.

## <a name="event-grid-subscriptions"></a>Event Grid-feliratkozások

- [Feliratkozás Azure-előfizetésre](scripts/event-grid-cli-azure-subscription.md) – előfizetés az Azure-előfizetés eseményeire. 
- [Előfizetés blob Storage](scripts/event-grid-cli-blob.md) -ra – előfizetés egy blob Storage-fiók eseményeire. 
- [Előfizetés egyéni témakörre](scripts/event-grid-cli-subscribe-custom-topic.md) – előfizet egy egyéni témakör eseményeire. 
- [Előfizetés erőforráscsoporthoz](scripts/event-grid-cli-resource-group.md) – előfizetés az erőforráscsoport eseményeire. 
- [Előfizetés az erőforráscsoporthoz, és szűrés egy](scripts/event-grid-cli-resource-group-filter.md) erőforrásra – előfizet egy erőforráscsoport eseményeire, és szűri az adott erőforráshoz tartozó eseményeket.

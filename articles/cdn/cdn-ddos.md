# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS Protection

Content delivery network a DDoS protection tervezési biztosít. A globális kapacitás-es támadások elháríthatók, mellett a Azure CDN további a DDoS protection rendelkezik, ahogyan az alábbi nem külön költségek.

## <a name="azure-cdn-from-microsoft"></a>A Microsoft Azure CDN

A Microsoft Azure CDN által védett [Azure alapszintű DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview). Alapértelmezés szerint, és további költségek nélkül integrálva van a Microsoft Platform az Azure CDN-be. A teljes méretezési és az Azure CDN a Microsoft globálisan üzembe helyezett hálózati kapacitását biztosít a hálózati forgalmat – a megfigyelési és a valós idejű kockázatcsökkentési keresztül közös réteg támadások elleni védelmet. Alapszintű DDoS elleni védelem is lelt ellen a leggyakoribb, milyen gyakran jelentkezik a réteg 7 DNS lekérdezés árvíz és a 3. rétegbeli és 4-es támadások, a cél CDN-végpontok. Ez a szolgáltatás egy jól bevált követése rekordot történő védelmét a Microsoft nagyvállalati és fogyasztói szolgáltatások nagy méretű támadások ellen is tartalmaz.

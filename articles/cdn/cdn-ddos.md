# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS védelem

A tartalomkézbesítési hálózat szándékosan biztosít DDoS-védelmet. A mennyiségi támadások globális elnyelési képessége mellett az Azure CDN további DDoS-védelemmel is rendelkezik az alábbiak szerint, többletköltség nélkül.

## <a name="azure-cdn-from-microsoft"></a>Azure CDN a Microsofttól

A Microsoft Azure CDN-jét az [Azure Basic DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview)védi. Alapértelmezés szerint integrálva van az Azure CDN-be a Microsoft platformról, és további költségek nélkül. A Microsoft globálisan telepített hálózatából származó Azure CDN teljes körű és kapacitása védelmet nyújt a hálózati réteg gyakori támadásaival szemben a forgalom folyamatos figyelése és a valós idejű megoldások révén. Az alapvető DDoS-védelem a CDN-végpontokat célzó leggyakoribb, gyakran előforduló 7. Ez a szolgáltatás bizonyítottan jó eredményeket ért el a Microsoft vállalati és fogyasztói szolgáltatásainak a nagyszabású támadásokkal szembeni védelmében is.

## <a name="azure-cdn-from-verizon"></a>Azure CDN a Verizontól

A Verizon Azure CDN-jét a Verzion saját fejlesztésű DDoS-kockázatcsökkentő platformja védi. Alapértelmezés szerint a Verizon azure CDN-be van integrálva, és további költségek nélkül. Alapvető védelmet nyújt a cdn-végpontokat célzó leggyakoribb, gyakran előforduló 7.

## <a name="azure-cdn-from-akamai"></a>Azure CDN az Akamai-tól

Az Akamai Azure CDN-t az Akamai saját fejlesztésű DDoS-kockázatcsökkentő platformja védi. Alapértelmezés szerint az Akamai azure-CDN-be van integrálva, és további költségek nélkül. Alapvető védelmet nyújt a cdn-végpontokat célzó leggyakoribb, gyakran előforduló 7.

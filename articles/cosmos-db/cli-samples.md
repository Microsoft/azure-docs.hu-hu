# <a name="azure-cli-samples-for-azure-cosmos-db-sql-core-api"></a>Azure CLI-minták az Azure Cosmos DB SQL (Core) API-hoz

Az alábbi táblázat az Azure Cosmos DB SQL (Core) API-hoz készült Azure CLI-parancsfájlok mintájára mutató hivatkozásokat tartalmaz. Az Azure Cosmos DB CLI összes parancsának referencialapjai az [Azure CLI-referenciadokumentumban](/cli/azure/cosmosdb)érhetők el. Az Azure CLI-mintákat más Azure Cosmos DB API-k: [Cassandra API,](cli-samples-cassandra.md) [MongoDB API,](cli-samples-mongodb.md) [Gremlin API](cli-samples-gremlin.md)és [Table API.](cli-samples-table.md) Az Azure Cosmos DB CLI parancsfájlminták megtalálhatók az [Azure Cosmos DB CLI GitHub-tárházban.](https://github.com/Azure-Samples/azure-cli-samples/tree/master/cosmosdb)

# <a name="azure-cli-samples-for-azure-cosmos-db-sql-core-api"></a>Azure CLI-minták a Azure Cosmos DB SQL (Core) API-hoz

Az alábbi táblázat a Azure Cosmos DB SQL (Core) API-hoz készült Azure CLI-szkriptekre mutató hivatkozásokat tartalmaz. Az összes Azure Cosmos DB CLI-parancsra vonatkozó hivatkozási lapok az [Azure CLI-referenciában](/cli/azure/cosmosdb)érhetők el. Az Azure CLI-minták más Azure Cosmos DB API-kkal kapcsolatban lásd: [Cassandra API](cli-samples-cassandra.md), [MongoDB API](cli-samples-mongodb.md), [Gremlin API](cli-samples-gremlin.md)és [Table API](cli-samples-table.md). A CLI-szkriptek összes Azure Cosmos DB a [Azure Cosmos db CLI GitHub-tárházban](https://github.com/Azure-Samples/azure-cli-samples/tree/master/cosmosdb)található.

| [Átviteli sebesség módosítása](scripts/cli/sql/throughput.md?toc=%2fcli%2fazure%2ftoc.json) | Az RU/s frissítése egy adatbázison és tárolón.|

| [A fiókok kulcsai és a kapcsolatok karakterláncai](scripts/cli/common/keys.md?toc=%2fcli%2fazure%2ftoc.json) | Fiókok kulcsai, csak olvasható kulcsok listázása, kulcsok újragenerálása és a kapcsolatok listázása.|

# <a name="azure-cli-samples-for-azure-cosmos-db-core-sql-api"></a>Azure CLI-minták az Azure Cosmos DB Core (SQL) API-hoz

Az alábbi táblázat az Azure Cosmos DB Core (SQL) API-hoz készült Azure CLI-szkriptekre mutató hivatkozásokat tartalmaz. Az összes Azure Cosmos DB CLI-parancsra vonatkozó hivatkozási lapok az [Azure CLI-referenciában](/cli/azure/cosmosdb)érhetők el. Az Azure CLI-minták más Azure Cosmos DB API-kkal kapcsolatban lásd: [Cassandra API](cli-samples-cassandra.md), [MongoDB API](cli-samples-mongodb.md), [Gremlin API](cli-samples-gremlin.md)és [Table API](cli-samples-table.md). A CLI-szkriptek összes Azure Cosmos DB a [Azure Cosmos db CLI GitHub-tárházban](https://github.com/Azure-Samples/azure-cli-samples/tree/master/cosmosdb)található.

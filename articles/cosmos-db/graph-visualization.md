# <a name="visualize-your-graph-data-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>A graph-adatok megjelenítése az Azure Cosmos DB Gremlin API Vizualizáció megoldásokkal

A következő megoldások által ajánlott a [Apache Tinkerpop közösségi](http://tinkerpop.apache.org/#poweredby) a Gremlin graph vizualizációt.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Linkurious embléma](./media/graph-visualization/linkurious-logo.jpg)

Linkurious vállalati graph technológiával és adatmegjelenítés összetett adatkészletek információkká interaktív vizuális hálózatokat használ. A platform az adatforrásokhoz csatlakozik, és lehetővé teszi, hogy kutató több milliárd entitások és kapcsolatok zökkenőmentesen navigálhat. Ez egy új lehetőség anélkül, hogy a lekérdezések vagy táblák munkaórák gyanús kapcsolatok észlelése.

Az interaktív vizualizációs felületének Linkurious vállalati segítségével csatlakoztatott adatokat egyszerű megoldást kínál. Keresse meg az adott entitásokhoz, bontsa ki a tárhat fel rejtett kapcsolatokat és a alkalmazni, összetett hálózatok untangle tetszőleges elrendezések történő csatlakozást. Linkurious már kompatibilis az Azure Cosmos DB Gremlin API-hoz. Teljes körű graph Vizualizáció szituációkra, és támogatja az írási és olvasási képességek a felhasználói felületen. 

![Linkurious bemutató](./media/graph-visualization/linkurious-demo.gif)

**Következő lépések:** Kérheti egy [bemutatót az Azure Cosmos DB-vel Linkurious](https://linkurio.us/contact/).

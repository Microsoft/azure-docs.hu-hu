# <a name="azure-cosmos-db-query-cheat-sheets"></a>Az Azure Cosmos DB lekérdezési adatlapok

A **Azure Cosmos DB lekérdezési adatlapok** segítségével gyorsan írási lekérdezések az adatok egyszerű nyomtatás PDF referencia-adatlapok gyakori adatbázis-lekérdezések, műveletek, függvények és operátorok megjelenítésével. A adatlapok részletes információ az SQL, a MongoDB, a tábla és a Gremlin API-kat tartalmazza. 

A4 méretű vagy A3 méretű letöltés közül választhat. 

## <a name="letter-sized-cheat-sheets"></a>A4 méretű adatlapok

Töltse le a [Azure Cosmos DB lekérdezési a4 adatlapok](https://go.microsoft.com/fwlink/?LinkId=623215) Ha fog Nyomtatás a4-es méretű papírra (8.5 "x 11").

![Az Azure Cosmos DB lekérdezési adatlapok – a4, az SQL API-t, a JavaScript, a MongoDB, a Gremlin és a Table API-lekérdezések és funkciók](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png)

## <a name="oversized-cheat-sheets"></a>Túlméretes adatlapok
Töltse le a [lekérdezést az Azure Cosmos DB A3 méretű adatlapok](https://go.microsoft.com/fwlink/?linkid=870413) Ha fog egy rajzgép vagy nagy méretű nyomtató operátorral A3 méretű papíron (11.7 "x 16.5-ös").

![Az Azure Cosmos DB lekérdezési adatlapok - A3 méretű, az SQL API-t, a JavaScript, a MongoDB, a Gremlin és a Table API-lekérdezések és funkciók](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-a3.png)

Az Azure Cosmos DB CLI összes parancsának referencialapjai az [Azure CLI-referenciadokumentumban](/cli/azure/cosmosdb)érhetők el. Az Azure Cosmos DB CLI parancsfájlminták megtalálhatók az [Azure Cosmos DB CLI GitHub-tárházban.](https://github.com/Azure-Samples/azure-cli-samples/tree/master/cosmosdb)

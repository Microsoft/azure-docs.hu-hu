# <a name="co-sell-partner-engagement"></a>Együttes értékesítési partnerek bevonása

Ha Ön egy partner, a Microsoft közös értékesítéssel foglalkozó programjával közvetlenül dolgozhat a Microsofttal közös értékesítési lehetőségekkel. A program segítséget nyújt partnereinknek az ügyfelek jelentős közösségének elérésében, és új értékesítési lehetőségek létrehozásában, amelyek segíthetnek az üzleti növekedés felgyorsításában.

## <a name="partner-selling-motions"></a>Partner-értékesítési mozgások

A Microsoft három partner-értékesítési indítványt kínál:

* *Értékesítés partneren*keresztül: a Microsoft-technológiákat a saját értékesítők használatával értékesítheti.
* *Szolgáltatások közös értékesítése*: az értékesítők és a partnerek együttműködve értékesítik a projektjét vagy a felügyelt szolgáltatásokat a Microsoft Technology Embedded szolgáltatással.
* *IP-közös értékesítés*: az értékesítők és a partnerek együttműködve értékesítik az ismételhető, csomagolt IP-megoldást. *

\*Ezen IP-címek közös értékesítésével kapcsolatos megoldások némelyike további ösztönzőket is kaphat, amikor a Microsoft Field Sales csapatával sikeresen eladják és telepítették azokat.

# <a name="how-to-sell-your-saas-offer"></a>Az SaaS-ajánlat értékesítése

Ez a cikk azokat a további lehetőségeket ismerteti, amelyekkel kiválaszthatja, hogy a Microsofton keresztül a szolgáltatott szoftvert (SaaS) ajánlja-e. Az ajánlatot a Microsofttal együtt értékesítheti, a [felhőalapú megoldás-szolgáltatók (CSP) programon](cloud-solution-providers.md)keresztül, vagy mindkettőn keresztül értékesítheti.

## <a name="co-sell-with-microsoft"></a>Együttes értékesítés a Microsofttal

A **Microsofttal való közös értékesítésről** szóló információk megadása teljes mértékben nem kötelező. Azonban a _közös értékesítésre kész_ és az _IP közös értékesítésre kész_ állapot elérésére van szükség. A Microsoft értékesítési csapatai ezeket az információkat használják a megoldással kapcsolatos további információkra, amikor kiértékelik az igényeinek megfelelő megoldást. Az ezen a lapon megadott információk nem érhetők el közvetlenül az ügyfeleknek.

A **közös értékesítés a Microsoft** lapon való konfigurálásával kapcsolatos részletekért és útmutatásért lásd: [közös értékesítés lehetőség a kereskedelmi piactéren](commercial-marketplace-co-sell.md).

## <a name="resell-through-csps"></a>Viszonteladás CSP-ken keresztül

Ha úgy dönt, hogy elérhetővé teszi az ajánlatot a Cloud Solution Provider (CSP) programban, a kriptográfiai szolgáltatás a terméket egy csomagban lévő megoldás részeként értékesítheti ügyfelei számára. További információ az ajánlatnak a CSP programon keresztüli viszonteladásáról, valamint az **viszonteladás a CSP** -n keresztül történő konfigurálásának lépésenkénti lépéseiről: [Cloud Solution Provider program](cloud-solution-providers.md).

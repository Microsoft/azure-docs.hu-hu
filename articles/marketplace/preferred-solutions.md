# <a name="preferred-solutions-in-microsoft-appsource-and-azure-marketplace"></a>Előnyben részesített megoldások a Microsoft AppSource és az Azure Marketplace-en

A Microsoft által előnyben részesített megoldás egy olyan felhőalapú alkalmazás, amely a minőség, a teljesítmény és az ügyfelek igényei szerint van kiválasztva egy adott iparági vertikális vagy megoldási területen. A Microsoft Experts csapata a konkrét bevált kompetenciákkal és képességekkel rendelkező partnerek megoldásait ellenőrzi. Ezek a megoldások a kereskedelmi piactér online üzleteiben, az Azure Marketplace-en és a AppSource, valamint a Azure Portal találhatók meg. Az iparági vertikális felderíti az AppSource preferált megoldásait. Az Azure Marketplace-en előnyben részesített megoldások és a Azure Portal a horizontális megoldási kategóriákban is felderíthető.

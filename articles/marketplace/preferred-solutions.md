# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Előnyben részesített megoldások az Azure Piactéren és az AppSource-ban

A Microsoft által előnyben részesített megoldás egy felhőalapú alkalmazás, amelyet a minősége, a teljesítménye és az ügyféligények kielégítésére való képessége érdekében választottak ki egy adott iparág vertikális vagy megoldási területén. A Microsoft szakértőiből álló csapat speciális, bizonyított kompetenciákkal és képességekkel rendelkező partnerek megoldásait érvényesíti. Ezek a megoldások a felhőbeli piactér kirakataiban, az Azure Marketplace-en és az AppSource-ban, valamint az Azure Portalon is megjelennek. Az AppSource-on előnyben részesített megoldásokat az iparági vertikális októl fedezhetifel. Az Azure Marketplace-en és az Azure Portalon előnyben részesített megoldások vízszintes megoldáskategóriákközött fedezhetők fel.

# <a name="create-power-bi-app-offer-technical-assets"></a>A Power BI alkalmazás ajánlat műszaki adategységek létrehozása

Az elsődleges ajánlat típus szükséges technikai eszköz egy Power BI-alkalmazást. A Power BI alkalmazás elsődleges adatkészleteket, jelentések vagy irányítópultok, és nem kötelező csatlakoztatott szolgáltatások és beágyazott adatkészletek gyűjteménye. Az eszköz típusa, korábbi nevén az eddig egy [tartalomcsomag](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction). Az ilyen típusú alkalmazás fejlesztésével kapcsolatos további információkért lásd: [Mik azok a Power bi-ban sablon alkalmazások?](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtain-an-installation-url"></a>Telepítési URL-cím beszerzése

Csak hozhat létre egy Power BI-alkalmazást a [Power BI szolgáltatás](https://powerbi.microsoft.com/). A kezdéshez, jelentkezzen be egy [Power BI Pro-licenc](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Miután létrehozta és service-alkalmazás tesztelése a Power bi-ban, mentse az alkalmazás telepítési URL-CÍMÉT, amely a Power BI-hoz létre. Végül adja meg az URL-címet a a [technikai információ](./cpp-technical-info-tab.md) a Cloud Partner Portalon jelentkezési űrlapot lapján.
 

## <a name="next-steps"></a>További lépések

Miután létrehozta és tesztelte a Power BI alkalmazást, és a kapcsolódó ajánlat létrehozott, [a Power BI alkalmazás-ajánlat közzététele](./cpp-publish-offer.md).

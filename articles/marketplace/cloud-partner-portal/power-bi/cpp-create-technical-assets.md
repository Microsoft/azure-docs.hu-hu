# <a name="create-power-bi-app-technical-assets"></a>Technikai eszközök Power BI-alkalmazás létrehozása

Az elsődleges műszaki eszköz kell ajánlat ilyen van egy Power BI alkalmazás, amely egy elsődleges adatkészlet gyűjteménye, jelentés és irányítópulton és választható csatlakoztatott szolgáltatások és beágyazott adatkészletek (vagy). (A power BI-alkalmazások az eszköz típusa, korábbi nevén alakulása [tartalomcsomagok](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction).)  Ezek az alkalmazások fejlesztésével kapcsolatos részletes információkért lásd: [Power BI-alkalmazások](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtaining-an-installation-url"></a>Telepítési URL-cím beszerzése

Power BI-alkalmazás csak a releváns létrehozásához a [Power BI szolgáltatásban](https://powerbi.microsoft.com/) és a bejelentkezéshez szükséges egy [Power BI Pro-licenccel](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Service-alkalmazás létrehozása és tesztelése a Power bi-ban, után mentse az alkalmazás telepítési URL-cím, amely a Power BI-hoz létre.  Ezt az URL-címet meg kell adnia a [technikai információ lapon](./cpp-technical-info-tab.md) , a Cloud Partner Portalon jelentkezési űrlapot.
 

## <a name="next-steps"></a>További lépések

Miután létrehozott és tesztelni a Power BI-alkalmazás és a kapcsolódó ajánlat létrehozása, [a Power BI-ajánlat közzététele](./cpp-publish-offer.md).

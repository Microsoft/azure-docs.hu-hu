# <a name="power-bi-apps-contacts-tab"></a>A Power BI alkalmazások ügyfelek lap

Használja a **névjegyek** lapján a **új ajánlat** lapot, hogy biztosítsa a műszaki és felhasználói erőforrások támogatása az ajánlatot. Két szakaszra osztottuk: **Mérnöki** és **ügyfél-támogatási**.

![Ügyfelek lap](media/contacts-tab.png)


## <a name="contacts-fields"></a>Kapcsolatba lép a mezők 

A következő táblázat ismerteti a nevét, és ezek a mezők célját. 

|   Mező               |   Leírás                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *Engineering Contact* | Appsource-ban és a szervezet között műszaki ügyfélként szolgál    |
| **Name (Név)**              | A személy vagy csoport, amely technikai és műszaki támogatási neve |
| **E-mail**             | A technikai kapcsolattartó e-mail-címe                                  |
| **Telefonszám**             | A technikai támogatási telefonszám                                       |
| *Támogatási kapcsolattartó*     | Appsource-on belüli ügyfelek által megnyitott támogatási jegyeket kap            |
| **Name (Név)**              | A személy vagy csoport, amely ügyfél-támogatási neve              |
| **E-mail**             | Ügyfél-támogatási e-mail-címe                                        |
|  |  |

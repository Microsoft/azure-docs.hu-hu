# <a name="container-publishing-prerequisites"></a>Tároló közzétételi Előfeltételek

Ez a cikk bemutatja a tároló az ajánlat az Azure piactér közzétételi előfeltételeit.  Ha még nem tette meg, tekintse át a [Containers ajánlat közzétételi útmutató](../../marketplace-containers.md).

## <a name="publishing-prerequisites"></a>Közzétételi Előfeltételek

Egy új tárolórendszerképet a közzétételhez meg kell megfelelnek a következő előfeltételeknek:

- A Cloud Partner portálra való hozzáférést. További információ az Azure Marketplace és AppSource közzétételi útmutatójában talál.
- Szerződés az Azure Marketplace használati feltételei
- A tároló műszaki eszköz az Azure Container Registry üzemeltetéséhez.
- Rendelkezik a tároló metaadatait, készen áll a használatra. Ha például a következő nem teljes listája:
  - Cím
  - Egy leírást (HTML formátumban)
  - Emblémakép (PNG formátumban), valamint ezek a rögzített lemezkép mérete: 40 x 40 képpont, 90 x 90 képpont, 115 x 115 képpont legyen, és 255 x 115 képpont.
- A *használati* és a egy *adatvédelmi szabályzat* utasítások
- Container megoldás dokumentációja
- Támogatási kapcsolattartók

# <a name="azure-application-support-tab"></a>Az Azure application támogatási lapja

A támogatás lap használatával biztosíthat a termékgondozó csoportja és a felhasználói támogatási információkat.
 
## <a name="identify-contacts-and-websites"></a>Névjegyek és webhelyek azonosítása

Alapján a következő képernyőfelvétel-készítés használatával adja meg a szükséges információkat. A piros csillaggal (*) fűzött összes mezőnevek kell végrehajtani.

  ![Támogatási űrlap ajánlat](./media/azureapp-support-tab.png)

1. A **Engineering Contact**, adjon meg információt a következő mezőket:

   - **Név\***  – adja meg az engineering contact az alkalmazás nevét. Ez a kapcsolat műszaki kommunikáció fog kapni a Microsoft.
   - **E-mailek\***  – adja meg az engineering contact e-mail-címét.
   - **Telefonos\***  – adja meg az engineering contact telefonszámát.
  
2. A **támogatási kapcsolattartó**, adjon meg információt a következő mezőket:

   - **Név\***  – adja meg az alkalmazás a támogatási kapcsolattartó nevét. Ez a kapcsolat támogatási kapcsolatos üzeneteket fog kapni a Microsoft.
   - **E-mailek\***  – adja meg a támogatási kapcsolattartó e-mail-címe az alkalmazás számára.
   - **Telefonos\***  – adja meg a támogatási kapcsolattartó telefonszáma.
 
3. A **URL-címeket támogatja a**, adja meg a megfelelő támogatási URL-címeket, amelyek a Microsoft fogja használni, amikor az ügyfelek nyissa meg a támogatási jegyeket. Az URL-címek Azure nyilvános vagy az Azure Government Cloud lesz.
4. Válassza ki **mentése** befejezése.

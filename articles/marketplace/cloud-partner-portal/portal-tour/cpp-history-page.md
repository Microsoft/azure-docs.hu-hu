# <a name="audit-history-page"></a>Naplózási előzmények oldal

A [**Naplózási előzmények**](https://cloudpartner.azure.com/#history) lap az összes piactéri ajánlat közzétételi eseményeinek listáját jeleníti meg.  Az oszlopok közé tartozik az esemény típusa, az előfordulási dátumidő, az ajánlat neve, a közzétevő és a műveletet kezdeményező felhasználó.  Az eredmények oszlop szerint rendezhetők vagy szűrhetők egy megadott karakterláncon.  Ez az oldal lehetővé teszi a felhasználó számára, hogy letöltse az aktuális naplózási lista vesszővel tagolt értékfájlját (.csv).

# <a name="create-azure-marketplace-and-appsource-offers"></a>Hozzon létre az Azure Marketplace és AppSource ajánlatok

Egyik alapvető célja a Cloud Partner portálra engedélyezéséhez a kiadók számára, hozzon létre (és tegye közzé) kínál, a Microsoft Azure és az appsource-ban Piacterein.  Jelölje ki a kívánt ajánlat a következővel kezdődik: mindig ezt a műveletet a [új ajánlat menü](../portal-tour/cpp-new-offer-menu.md).  A válasz, a megfelelő **új ajánlat** lap jelenik meg, hogy az ajánlat típusát.  Például az alábbi képen látható, az alapértelmezett **új ajánlat** oldal egy Azure-alkalmazás esetén.

![Új ajánlat alapértelmezett lap](./media/new-offer-page.png)

Nincsenek elérhető a vízszintes menüsávon, felül szerepel ezen a lapon megjelenik a két lap beállításokat: 
- **Editor** lap – lehetővé teszi, hogy az adatok és az eszközök az új ajánlat példány feltöltése.  Ezen a lapon alapértelmezés szerint megjelenik.
- **Állapot** lap – biztosít a közzétételi állapot és az érvényesítés, és nézze problémákat. 

Amikor létrehoz egy ajánlatot, használja a **szerkesztő** fülre, és adja meg, hogy az ajánlat adatait. 

## <a name="editing-operations"></a>Szerkesztési műveletek

A vízszintes eszköztár, a bemeneti adatok területet, amely a következő gombokkal jeleníti meg:

|   Gomb    |   Cél                                                          |
|   ------    |  --------                                                          |
| **Mentés**    | Adatok bejegyzés legutóbbi módosítások mentése.  Módosítások mentése kell manuálisan meg elhagyni a lapot, vagy a módosítások elvesznek. | 
| **Elvetése** | Elveti a legújabb adatváltozásokat bejegyzés (mivel az utolsó mentése)             |
| **Hasonlítsa össze** | Hasonlítja össze az aktuális ajánlat közzétett szolgáltatásegységeket állapotát.  Csak akkor engedélyezett, miután az ajánlat közzététele sikeresen befejeződött.  |
| **Közzététel** | Ezt az ajánlatot a közzétételi folyamat megkezdése                       |
| **Törlés**  | Törli ezt az ajánlatot a létrehozása után, de mielőtt azt közzé lett téve. |
|   |   |


## <a name="editing-tabs"></a>Szerkesztés lap

Ajánlat létrehozásakor, adja meg a szükséges és választható adatokat az egyes lapokon található, a bal oldali függőleges oszlopban a **új ajánlat** lapot.  Általános jogú felhasználói felületi vezérléssel--szövegdobozok, legördülő menük és jelölőnégyzetek – például az adatgyűjtés jelennek meg.  Bár az adott gyűjtemény szerkesztési lapokat, attól függ, hogy az ajánlat típusát, az alábbi táblázat néhány gyakori lapokon.

|      Lap neve       |   Cél                                                            |
|      --------       |   -------                                                            |
| **Ajánlat beállításai**  | Gyűjti az ajánlat és a közzétevő azonosító adatokat.                    |
| **Termékváltozatok**            | Minden készletkezelési egysége (Termékváltozat) verzióhoz az Ön ajánlatát a technikai és üzleti jellemzők meghatározása |
| **Test Drive**      | Ezt az opcionális szolgáltatást támogató-típusokhoz határozza meg az ajánlatban bemutatója.  További információkért lásd: [Mi a Test Drive?](../test-drive/what-is-test-drive.md)  |
| **Marketplace-en** vagy **áruház oldala** | Gyűjti a szöveges karakterláncot, dokumentumok és képek listáját tartalmazza az ajánlatot a piactéren |
| **Támogatás**         | Összegyűjti az ügyfél, az adatfeldolgozás és online támogatást kapcsolattartási adatok  |
|  |  |

A hasonló nevű lapok tartalmát többek között a különböző típusú eltérhet.  Az egyes ajánlat vonatkozó ajánlat részleteit az alábbi lapfülek biztosított belül a "Ajánlat létrehozása" szakaszban.

# <a name="azure-virtual-machine-offer"></a>Azure-beli virtuális gépek ajánlata

[Cloud Partner Portal](https://cloudpartner.azure.com/) már nem támogatja az Azure-beli virtuális gépekkel kapcsolatos ajánlatok létrehozását és felügyeletét. Ez a funkció át lett helyezve a Microsoft [partner Centerbe](https://partner.microsoft.com/pcv/). További információ: Azure-beli [virtuális gépek ajánlatának létrehozása](../../partner-center-portal/azure-vm-create-offer.md).

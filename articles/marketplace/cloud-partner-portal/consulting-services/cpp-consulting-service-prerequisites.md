# <a name="consulting-service-prerequisites"></a>Tanácsadási szolgáltatás előfeltételei

Tekintse át a leírását, és írja be az ajánlat követelményei [Consulting Services, Azure Marketplace és AppSource](../../consulting-services.md).  A következő speciális tanácsadási szolgáltatás típusokat további követelményei:

|     Csomag típusa   |    Követelmény   |
|     ----------   |    -----------   |
| Dynamics 365 for Customer Engagement | Partnerek számára a Silver vagy Gold minősítési kell rendelkeznie a [felhőalapú Ügyfélkapcsolat-kezelés kompetencia](https://partner.microsoft.com/membership/cloud-customer-relationship-management-competency). |
| Dynamics 365 for Finance és Operations | Partnerek számára a Silver vagy Gold minősítési kell rendelkeznie a [vállalati erőforrás-tervezés kompetencia](https://partner.microsoft.com/membership/enterprise-resource-planning-competency). Partnerek is kell rendelkeznie egy minimális bevétel $ 25 000 Felhőbeli műveletek az elmúlt 12 hónapban. |
| Dynamics 365 Business Central | Partnerek egy Felhőszolgáltató (CSP) vagy egy digitális partnert, rekord (DPOR) legalább egy ügyfél járt el kell rendelkeznie. <br/> Hogyan válhat egy CSP-hez a további információkért lásd: [első lépései egy CSP-hez mint](https://partner.microsoft.com/cloud-solution-provider/get-started). <br/> Dpor-partnerként választotta követelmények, lásd: a [dpor-partnerként választotta áttekintése](https://partner.microsoft.com/membership/digital-partner-of-record#Membership_Digital_Partner_of_Record_Navigated_Rich_Text_Node1). |
|  Power BI  | Felkerülés a a Power BI-partner bemutatása. További információkért lásd: [Felhőszolgáltatói Partner feltételek](https://powerbi.microsoft.com/become-a-partner). |
|  PowerApps       | Partnerek rendelkeznie kell egy közzétett megoldás a [Partnermegoldás-Bemutatónkban](https://powerapps.microsoft.com/partner-showcase). |
|  |  |

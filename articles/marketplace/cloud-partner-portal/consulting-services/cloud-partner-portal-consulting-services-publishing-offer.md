# <a name="azure-and-dynamics-365-consulting-service-offer"></a>Az Azure és a Dynamics 365 tanácsadási szolgáltatásajánlata

Ebből a cikkből megtudhatja, hogy miként tehet közzé tanácsadási szolgáltatási ajánlatot a Microsoft <a href="https://azuremarketplace.microsoft.com">Azure Piactéren</a> vagy az <a href="https://appsource.microsoft.com">AppSource-on.</a> A Microsoft <a href="https://dynamics.microsoft.com">Dynamics 365,</a> <a href="https://powerbi.microsoft.com">a Power BI</a>és a Power <a href="https://powerapps.microsoft.com">Apps</a> alkalmazáson alapuló megoldások az AppSource webhelyen is megjelenhetnek. Más Microsoft <a href="https://azure.microsoft.com/services">Azure-szolgáltatásokon</a> alapuló egyéb ajánlatok is felsorolhatók az Azure Piactéren.

## <a name="publishing-benefits"></a>Közzétételi előnyök

A Microsoft piactereken való közzététel a következő előnyökkel jár:

- Népszerűsítse vállalatát a Microsoft márka kihasználásával.
- Potenciálisan több mint 100 millió Office 365- és Dynamics 365-felhasználót érhet el az AppSource-on, és több mint 200 000 szervezetet érhet el az Azure Piactéren keresztül.
- Kiváló minőségű érdeklődőket szerezhet be ezekről a piacterekről.
- A Microsoft helyszíni és teleértékesítési csapatai népszerűsíthetik szolgáltatásait.

## <a name="publish-a-consulting-service-offer"></a>Tanácsadási szolgáltatási ajánlat közzététele

Ez a szakasz a tanácsadási szolgáltatási ajánlat közzétételének elemeit ismerteti. A közzététel a következő fő részekre oszlik:

- [A tanácsadási szolgáltatás előfeltételei](./cpp-consulting-service-prerequisites.md) a következő tanácsadási szolgáltatási ajánlat közzétételének követelményeit sorolják fel:
 
    - Microsoft Azure
    - Dynamics 365 ügyfélelköteleződés 
    - Dynamics 365 pénzügyi & műveletek 
    - Dynamics 365 Customer Insights
    - Dynamics 365 Business Central 
    - Power BI 
    - Power Apps
    - Power Automate
- [Hozzon létre egy új ajánlatot,](./cpp-consulting-service-create-offer.md) amely felsorolja a tanácsadói szolgáltatásajánlat-bejegyzés létrehozásához szükséges lépéseket a Cloud Partner Portal használatával:
    - [Adja meg az ajánlatbeállításait](./cpp-consulting-service-define-offer-settings.md).
    - [Adja meg a kirakat adatait, és hogy közzé szeretne-e tenni az Azure Piactéren vagy az AppSource-on.](./cpp-consulting-service-storefront-details.md)
- [Tegye közzé az ajánlatot,](./cpp-consulting-service-publish-offer.md) amely elmagyarázza, hogyan küldheti el az ajánlatot közzétételre.

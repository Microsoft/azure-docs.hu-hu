# <a name="cloud-solution-providers"></a>Felhőszolgáltatók

Szoftver ajánlatok most már elérheti a minimális Cloud Solution Provider (CSP) partner által kiszolgált minősített Microsoft ügyfélbázisra marketing és értékesítés befektetése jelenti.  A kiadók már kihasználhatják a CSP-partner csatornákon keresztül ajánlatok nyilvános rendelkezésre állás mellett [a Microsoft web kirakattípus](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

A kiadók a vehetnek részt a rendelkezésre állás érdekében a CSP partner egy új ajánlat és a egy meglévő csatorna ajánlat választhatja. Ez a beállítás lehetővé teszi, hogy a CSP-partnerek számára a termékek értékesítése és csomagolt megoldásokat hozhat létre az ügyfelek.

A kiadók felelősek, a végfelhasználók számára, amelyek biztosítják az hibajavítási és a egy CSP-partnerek mechanizmust és/vagy az ügyfelek számára, hogy felvegye Önnel a kapcsolatot támogatási, amelyek biztosítják. Adja meg a CSP-partnerek a dokumentáció, képzés és a service health/leállási értesítés (ha létezik), hogy CSP-partnerek ellátni a réteg 1 támogatási kérelem érkezett kezelésére vonatkozó ajánlott eljárásokat.

Az alábbi táblázat ajánlattípusokról és vehetnek részt a CSP-partner csatorna díjszabási modellek:

| **Csomag típusa**    | **BYOL**  |  **Ingyenes** | **HASZNÁLATALAPÚ**   |
| :---------------- | :---------|:----------|:-----------|
| Virtuális gépek  | Igen | Igen | Igen |
| Containers | – | Igen | – |
| Megoldássablonok | – | Igen | – |
| Felügyelt alkalmazások | – | Igen | Igen |
| IT-megoldást SaaS (/ hely díjszabása) | – | Igen | Igen |
|   |   |   |

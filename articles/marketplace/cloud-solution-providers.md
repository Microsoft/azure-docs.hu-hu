# <a name="cloud-solution-providers"></a>Felhőszolgáltatók

Szoftver ajánlatok elérheti a Cloud Solution Provider (CSP) program mellett a nyilvános rendelkezésre állás az ajánlatok keresztül a partnerek által kiszolgált minősített Microsoft ügyfélbázisra [a Microsoft web kirakattípus](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

A kiadók ajánlatokat a rendelkezésre állás érdekében konfigurálása választható bővítjük a CSP programban (alapértelmezett ki van zárva), vagy egy meglévő egy új ajánlatunk, a termékek értékesít, és hogy kötegelve megoldások ügyfelek számára.

A kiadók felelősek, a végfelhasználók számára, amelyek biztosítják az hibajavítási és a egy CSP-partnerek mechanizmust és/vagy az ügyfelek számára, hogy felvegye Önnel a kapcsolatot támogatási, amelyek biztosítják. Adja meg a CSP-partnerek a dokumentáció, képzés és a service health/leállási értesítés (ha létezik), hogy CSP-partnerek ellátni a réteg 1 támogatási kérelem érkezett kezelésére vonatkozó ajánlott eljárásokat.

Az alábbi táblázat ajánlattípusokról és vehetnek részt a CSP program díjszabási modellek:

| **Csomag típusa**    | **BYOL**  |  **Ingyenes** | **HASZNÁLATALAPÚ**   |
| :---------------- | :---------|:----------|:-----------|
| Virtuális gépek  | Igen | Igen | Igen |
| Containers | – | Igen | – |
| Megoldássablonok | – | Igen | – |
| Felügyelt alkalmazások | – | Igen | Igen |
| Szoftver--szolgáltatásként (SaaS) | – | Igen | Igen |
|   |   |   |

## <a name="how-to-configure-an-offering"></a>Egy adott ajánlat konfigurálása

A CSP program opcionális beállítása konfigurálva van a Partner Center vagy Cloud Partner Portalon ajánlat létrehozási folyamatának. ([További információ a változó közzétevő élményt. ](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293))

A Cloud Partner portálra akkor van beállítva a Marketplace-en vagy kirakat lapon. A Partner Center azt állítja be az ajánlat beállítása lapon a telepítés részleteit.

![CSP részvétel a felhasználói felület az CPP](media/marketplace-publishers-guide/csp-opt-in.png)

Partnerközpont, az alább látható módon a kiadók a választásának ajánlatait a CSP program elérhetővé tétele a négyzet bejelölésével **saját ajánlat elérhetővé a CSP programban**.

![A Partner Center CSP vehetnek részt élmény](media/marketplace-publishers-guide/pc-csp-opt-in.png)

## <a name="sharing-sales-and-support-materials-with-csp-partners"></a>Értékesítés és a támogatási anyagok osztanak meg CSP-partnerek

Ahhoz, hogy a leghatékonyabban jelölik az ajánlat és kapcsolatba léphet a szervezet, küldje el az értékesítési és támogatja az anyagokat, amely a viszonteladók számára elérhető lesz a Cloud Solution Provider program partnerei. Ezeket az erőforrásokat a Marketplace-en kirakattípus ügyfelei számára nem lesz közzétéve.

Ahogy az alábbi Cloud Partner portálra, az közzétevők adhat meg egy URL-címet, amelyen a megfelelő marketinganyagokat, és a csatorna kapcsolattartási adatokat a CSP-csatornára:

![Collateral CSP-adatokat](media/marketplace-publishers-guide/cpp-csp-information.png)

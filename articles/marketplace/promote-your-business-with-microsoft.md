# <a name="promote-your-business-with-microsoft"></a>Vállalkozás népszerűsítése a Microsofttal 

A Microsoft célja, hogy támogassa a partnereket (Ön) és egyszerűsítse az alkalmazás vagy ajánlat értékesítését a piactéren lévő kirakatokban. A marketplace-en való listázással elérheti a piactéren lévő programokat és erőforrásokat. 

Ha még nem regisztrált a Microsoft Partner Network (MPN) hálózatra, akkor ezt a folytatás előtt tegye meg.  
*   Az MPN-ről további információt a Microsoft [partner.microsoft.com](https://partner.microsoft.com)Partner Network partner.microsoft.com .  

Az MPN a következő témakörökben nyújt útmutatást.  
*   Új üzleti lehetőségek  
*   Csatlakozás csapatokhoz vagy partnerekhez  
*   Megoldások és képzés, hogy segítsen növelni a skillset  

Az ajánlat vagy az alkalmazás hirdetéséhez az alábbi táblázat segítségével további előnyöket és erőforrásokat érhet el. 

| Előny | resources |  
|:--- |:--- |  
| Használja az alapvető előnyöket | A Microsoft Partner Network részeként számos alapvető előnyt kap, amelyek időt és pénzt takaríthatnak meg. Az előnyök segíthetnek a képességei megerősítésében, az ügyfelek jobb kiszolgálásában és kapcsolatok kiépítésében, hogy teljes üzleti potenciált érhessünk el.<ul> <li>Az alapvető előnyökkel kapcsolatos további információkért látogasson el a [partner.microsoft.com/membership/core-benefits](https://partner.microsoft.com/membership/core-benefits).</li> </ul> |  
| Szerezze meg a Cloud Platform kompetenciáját | A kompetencia megszerzésével megkülönböztetheti magát a Microsoft többi partnere között. A kompetencia megszerzésével bizonyíthatja a műszaki szakértelmet és az ügyfelek sikerét a piacon. Kompetencia előfeltétele számos kulcsfontosságú partner programok, mint a Co-Sell.<ul> <li>A Cloud Platform kompetenciájával kapcsolatos további információkért látogasson el a Cloud Platform kompetencia oldalára, amely [a partner.microsoft.com/membership/cloud-platform-competency](https://partner.microsoft.com/membership/cloud-platform-competency).</li> </ul> |  
| Legyen Co-Sell Kész | A Co-Sell Ready program lehetővé teszi, hogy közvetlenül együttműködjön a Microsoft eladóival és más partnerekkel a megcélzott ügyféllehetőségek és a fióktervezés terén. A megoldás láthatóvá válik a Microsoft eladói számára az eladói megoldáskatalógusban. A Microsoft eladóit az Önnel való együttműködésért és a győzelemért jutalmazzuk.<ul> <li>A Co-Sell készültségről további információt a Vállalkozás hirdetése oldalon talál, amely [a partner.microsoft.com/reach-customers/promote-your-business](https://partner.microsoft.com/reach-customers/promote-your-business).</li> </ul> |

<a name="seller-insights-definitions"></a>Eladó insights definíciók
=======================

Az alábbi táblázat az Seller Insights számos kifejezésének definícióit tartalmazza.

|  **Kifejezés**                 |  **Meghatározás**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Azure-licenctípus**                               | Az ügyfelek által az Azure megvásárlásához használt licencszerződés típusa.  Más néven Csatorna.                                                  |
| **Azure licenctípusa: Felhőszolgáltató**      | A végfelhasználó az Azure-t és a Marketplace-ajánlatot a felhőszolgáltatón keresztül szerzi be, amely viszonteladóként működik.                 |
| **Azure licenctípusa: Enterprise**                   | A végfelhasználó az Azure-t és a Marketplace-ajánlatot egy közvetlenül a Microsofttal aláírt nagyvállalati szerződésen keresztül szerzi be.                  |
| **Azure licenctípus: Nagyvállalati viszonteladón keresztül**  | A végfelhasználó az Azure-t és a Marketplace-ajánlatot egy viszonteladón keresztül szerzi be, aki megkönnyíti a Microsofttal kötött nagyvállalati szerződésüket.     |
| **Azure licenctípusa: Pay as You Go**                | A végfelhasználó az Azure-t és a Marketplace-ajánlatot egy közvetlenül a Microsofttal aláírt "Csak annyiban fizetendő" szerződés keretében szerzi be az Azure-t és a Marketplace-ajánlatot.                |
| **Költség összege (CC)**                              | A vevőnek felszámított összeg a *számlázásügyfél pénznemében* (CC).                                                                 |
| **Díj összege (PC)**                               | Az ügyfélnek felszámított összeg a *kifizetési pénznemben* (PC).                                                                      |
| **Költséghatár dátuma**                                      | A vevő költségének kiszámításának dátuma (általában közvetlenül a használati időszak után).                                             |
| **Felhőpéldány neve**                              | Az a Microsoft Cloud, amelyben a virtuális gép központi telepítése történt.                                   |
| **Felhőpéldány neve: Azure Global**                | A nyilvános globális Microsoft-felhő.                           |
| **Felhőpéldány neve: Azure Government**        | A kormányspecifikus Microsoft-felhők a következő kormányok egyikéhez: Kína, Németország vagy az Amerikai Egyesült Államok.                           |
| **Gyűjtemény állapota**  | Egy adott díj esetén a számlázási és beszedési életciklus legújabb állapota.  Az állapot lehet például a folyamatban lévő beszedés, a beszedés vagy a visszatérítés.                  |
| **Mag méret**  | Az ajánlat által futtatott virtuális gép által bemutatott virtuális magok száma.               |
| **Ügyfél városa**  | Az ügyfél által megadott városnév. Ez eltérhet az ügyfél Azure-előfizetésében szereplő várostól.  |
| **Ügyfél kommunikációs nyelve**        | Az ügyfél által a kommunikációhoz előnyben részesített nyelv.                                  |
| **Vevő vállalatneve**                  | A vevő által megadott vállalatnév. Ez eltérhet az ügyfél Azure-előfizetésében szereplő várostól.                                  |
| **Vevő országa**                       | A vevő által megadott ország-/régiónév.  Ez eltérhet az ügyfél Azure-előfizetésében szereplő országtól/régiótól.                               |
| **Vevő pénzneme (CC)**                 | Az ügyfél által az árképzéshez és a számlázáshoz előnyben részesített pénznem.                            |
| **Ügyfél e-mail címe**                         | A végfelhasználó által megadott e-mail cím.  Ez eltérhet az ügyfél Azure-előfizetésében szereplő e-mail címtől.                   |
| **Vevő utóneve**                              | A vevő által megadott utónév.  Ez eltérhet az ügyfél Azure-előfizetésében megadott névtől.                    |
| **Vevő vezetékneve**                               | A vevő által megadott vezetéknév.  Ez eltérhet az ügyfél Azure-előfizetésében megadott névtől.                    |
| **Ügyfél azonosítója**                               | A vevőhöz rendelt egyedi azonosító.  Előfordulhat, hogy egy ügyfél nulla vagy több Azure Piactér-előfizetéssel rendelkezik.                    |
| **Vevői kifizetés típusa**                            | A vevő által használt fizetési eszköz típusa.  Ez lehet például karton, számla vagy elektronikus csekk.                             |
| **Vevő irányítószáma**                             | A vevő által megadott irányítószám.  Ez eltérhet az ügyfél Azure-előfizetésében megadott irányítószámtól.                                                                 |
| **Vevő állapota**                                   | Az ügyfél által megadott állapot (cím).  Ez eltérhet az ügyfél Azure-előfizetésében megadott vezetéknévtől.                                                                 |
| **Beszerzés dátuma**                                    | Az első nap, amikor az ügyfél megvásárolta az Ön által közzétett ajánlatot.                 |
| **Elveszett dátum**                                        | Az utolsó dátum, amikor a vevő visszavonta a korábban megvásárolt utolsó ajánlatot.                                                         |
| **Becsült kiterjesztett díj (CC)**                   | Egy adott termékváltozat használati egységeinek becsült kiterjesztett díja (a vevő pénznemében). Ez az érték nem lehet pontos a kerekítési vagy csonkolási hibák miatt.                            |
| **Becsült kiterjesztett töltés (PC)**                   | Egy adott termékváltozat használati egységeinek becsült kiterjesztett díja a használat napján történő devizaátváltás alapján (a közzétevő pénznemében). Ez az érték nem lehet pontos a kerekítési vagy csonkolási hibák miatt.                  |
| **Becsült kifizetés (PC)**                            | Egy adott termékváltozat használati egységeinek becsült kifizetése, amely a használat kiszámításának napján (a közzétevő pénznemében) történő devizaátváltáson alapul. Ez az érték nem lehet pontos a kerekítési vagy csonkolási hibák miatt.                        |
| **Becsült ár (PC)**                             | Egy adott Termékváltozat használati egységének becsült ára, amely a használat napján érvényes devizaátváltáson alapul (a közzétevő pénznemében). Ez az érték nem lehet pontos a kerekítési vagy csonkolási hibák miatt.                                             |
| **Új vevő**                                  | Az érték tudatja Önvel, ha egy új ügyfél, aki először fogyaszt vagy vásárol meg egy vagy több Marketplace-ajánlatot (vagy nem).  Az érték "Igen" lesz, ha az azonos naptári hónap a "Beszerzett dátum" esetében.  Az érték "Nem" lesz, ha az ügyfél a naptári hónap előtt megvásárolta az Ön ajánlatait.                                                       |
| **Az előnézeti termékváltozat**                                  | Az érték tudatja Önvel, ha a termékváltozatot "előnézetként" címkézte. Az érték "Igen" lesz, ha a termékváltozat ennek megfelelően lett címkézve, és csak az Ön által engedélyezett Azure-előfizetések telepíthetik és használhatják ezt a lemezképet.  Az érték "Nem" lesz, ha a termékváltozat nincs "előzetes verzióként" azonosítva.                                          |
| **A promóciós kapcsolat opt-in**                   | Az érték tudatja Önnel, ha az ügyfél proaktív módon engedélyezte a közzétevők promóciós kapcsolatát. Jelenleg nem mutatjuk be a lehetőséget az ügyfeleknek, ezért nemet jelöltünk meg. A funkció telepítése után ennek megfelelően elkezdjük a frissítést.                                          |
| **Piactéri licenc típusa**                         | A Marketplace-ajánlat számlázási módszere.                             |
| **Piactérlicenc típusa: Számlázás az Azure-on keresztül**   | A Microsoft az Ön ügynöke ehhez a Marketplace-ajánlathoz, és az Ön nevében számláz az ügyfeleknek. (PayG hitelkártya vagy vállalati számla)       |
| **Piactéri licenc típusa: Hozza magával a saját licencét** | A virtuális gép telepítése szükséges egy licenckulcsot az ügyfél által biztosított. A Microsoft nem számít fel díjat az ügyfeleknek a jelen Marketplace-ajánlatok használatáért. |
| **Piactéri licenc típusa: Ingyenes**                   | A Marketplace-ajánlat úgy van beállítva, hogy minden felhasználó számára ingyenes legyen. A Microsoft nem számít fel díjat az ügyfeleknek a jelen Marketplace-ajánlat használatáért.    |
| **Piactéri licenc típusa: A Microsoft mint viszonteladó**  | A Microsoft az Ön viszonteladója ehhez a Marketplace-ajánlathoz.                               |
| **Piactéri előfizetés cégneve**  | Az ügyfél által az Azure-előfizetésben megadott cégnév.                           |
| **Marketplace-előfizetés azonosítója**            | Az ügyfél által a Marketplace-ajánlat megvásárlásához használt Azure-előfizetéshez társított egyedi azonosító.  Korábban az Azure-előfizetés GUID.It was former the Azure Subscription GUID.  Bizonyos esetekben, például egy SAAS-rendelés, nincs előfizetés szükséges.           |
| **Microsoft-díj (CC)**                               | A tranzakció Microsoft-díja az ügyfél pénznemében.                              |
| **Ajánlat neve**                                | A Marketplace-ajánlat neve.                                                             |
| **Ajánlat típusa**                                | A Microsoft Piactér ajánlatának típusa.  Például lehet felügyelt alkalmazás, virtuálisgép-licencek vagy tároló.                           |
| **Rendelés érvénytelenítési dátuma**                         | A Marketplace-rendelés érvénytelenítésének dátuma.                                                       |
| **Rendelésazonosító**                                  | A Marketplace-szolgáltatás vevői rendelésének egyedi azonosítója.  A virtuális gép használatalapú ajánlatai nincsenek rendeléshez társítva.                 |
| **Rendelés beszerzési dátuma**                       | A Marketplace-rendelés létrehozásának dátuma.                                                        |
| **Megrendelés mennyisége**                       | Az Azure-előfizetéshez társított ugyanazon termékváltozatrendeléseinek mennyisége.  A SAAS-rendelések esetében a rendelés ezen példányához társított helyek számát jelöli.                                            |
| **Rendelés állapota**                              | A Marketplace-rendelés állapota az adatok utolsó frissítésekor.  Lehet például, hogy a rendelés aktív vagy visszavont.             |
| **Rendelés állapota: Aktív**                           | A vevő megvásároltegy rendelést, és nem mondta vissza a rendelést.   |
| **Rendelés állapota: Visszavonva**                        | A vevő korábban megvásárolt egy rendelést, és ezt követően visszavonta a rendelést.        |
| **Kifizetés összege (PC)**                             | Az Önnek kifizetett összeg a kívánt kifizetési pénznemben (PC) történik.                                       |
| **Kifizetés pénzneme (PC)**                           | A kifizetésekhez használt pénznem.                                                                |
| **Kifizetés dátuma**                                    | A fizetési kérelem Microsoft által a kiválasztott pénzintézetnek történő elküldésének dátuma.                                                                         |
| **Kifizetés állapota**                                  | A kifizetés állapota az adatok utolsó frissítésekor.                                   |
| **Kifizetési állapot: Folyamatban**               | A tranzakció nem áll készen a kifizetésre. (További információ: Collection Status)                             |
| **Kifizetési állapot: Kifizetve**                          | A tranzakció egy korábbi kifizetési számításban szerepelt. A pozitív értékek et a függvény fizeti, a negatív értékeket pedig nettósítják a teljes esedékes összeggel szemben. |
| **Kifizetési állapot: Közelgő kifizetés**                   | A tranzakció készen áll a kifizetésre, és szerepelni fog a következő rendelkezésre álló kifizetési számításban.                      |
| **Ár (CC)**                                   | Egy adott termékváltozat használati egységének ára (a vevő pénznemében).                                                                  |
| **Szolgáltató e-mail címe**                                 | A Microsoft és a végfelhasználó közötti kapcsolatban részt vevő szolgáltató e-mail címe. Ha az ügyfél viszonteladón keresztül enterprise, akkor az a viszonteladó lesz.  Ha egy felhőszolgáltató (CSP) is részt vesz, az a kriptár-szolgáltató lesz.                                                                 |
| **Szolgáltató neve**                                 | A Microsoft és a végfelhasználó közötti kapcsolatban részt vevő szolgáltató neve. Ha az ügyfél viszonteladón keresztül enterprise, akkor az a viszonteladó lesz.  Ha egy felhőszolgáltató (CSP) is részt vesz, az a kriptár-szolgáltató lesz.                                                                 |
| **Erőforrás URI-ja**                                  | Az az útvonal, ahol a központi telepítési összetevő telepítve volt.                                            |
| **Sku**                                           | A termékváltozat neve a közzététel során meghatározott aknak. Egy ajánlat sok termékváltozattal rendelkezhet, de egy termékváltozat csak egyetlen ajánlathoz társítható.              |
| **Termékváltozat számlázási típusa**                                 | A termékváltozat számlázási módja.                                                                                                             |
| **Termékváltozat számlázási típusa: Saját licenc cel**         | A virtuális gép vagy szolgáltatás üzembe helyezéséhez az ügyfél által biztosított licenckulcs szükséges. A Microsoft nem számláz az ügyfeleknek a Marketplace-ajánlatok használatáért.   |
| **Termékváltozat számlázási típusa: A Microsoft viszonteladóként**          | A Microsoft a termékváltozat viszonteladója.   |
| **Termékváltozat számlázási típusa: ingyenes**                           | A termékváltozat úgy van beállítva, hogy minden felhasználó számára ingyenes legyen. A Microsoft nem számláz a vevőknek a termékváltozat használatáért.                           |
| **Termékváltozat számlázási típusa: Fizetett**                           | A Microsoft az Ön ügynöke ehhez a termékváltozathoz, és az Ön nevében számláz az ügyfeleknek. (PayG hitelkártya vagy vállalati számla)                   |
| **Termékváltozat számlázási típusa: Próbaverzió**                          | Az ügyfél a próbaidőszak, és át kell számítani a fizetett, ha nem törli vagy törli.                                         |
| **Adó összege (CC)**                                  | A vevő számlájára alkalmazott adóösszeg a vevő pénznemében (CC).                                                               |
| **Tranzakció dátuma**                                 | A tranzakció dátuma a kifizetési jelentésben.                                                                           |
| **Tranzakció típusa**                               | A jelentett tranzakció típusa.  Ez lehet például díj, visszatérítés vagy kifizetési korrekció.                                                                                        |
| **Próba időszak záró dátuma**                                 | A rendelés próbaidőszakának kezdőnapja vagy vége.                                                                           |
| **Usage Date (Használat dátuma)**                                       | A vevői felhasználás dátuma.                                                                                                             |
| **Használati mennyiség**                                       | A termékváltozat jelentett vevői használata. Virtuálisgép-rendszerképek esetén a használati rekordok az adott virtuális gép méretének és termékváltozatának jelentett időszakának használatát jelölik.                                               |
| **Használati útmutató**                                       | Egy adott termékváltozat hoz tartozó egy adott termékváltozat egy vagy több nap vevői felhasználásának azonosítója, amely a kifizetési jelentésben szereplő bejegyzéshez van társítva.                                               |
| **Usage Type** (Használat típusa)                                       | A mért használat leírása. (Normalizált használat vagy nyers használat)                                                              |
| **Használat típusa: Normalizált használat**                | Használati órák normalizálva a használatban részt vevő virtuálisgép-magok számának figyelembevételével: [Virtuálisgép-magok száma] x [Nyers használat órái]. A "SHAREDCORE" néven kijelölt virtuális gépek 1/6 (vagy 0.1666) a [Virtuálisgép-magok száma] szorzó.                                        |
| **Felhasználás típusa: Nyers használat**                        | A Marketplace-ajánlat használati órái.  A "naptár" órákban jelenik meg.  Ez a virtuális gépek futási idejének száma.                           |
| **Használati egységek**                                   | A megadott használati mértékegység. A virtuális gépeket mindig óránkénti mértékegységekkel mérik.                               |
| **Virtuális gép mérete**                                       | A virtuális gép hardverméretét jelöli az Azure-ajánlatnak megfelelően. Ilyen `Basic_A0`például `Standard_D12`a `Standard_G4`, `Standard_A11`, és .   |

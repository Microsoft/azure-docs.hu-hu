# <a name="azure-saas-application-offer"></a>Azure SaaS-alkalmazásra vonatkozó ajánlat

<table> <tr> <td>Ez a szakasz ismerteti a Microsoft szoftver--szolgáltatásként (SaaS-) alkalmazás közzététele <a href="https://azuremarketplace.microsoft.com/marketplace/">Azure Marketplace-en</a> vagy <a href="https://appsource.microsoft.com">Microsoft AppSource</a>. Használja az SaaS-alkalmazásokat, ha a megoldás lesz telepítve, a saját Azure-előfizetés és az ügyfelek kezelőfelületén, tervezheti meg és kezelheti az alkalmazás teszteléséhez jelentkezzen be. <a href="https://azure.microsoft.com/services/active-directory">Az Azure Active Directory</a> (AAD) segítségével kihasználhatja a meglévő próbaverzió környezet. </td> <td><img src="./media/cloud-partner-portal-publish-web-app/saasapp-icon1.png"  alt="Azure SaaS application icon" /></td> </tr> </table>


## <a name="publishing-benefits"></a>Közzétételi lehetőségekről

Az alkalmazások a Microsoft marketplace-en listázása előnyei a következők:

- 100 millió Azure Active Directory-felhasználók elérni az Office 365 és Dynamics 365 között.
- Az értékesítési csoport kiterjesztése: érhet el üzleti felhasználóknak világszerte, és így olyan értékesítési csatorna, amely felkelti az ügyfelek teljes, hozzájárul az érdeklődői és kapcsolatot teremt az új ügyfelek különböző ágazatok között.

- Gyakorlatban hasznosítható elemzéseket kaphat: hogyan működik az alkalmazás az appsource-on, milyen jól működik, és hogyan tovább javíthatja az értékesítési eljárások betekintést nyújtunk.


## <a name="publishing-process"></a>Közzétételi folyamat

Létrehozása és közzététele a Microsoft Azure Marketplace-en vagy a Microsoft AppSource alkalmazás(ok), ez a szakasz végigvezeti a szükséges, ha az alábbi lépéseket:

1. Győződjön meg arról, hogy megfelel-e az előfeltételeket, beleértve a partner regisztrációk: [fejlesztői központ-regisztráció](https://dev.windows.com/en-us/registration?accountProgram=Azure) *vagy* [Microsoft Partner Network-regisztráció](https://partner.microsoft.com/).
2.  Adja meg a Piactéri ajánlat és a terveket.
3.  Adja meg a tartalom és díjszabási.
4.  Tesztelje, és tanúsítvány beszerzése az ajánlatot a [Cloud Partner Portalon](https://cloudpartner.azure.com).
5.  Élesben az előfizetési csomag keretében.

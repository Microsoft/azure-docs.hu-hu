# <a name="azure-application-offer"></a>Az ajánlat Azure-alkalmazás

<table> <tr> <td>Ez a szakasz ismerteti, hogyan tehet közzé egy új felügyelt Azure-alkalmazás ajánlattal a <a href="https://azuremarketplace.microsoft.com/marketplace/">a Microsoft Azure Marketplace-en</a>.  Minden felügyelt alkalmazás tartalmaz egy Azure Resource Manager-sablon, amely meghatározza az alkalmazás, amely általában tartalmaz egy vagy több virtuális gépet és egyéb támogató Azure - vagy Web-alapú szolgáltatások által használt összes technikai eszközök. </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>

## <a name="publishing-process"></a>Közzétételi folyamat

A következő cikkek végigvezeti Önt a folyamat létrehozásának és a egy felügyelt Azure-alkalmazás közzététele:

1. [A Microsoft Developer Center-fiók regisztrálása](./cloud-partner-portal-dev-center-accounts-registration.md) válnak az Azure piactéren való közzétételre jóvá.
2. [Egy új Azure-alkalmazás-ajánlat közzététele](./cloud-partner-portal-managed-app-publish.md) az Azure piactéren.
3. [Győződjön meg arról, az ajánlat élő](./cloud-partner-portal-managed-app-offer-go-live.md)!

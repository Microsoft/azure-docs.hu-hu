# <a name="lcs-subscription"></a>LCS-előfizetés

Ebben az oktatóanyagban megtudhatja, hogyan hozzon létre egy előfizetést. Ez az előfizetés kap egy Microsoft Online tesztbérlővel és a egy Microsoft Dynamics Lifecycle Services-projektet, telepíthet egy környezetet. Fogja további felhasználók beállítása a Microsoft Online bérlőben, és a következő szolgáltatás-felügyeleti képességekkel rendelkező tapasztalatszerzés. Az alábbiakban ismertetjük képességek:

- Előfizetés és a egy új Microsoft Online létrehozása tesztelje a bérlőben.
- Lépjen a Lifecycle Services-projektek.
- A Lifecycle Services szolgáltatásaival.
- További felhasználók hozzáadása a Microsoft Azure Active Directory és az ügyfél.
- Erőforrások megtekintése az előfizetés e-mailben.

## <a name="key-terms"></a>Fontos kifejezések:

**A Microsoft Online Services-bérlő** -bérlő a csoport az összes előfizetés és a felhasználók a szervezet számára. A bérlő egy időben, mint az első a Microsoft Online Services-előfizetés létrehozása.

**Előfizetés** -előfizetés online környezetet biztosít. Ebben a környezetben lehetővé teszi a felhőbeli műveletek a Dynamics 365-élmény és is megtekintheti, hogy hogyan fejleszt testreszabásokat is üzembe helyezhetők a felhőben.

**A Microsoft Azure Active Directory** – a felhőalapú környezet magában foglalja az Azure Active Directory (AD), amely segítséget nyújt a felhasználók, csoportok, biztonsági szerepköröket és online alkalmazások, hasonlóan ahhoz, ahogyan a helyszíni kezelése licencek kezelése környezetek.

**Felhasználók** -felhasználók a szolgáltatások, a szervezet által előfizetett a felügyelt Azure AD-ben. A bérlő minden felhasználója hozzáadható és hozzárendelt biztonsági szerepkörök.

**A fejlesztők és rendszergazdák** -fejlesztők és rendszergazdák által a hozzáférés Lifecycle Services, amely lehetővé teszi, hogy azok a felhasználók kezelhetik projektjeiket és a környezetek. Ezek a felhasználók is rendelkezésre állnak a végfelhasználók.

Lásd: [LCS előfizetés](https://docs.microsoft.com/dynamics365/operations/dev-itpro/dev-tools/sign-up-preview-subscription) további információ.

# <a name="azure-marketplace-participation-policies"></a>Az Azure Piactér részvételi szabályzata

A Microsoft Azure Piactér részvételi szabályzata általánosan érvényes az összes kiadóra és ajánlatra a Microsoft Azure Marketplace-en. A jelen szabályzat a Microsoft Marketplace Kiadói szerződésében rögzített feltételeket egészíti ki. Az Azure piactéren való részvételhez, a közzétevők mindig meg kell felelnie a házirendeket ismerteti, és/vagy hivatkozott ebben a dokumentumban. Ha egy kiadó bármikor nem teljesíti hiánytalanul az összes feltételt, a Microsoft eltávolíthatja a kiadó ajánlatát az Azure Piactérről. A jelen dokumentumot időről időre frissíthetjük.

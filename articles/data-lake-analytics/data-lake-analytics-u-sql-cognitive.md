# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Ismerkedés az U-SQL kognitív képességeivel

## <a name="overview"></a>Áttekintés
A U-SQL kognitív képességei lehetővé teszik a fejlesztők számára, hogy az intelligenciát használják big data programjaikban. 

A következő minták állnak rendelkezésre kognitív képességek használatával:
* Képalkotás: [arcok észlelése](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Képalkotás: [érzelem észlelése](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Képalkotás: [objektumok észlelése (címkézés)](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Képalkotás: [OCR (optikai karakterfelismerés)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Text: [Kulcsszókeresés & Hangulatelemzés](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)

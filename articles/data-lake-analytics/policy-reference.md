# <a name="azure-policy-built-in-definitions-for-azure-data-lake-analytics"></a>Azure Policy beépített definíciók a Azure Data Lake Analytics

Ez az oldal a Azure Policy [beépített](../governance/policy/overview.md) szabályzatdefiníciók indexe Azure Data Lake Analytics. Az egyéb Azure Policy további beépített szolgáltatásaival kapcsolatban lásd a Azure Policy [definíciókat.](../governance/policy/samples/built-in-policies.md)

Az egyes beépített szabályzatdefiníciók neve a szabályzatdefinícióra mutató hivatkozásokat tartalmaz a Azure Portal. A Verzió oszlopban található **hivatkozással** megtekintheti a forrást a [GitHub Azure Policy tárban.](https://github.com/Azure/azure-policy)

# <a name="azure-policy-built-in-definitions-for-azure-iot-hub"></a>Azure Policy beépített definíciók a Azure IoT Hub

A IoT Hub IoT-forgatókönyvek implementáját bemutató mintakódért tekintse meg a rövid [IoT Hub útmutatót.](./quickstart-send-telemetry-cli.md) Több programozási nyelvhez is íme rövid útmutató, például a C, Node.js és a Python.

Ez az oldal a Azure Policy [beépített](../governance/policy/overview.md) szabályzatdefiníciók indexe Azure IoT Hub. További Azure Policy más szolgáltatások beépített szolgáltatásaihoz: Azure Policy [beépített definíciók.](../governance/policy/samples/built-in-policies.md)

Az egyes beépített szabályzatdefiníciók neve a szabályzatdefinícióra mutató hivatkozásokat tartalmaz a Azure Portal. A Verzió oszlopban található **hivatkozással** megtekintheti a forrást a [GitHub Azure Policy tárban.](https://github.com/Azure/azure-policy)

# <a name="quotas-and-limitations"></a>Kvóták és korlátozások

[!INCLUDE [media services api v2 logo](./includes/v2-hr.md)]

Ez a cikk a Microsoft Azure Media Serviceshöz kapcsolódó kvótákat és korlátozásokat ismerteti.

[!INCLUDE [azure-mediaservices-limits](../../../includes/azure-mediaservices-limits.md)]

## <a name="open-a-support-ticket-to-request-changes-to-the-default-quotas"></a>Nyisson meg egy támogatási jegyet az alapértelmezett kvóták módosításának kérelmezéséhez
Ha módosítani kívánja a megadott alapértelmezett kvótákat, nyisson meg egy támogatási jegyet. Adja meg a kérésben szereplő részletes információkat a kívánt kvóta-változásokkal, a használati esetekkel és a szükséges régiókkal kapcsolatban.

### <a name="how-to-open-a-support-ticket"></a>Támogatási jegy megnyitása
A Azure Portal lépjen a [Súgó + támogatás](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportrequest)elemre. Ha nem jelentkezett be az Azure-ba, a rendszer kérni fogja a hitelesítő adatok megadását.

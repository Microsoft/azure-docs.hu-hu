# <a name="sync-service-manager-metaverse-search"></a>Szolgáltatáskezelő metaverzum-keresése

![Szolgáltatáskezelő szinkronizálása](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

A metaverzum-keresés lap az adatokkal kapcsolatos problémák elhárításához hasznos. A felső felében attribútumok kombinációján alapuló lekérdezést hozhat létre. Ha elégedett a lekérdezéssel, kattintson a **Keresés gombra.** Az eredmény látható az alsó rácson. Az Oszlopbeállítások segítségével kiválaszthatja, hogy mely oszlopok legyenek **láthatók.**

A keresési eredmények között jelöljön ki egy objektumot és **tulajdonságokat** a [metaverzum-objektum tulajdonságainak](tshoot-connect-object-not-syncing.md#metaverse-object-properties)megtekintéséhez.

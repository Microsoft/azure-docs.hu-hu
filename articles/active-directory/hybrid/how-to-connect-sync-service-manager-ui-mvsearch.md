# <a name="sync-service-manager-metaverse-search"></a>Service Manager metaverse-keresés szinkronizálása

![Service Manager szinkronizálása](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

A metaverse Search lap hasznos az adatkezeléssel kapcsolatos problémák elhárításához. A felső felében az attribútumok kombinációja alapján hozhat létre lekérdezést. Ha elégedett a lekérdezéssel, kattintson a **Keresés**gombra. Az eredmény látható az alsó rácsban. Kiválaszthatja, hogy mely oszlopokat kell megjeleníteni az **oszlop beállításainál**.

A keresési eredmények között válasszon ki egy objektumot és **tulajdonságokat** a [metaverse objektum tulajdonságainak](tshoot-connect-object-not-syncing.md#metaverse-object-properties)megtekintéséhez.

# <a name="how-to-configure-a-new-multi-tenant-application"></a>Egy új több-bérlős alkalmazás konfigurálása

További információ a több-bérlős alkalmazások ajánlott témakörök listáját a következő:

- Első általános ismeretekkel [mit jelent, ha egy több-bérlős alkalmazás](https://docs.microsoft.com/azure/active-directory/develop/active-directory-dev-glossary#multi-tenant-application)
- Első általános ismeretekkel [konfigurálása egy több-bérlős alkalmazást](https://docs.microsoft.com/azure/active-directory/develop/howto-convert-app-to-be-multi-tenant)
- Részletes áttekintést [hozzájárulási megvalósítása az Azure ad-ben hozzájárulási keretrendszer használatáról](https://docs.microsoft.com/azure/active-directory/develop/active-directory-integrating-applications), azért szükség, több-bérlős alkalmazásokhoz
- Ismerje meg részletesebben olvashat róluk, a [hogyan több-bérlős alkalmazás van konfigurálva, és teljes körű kódolt](https://docs.microsoft.com/azure/active-directory/develop/active-directory-devhowto-multi-tenant-overview), regisztrálása, beleértve az "általános" végpontot használja, a megvalósítása "user" és "admin" a jóváhagyás, speciális többrétegű megvalósítása forgatókönyvek

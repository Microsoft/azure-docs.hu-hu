# <a name="customer-data-storage-for-australian-and-new-zealand-customers-in-azure-active-directory"></a>Ügyféladatok tárolása ausztráliai és új-zélandi ügyfelek számára Azure Active Directory 

Azure Active Directory (Azure AD) a Microsoft Online szolgáltatásra való feliratkozáskor megadott ország alapján földrajzi helyen tárolja az ügyféladatokat. A Microsoft Online-szolgáltatások közé tartozik a Microsoft 365 és az Azure. 

Az Azure AD és más Microsoft-szolgáltatások adatainak helyéhez a Microsoft Adatvédelmi központ Hol találhatók az [adatai?](https://www.microsoft.com/trustcenter/privacy/where-your-data-is-located) című szakaszában találhatók további információk.

2020. február 26-tól a Microsoft megkezdte az Azure AD ügyfél-adatainak tárolását az ausztrál vagy új-zélandi számlázási címmel az ausztráliai adatközpontokban található új bérlők esetében.

Emellett bizonyos Azure AD-funkciók még nem támogatják az ügyféladatok ausztráliában való tárolását. A funkcióval kapcsolatos információkért lépjen az [Azure AD](https://msit.powerbi.com/view?r=eyJrIjoiYzEyZTc5OTgtNTdlZS00ZTVkLWExN2ItOTM0OWU4NjljOGVjIiwidCI6IjcyZjk4OGJmLTg2ZjEtNDFhZi05MWFiLTJkN2NkMDExZGI0NyIsImMiOjV9)adattérképére. A Multi-Factor Authentication Microsoft Azure AD például az Usa-ban tárolja az ügyféladatokat, és globálisan feldolgozza az adatokat. Lásd: [Az Azure AD Multi-Factor Authentication](../authentication/concept-mfa-data-residency.md)adattára és ügyféladata.

> [!NOTE]
> Az Azure AD-val integrált Microsoft-termékek, -szolgáltatások és harmadik féltől származó alkalmazások hozzáférnek az ügyféladatokhoz. Értékelje ki az egyes használt terméket, szolgáltatást és alkalmazást annak meghatározásához, hogy az adott termék, szolgáltatás és alkalmazás hogyan feldolgozta az ügyféladatokat, és hogy megfelelnek-e a vállalat adattárolási követelményeinek. A Microsoft-szolgáltatások adattárolási helyével kapcsolatban a Microsoft biztonsági és adatkezelési központ [az adatok tárolási helyével](https://www.microsoft.com/trustcenter/privacy/where-your-data-is-located) foglalkozó szakaszában találhat további információt.

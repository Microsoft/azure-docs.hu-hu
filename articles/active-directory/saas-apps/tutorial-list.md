# <a name="tutorials-for-integrating-saas-applications-with-azure-active-directory"></a>Oktatóanyagok az SaaS-alkalmazások integrálásához a Azure Active Directory

A felhőalapú [szoftveres (SaaS)](https://azure.microsoft.com/overview/what-is-saas/) alkalmazások Azure Active Directorysal való integrálásának elősegítése érdekében olyan oktatóanyagokat fejlesztettünk ki, amelyek végigvezetik a konfiguráción.

Az Azure AD-be előre integrált SaaS-alkalmazások listáját a [Active Directory piactéren](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps)tekintheti meg. Kattintson [ide](https://docs.microsoft.com/azure/active-directory/develop/howto-app-gallery-listing) , ha egy alkalmazást szeretne felvenni az alkalmazás-gyűjteménybe.

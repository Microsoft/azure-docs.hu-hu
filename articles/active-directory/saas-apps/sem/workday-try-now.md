# <a name="workday-and-azure-active-directory"></a>Munkanap és Azure Active Directory

A munkanapokat a Azure Active Directory (Azure AD) használatával csatlakoztathatja az előre összeállított integrációval, és automatikusan kiépítheti vagy kiépítheti a felhasználói fiókokat, amikor az alkalmazottak csatlakoznak, áthelyezhetők

![Megjeleníti a munkanap alkalmazás emblémáját](./media/workday.png)

> [!div class="nextstepaction"]
> [A munkanapok kipróbálása az Azure AD-vel >](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Mire használható a munkanap és az Azure AD?

- A **felhasználók** kiosztásának automatizálása – a munkanapokon a csoport, a szerepkör vagy más attribútumok alapján automatikusan rendelhet alkalmazásokat az alkalmazottakhoz.
- **Alkalmazotti profilok szinkronizálása** – gondoskodjon arról, hogy az alkalmazottak adatai naprakészek legyenek az Azure ad és a munkanap között.
- Leegyszerűsítheti a bevezetést **és a offboarding** – automatikusan kiépítheti a felhasználói fiókokat az új bérlők számára, így azok az első napon dolgozhatnak, és csökkenthetik a biztonsági kockázatokat a felhasználói fiókok automatikus kiépítésével, amikor az alkalmazottak távoznak.
- **Egyszerű egyszeri bejelentkezés** A felhasználónevek és jelszavak eltávolítása, valamint a felhasználók bejelentkezésének engedélyezése az Azure AD-ben üzemeltetett szervezeti fiókok használatával.

# <a name="servicenow-and-azure-active-directory"></a>ServiceNow és Azure Active Directory

Gyorsítsa fel a ServiceNow bevezetését a szervezeten belül, és növelje a felhasználók bevezetését az előre elkészített integrációval. A ServiceNow és a Azure Active Directory (Azure AD) összekapcsolásával egyszerűsítheti a IT-szolgáltatásmenedzsment.

![A ServiceNow alkalmazás emblémájának megjelenítése](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Az Azure AD-vel való ServiceNow kipróbálása >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>Mire használható a ServiceNow és az Azure AD?

- **Egyszerű egyszeri bejelentkezés** – több Felhasználónév és jelszó kiiktatása és a hozzáférés egyszerűsítése azáltal, hogy a felhasználók Azure ad-fiókjaik használatával bejelentkezhetnek a ServiceNow.
- A felhasználók kiépítésének automatizálása – automatizálhatja a felhasználói fiókok létrehozását, karbantartását és eltávolítását a ServiceNow, amikor a felhasználók csatlakoznak, áthelyezik vagy elhagyják a szervezetét.
- A biztonságos hozzáférés biztosítása **, szabályozása és figyelése** a ServiceNow a feltételes hozzáférési szabályzatok alkalmazásával, a hozzáférési felülvizsgálatokon keresztüli hozzáférés szabályozása és a bejelentkezési naplókhoz való hozzáférés figyelése.

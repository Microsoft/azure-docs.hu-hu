# <a name="sign-in-activity-report-error-codes"></a>Bejelentkezési tevékenység jelentés hibakódjai 

A [felhasználói bejelentkezések jelentésben](concept-sign-ins.md)megadott információk alapján választ kaphat az olyan kérdésekre, mint például:

- Ki jelentkezett be a jelentkezésembe?
- Mely alkalmazásokba jelentkeztek be?
- Mely bejelentkezések sikertelenek és miért?

Ha egy bejelentkezés sikertelen, megjelenik egy hibakód, amely megfelel a hibának. [Keresse meg a hibakód leírását.](https://login.microsoftonline.com/error)

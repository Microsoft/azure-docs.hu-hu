# <a name="set-up-an-office-phone-as-your-two-factor-verification-method"></a>Irodai telefon beállítása kétfaktoros ellenőrzési módszerként

Beállíthatja, hogy az irodai telefon kétfaktoros ellenőrzési módszerként működjön.

>[!Note]
> Ha az Office-telefon beállítás szürkén jelenik meg, előfordulhat, hogy a szervezet nem engedélyezi az irodai telefonszám használatát ellenőrzésre. Ebben az esetben másik módszert kell választania, vagy további segítségért forduljon a rendszergazdához.

## <a name="set-up-your-office-phone-number-as-your-verification-method"></a>Az irodai telefonszám beállítása ellenőrzési módszerként

1. A **További biztonsági ellenőrzés** lapon válassza az **Office-telefont** az **1.**

    ![További biztonsági ellenőrző oldal hitelesítési telefonhívással és telefonhívással](media/multi-factor-authentication-verification-methods/multi-factor-authentication-office-phone.png)

2. A Microsoft telefonhívást fog kapni, amelyben arra kéri, hogy nyomja meg az irodai telefonon található font (#) jelet a személyazonosságának ellenőrzéséhez.

    ![A megadott telefonszám tesztelése](media/multi-factor-authentication-verification-methods/multi-factor-authentication-office-phone-test.png)

3. **3. lépés: Továbbra is használja a meglévő alkalmazásterületet,** másolja a megadott alkalmazásjelszót, és illessze be egy biztonságos helyre.

    ![A További biztonsági ellenőrzés lap alkalmazásjelszavak területe](media/multi-factor-authentication-verification-methods/multi-factor-authentication-app-passwords.png)

    >[!Note]
    >Az alkalmazásjelszó régebbi alkalmazásokkal való használatáról az [Alkalmazásjelszavak kezelése című](multi-factor-authentication-end-user-app-passwords.md)témakörben talál további információt. Csak akkor kell alkalmazásjelszavakat használnia, ha továbbra is olyan régebbi alkalmazásokat használ, amelyek nem támogatják a kétfaktoros ellenőrzést.

4. Válassza a **Done** (Kész) lehetőséget.

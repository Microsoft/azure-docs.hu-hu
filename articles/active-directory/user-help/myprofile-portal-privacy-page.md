# <a name="view-your-privacy-related-info-from-the-my-profile-preview-portal"></a>Az adatvédelemmel kapcsolatos információk megtekintése a saját profil (előzetes verzió) portálján

Megtekintheti, hogyan használja a szervezet az adatokat a **saját profil** portál **Adatvédelem** oldaláról.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-privacy-related-info"></a>Adatvédelemmel kapcsolatos információk megtekintése

1. Jelentkezzen be a munkahelyi vagy iskolai fiókjába, majd lépjen a https://myprofile.microsoft.com/ lapra.

2. Válassza az **Adatvédelem** lehetőséget a bal oldali navigációs ablaktáblán, vagy az **adatvédelmi blokkban** válassza az adatvédelem **áttekintése** hivatkozást.

    ![Saját profil oldal, Kiemelt adatvédelmi hivatkozások megjelenítése](media/myprofile-portal/myprofile-portal-privacy.png)

3. Tekintse át az információkat, beleértve a következőket:

    - **Services.** A munkahelyi vagy iskolai fiókkal összekapcsolt online szolgáltatások listája.

    - **Használati feltételek**. A szervezet használati feltételei.

## <a name="next-steps"></a>További lépések

- Miután megtekintette, hogy a szervezet hogyan használja az adatokat, kiválaszthatja a [biztonsági](user-help-security-info-overview.md)adatok, a csatlakoztatott [eszközök](myprofile-portal-devices-page.md)vagy a [szervezetek](myprofile-portal-organizations-page.md)megtekintését és kezelését.

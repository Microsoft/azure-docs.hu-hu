# <a name="azure-kinect-support-options-and-resources"></a>Az Azure Kinect támogatási lehetőségei és erőforrásai

Ez a cikk végigvezeti a különböző támogatási lehetőségeken.

## <a name="community-support"></a>Közösségi támogatás

A nyilvános fórumokon több módon is választ kaphat kérdéseire:

- [StackOverflow](https://stackoverflow.com/search?q=azurekinect&s=3b855ed0-8564-4961-856f-9614aeab4c0d&s=fd9ea920-622c-4d8e-b908-ec996e1f1403), ahol kérdéseket tehet fel vagy kereshet a meglévő kérdések között.
- [GitHub](https://github.com/Microsoft/Azure-Kinect-Sensor-SDK), ahol kérdéseket tehet fel, megnyithat új hibákat, vagy hozzájárulhat az Azure Kinect Sensor SDK fejlesztéséhez.
- [Visszajelzéseket adhat](https://feedback.azure.com/forums/920053-azure-kinect-dk)meg, ahol megoszthatja ötleteit a termék jövőjével kapcsolatban, és szavazhat egy meglévő ötletről.

## <a name="assisted-support"></a>Támogatott támogatás

Az Azure Kinect támogatásának több módja is van.

### <a name="microsoft-qa"></a>Microsoft Q&A

A Microsoft mérnököktől, az Azure legértékesebb szakemberektől (MVP), illetve a szakértői Közösségtől érkező technikai kérdésekkel kapcsolatos gyors és megbízható válaszokért forduljon hozzánk a [Microsoft Q&a](/answers/products/azure), az Azure előnyben részesített céljaként a közösségi támogatáshoz.

- [Microsoft Q&a for Azure Kinect](/answers/topics/azure-kinect-dk.html), ahol kérdéseket tehet fel vagy kereshet a meglévő kérdések között.

### <a name="development-azure-kinect-on-azure"></a>Azure Kinect fejlesztése az Azure-ban

Az Azure-előfizetők támogatási kérelmeket hozhatnak létre és kezelhetnek a Azure Portalban. A Body Tracking, az Sensor SDK, a Speech Device SDK vagy az Azure Cognitive Services egy fejlesztéses támogatása elérhető Azure-előfizetők számára az előfizetéshez társított [Azure-támogatási csomaggal](https://azure.microsoft.com/support/plans/) .

  - Azure- [támogatási csomagot](https://azure.microsoft.com/support/plans/) társít az Azure-előfizetéséhez? Incidens beküldéséhez  [Jelentkezzen be Azure Portalba](https://ms.portal.azure.com/) .
  - Azure-előfizetésre van szüksége? Az [Azure-előfizetési lehetőségek](https://azure.microsoft.com/pricing/purchase-options/) további információkat biztosítanak a különböző lehetőségekről.
  - Támogatási csomagra van szüksége? [Támogatási csomag kiválasztása](https://azure.microsoft.com/support/plans/)

### <a name="azure-kinect-on-premises-or-other-cloud-services"></a>Azure Kinect helyszíni vagy más felhőalapú szolgáltatások

A helyszíni Sensor SDK-val és a Body Tracking SDK-val kapcsolatos technikai támogatásért nyissa meg a [Microsoft támogatási portálon](https://support.microsoft.com/supportforbusiness/productselection?sapId=c49ea5bb-2b09-8612-be35-d55159732667)a professzionális támogatási jegyet.

### <a name="azure-kinect-dk-device"></a>Azure Kinect DK-eszköz

Mielőtt felveszi a kapcsolatot a hardver támogatásával, győződjön meg arról, hogy az Azure Kinect DK beállítása és frissítése megtörtént. Ha tesztelni szeretné, hogy az eszköz működik-e, használja az [Azure Kinect Viewert](azure-kinect-viewer.md). Tudjon meg többet az [Azure Kinect DK súgójának](./index.yml) oldaláról.
Érdemes lehet megtekinteni az [ismert problémákat és a hibaelhárítást](troubleshooting.md)is.

[Segítség kérése](https://support.microsoft.com/supportforbusiness/productselection?sapId=f77b1b95-721e-43a0-2db8-b01e81a3f813) az eszköz vagy az érzékelő funkcióinak, a belső vezérlőprogram frissítéseinek és a vásárlási lehetőségeknek.

A támogatási ajánlatokkal kapcsolatos további információkért tekintse meg a [Microsoft támogatási részlegét](https://support.microsoft.com/help/4341255/support-for-business).

# <a name="what-is-azure-database-migration-service"></a>Mi az Azure Database Migration Service?

Azure Database Migration Service egy teljes körűen felügyelt szolgáltatás, amely lehetővé teszi a zökkenőmentes migrálást a több adatbázis-forrásokhoz, az Azure data platformokra minimális állásidővel (online áttelepítések).

## <a name="migrate-databases-to-azure-with-familiar-tools"></a>Adatbázisok migrálása az Azure-bA a jól ismert eszközökkel

Azure Database Migration Service integrálható a meglévő eszközökkel és a szolgáltatások működésének néhány. Ügyfelek biztosít egy átfogó, magas rendelkezésre állású megoldás. A szolgáltatás által használt a [Data Migration Assistant](https://aka.ms/dma) , amely végigvezeti a szükséges áttelepítés előtt módosításokat javaslatokkal elemzéseket és jelentéseket létrehozni. Van arra, hogy minden szükséges szervizelés végrehajtása. Ha készen áll az áttelepítési folyamat megkezdéséhez, Azure Database Migration Service hajt végre a szükséges lépéseket. Indul el, és felejtse el a migrálási projektek, a biztonságot, tudván, hogy a folyamat kihasználja a Microsoft meghatározása szerint ajánlott eljárásokat.

> [!NOTE]
> Azure Database Migration Service egy online migrálás végrehajtásához használatához hozzon létre egy példányt prémium tarifacsomagban alapján.

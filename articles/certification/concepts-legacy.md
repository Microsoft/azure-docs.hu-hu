# <a name="legacy-devices-on-the-azure-certified-device-catalog"></a>Örökölt eszközök az Azure Certified-eszköz katalógusában

Előfordulhat, hogy észrevette az Azure Certified-eszköz katalógusában, hogy egyes eszközök nem rendelkeznek kék körvonalsal vagy az "Azure Certified Device" címkével. Ezek az eszközök korábban az örökölt program keretében lettek hitelesítve.

## <a name="certified-for-azure-iot-program"></a>Minősítés az Azure IoT programhoz

Az Azure Certified eszköz program elindítása előtt a hardveres partnerek korábban igazolják termékeiket a Certified for Azure IoT program keretében. Az Azure Certified eszköz minősítési programja arra a küldetésre összpontosít, hogy a technikai eszköz képességei helyett az ügyfelek ígéreteit adja.

Az "IoT Hub tanúsított eszköz" minősítéssel rendelkező eszközök "örökölt eszközként" jelennek meg az Azure Certified-eszköz katalógusában. Ez a címke azt jelzi, hogy az eszközök korábban már ki lettek minősítve a már kivont programon keresztül, de nem tanúsították a frissített Azure Certified Device programon keresztül. Ezeket az eszközöket a katalógusban egyértelműen fel kell tüntetni a kék körvonalak hiányában, és a "IoT Hub Certified Devices (Legacy)" szűrőn keresztül találhatók.

# <a name="what-is-the-azure-certified-device-program"></a>Mi az az Azure Certified Device program?

Köszönjük, hogy érdeklődik az Azure Certified Device program iránt! Az Azure Certified Device egy ingyenes program, amely lehetővé teszi az Azure-on való futtatáshoz készült IoT-eszközök különböziksé, minősítéséhez és népszerűsítéséhez. Az intelligens kameráktól a csatlakoztatott érzékelőkön át a peremhálózati infrastruktúráig ez a továbbfejlesztett IoT-eszközminősítő program segít az eszközkészítőknek növelni a termék láthatóságát, és időt takarít meg az ügyfeleknek a megoldások kiépítése során.

## <a name="our-certification-promise"></a>A tanúsítási ígéretünk

Az Azure Certified Device program biztosítja, hogy az ügyfélmegoldások nagyszerűen működnek az Azure-ban. Ez egy olyan program, amely eszközöket, szolgáltatásokat és katalógust használ az iparági ismereteknek az IoT-ökoszisztémán belüli szerkesztői közösségekkel való megosztásához, így segítve a szerkesztőket és az ügyfeleket egyaránt.

Az eszköztanúsítási folyamat során a program három célja a következő:

- **Az ügyfelek bizalmának megadva:** Az ügyfelek magabiztosan vásárolhatnak Olyan Azure-tanúsítvánnyal rendelkező eszközöket, amelyek a Microsoft ígéretét hordozják.

- **Ügyfelek egyeztetése a számukra megfelelő eszközökkel:** Az eszközkészítők az egyedi képességeiket kiemelt minősítéssel külön-külön állíthatják be, és az ügyfelek könnyedén megtalálják az igényeiknek megfelelő IoT-minősített eszközöket.

- **Minősített eszközök népszerűsítése:** Az eszközkészítők nagyobb láthatóságot, az ügyfelekkel való kapcsolatot és a Microsoft Azure Certified Device márka használatát biztosítják.

## <a name="user-roles"></a>Felhasználói szerepkörök

Az Azure Certified Device program két különböző célközönséget szolgál ki.

1. **Eszközkészítők:** Építenek IoT-eszközöket? Könnyedén megkülönböztetheti IoT-eszközképességeit, és hozzáférést nyerhet egy globális célközönséghez, akik megbízhatóan megvásárolják az Azure-ban futtatott eszközöket. Az Azure minősített eszközkatalógusának használatával növelheti a termékek láthatóságát, és az eszköz minősítésével csatlakozhat az ügyfelekhez, és megmutathatja, hogy megfelel-e az adott IoT-eszközkövetelményeknek.
1.  **Megoldáskészítők:** Szeretné tudni, mik azok az IoT-minősített eszközök? Az Azure-ban futtatott IoT-eszközök magabiztos megkereshetőek és megvásárolhatók, tudva, hogy megfelelnek bizonyos IoT-követelményeknek. Könnyedén megkeresheti és kiválaszthatja az IoT-megoldáshoz megfelelő tanúsított eszközt az [Azure Certified Device catalog (Azure minősített eszközkatalógus) oldalon.](https://devicecatalog.azure.com/)

## <a name="our-certification-programs-and-iot-device-requirements"></a>Tanúsítási programjaink és IoT-eszközkövetelményeink.

Jelenleg négy különböző tanúsítvány érhető el! Minden tanúsítvány célja más ügyfélértékek kézbesítése. Az eszköz típusától és a célközönségtől függően kiválaszthatja, hogy mely minősítés(eket) kell a leginkább alkalmaznia. A programokkal és az IoT-követelményekkel kapcsolatos további információkért válassza ki az egyes programok címeit.

| Tanúsítási program         |  Áttekintés                      |
------------------------------|-------------------------------------------------|
| [Azure-tanúsítvánnyal rendelkező eszköz](program-requirements-azure-certified-device.md)          | Az Azure-tanúsítvánnyal rendelkező eszköztanúsítás ellenőrzi, hogy az eszköz Azure IoT Hub és biztonságosan kiépíthető-e a Device Provisioning Service (DPS) használatával. Ez a tanúsítvány az eszköz funkcióit és együttműködését tükrözi, amelyek **minden** más tanúsítványhoz szükséges alapkonfigurációk.          |
| [IoT Plug and Play](program-requirements-pnp.md) | Az IoT Plug and Play tanúsítványt, amely az Azure Certified Device alapkonfiguráción túli növekményes tanúsítvány, az eszközmodell alapján ellenőrzi a Digital Twin Definition Language 2. verzióját (DTDL) és az interakciót. Zökkenőmentes eszköz-felhő integrációt tesz lehetővé, és lehetővé teszi a hardverpartnerek számára olyan eszközök felépítését, amelyek zökkenőmentesen integrálhatók anélkül, hogy egyéni kódot kellene írniuk.  |
| [Edge Managed](program-requirements-edge-managed.md) | A Peremhálózati felügyelt tanúsítvány, amely az alapkonfiguráción túli, növekményes minősítés, az Azure-beli csatlakoztatott eszközök eszközkezelési szabványára összpontosít.  |
| [Edge Secured Core](program-requirements-edge-secured-core.md)                             | Az Edge által biztosított alaptanúsítás, amely az Azure Certified Device alapkonfiguráción túli növekményes minősítés, teljes operációs rendszert, például Linuxot vagy Windows 10 IoT-t futtató IoT-eszközökhöz használható. Ellenőrzi, hogy az eszközök megfelelnek-e az eszközidentitásra, a biztonságos rendszerindításra, az operációs rendszerek védelmére, az eszközfrissítésre, az adatvédelemre és a biztonsági rések felfedésére vonatkozó további biztonsági követelményeknek. |

## <a name="how-to-certify-your-device"></a>Az eszköz tanúsítása

Az eszközök minősítése több fő lépésből áll az [Azure Certified Device portálon:](https://certify.azure.com)

1. Válassza ki az eszközhöz megfelelő minősítést az IoT-eszközkövetelmények alapján.
1. Hozza létre a projektet az [Azure Certified Device portálon.](https://certify.azure.com)
1. Adja meg az eszköz adatait, beleértve a hardverképességgel kapcsolatos információkat az eszköztanúsítási folyamat megkezdéséhez.
1. Eszközfunkciók ellenőrzése
1. A felülvizsgálati folyamat elküldése és befejezése

> [!Note]
> A felülvizsgálati folyamat akár egy hetet is igénybe vehet, de néha tovább is tarthat.

Az eszköz minősítése után az alábbi két tevékenység közül választhat: 

1. Közzététel az Azure minősített eszközkatalógusában (nem kötelező)
1. A projekt frissítése a jóváhagyás/közzététel után (nem kötelező)

# <a name="migrate-to-azure-app-service"></a>Migrálás Azure App Servicere

[App Service Migration Assistant](https://azure.microsoft.com/services/app-service/migration-assistant/) használatával áttelepítheti a helyszíni alkalmazást a Azure app Servicere. App Service Migration Assistant úgy lett kialakítva, hogy az ingyenes, egyszerű és gyors megoldást kínáljon a felhőbe való áttérésre, hogy az alkalmazások a helyszínről a felhőbe legyenek áttelepítve.

A Azure App Service Migration Assistant használatával gyorsan elvégezheti a következőket:

- Az alkalmazás URL-címének vizsgálata annak megállapításához, hogy jó-e az áttelepítéshez
- A migrálás megkezdéséhez töltse le a Migration Assistant.
- Az eszköz használata a készültségi ellenőrzések futtatásához és az alkalmazás konfigurációs beállításainak általános értékeléséhez
- Telepítse át az alkalmazást vagy a webhelyet Azure App Service az eszközön keresztül.

[Tekintse át a webalkalmazások Azure app Service-be való áttelepítésének módját](https://www.youtube.com/watch?v=9LBUmkUhmXU).

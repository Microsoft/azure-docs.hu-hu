# <a name="how-to-delete-an-invitation-to-a-recipient-in-azure-data-share"></a>Címzettnek szóló meghívás törlése az Azure Data Share szolgáltatásban

Ebből a cikkből megtudhatja, hogy miként törölheti a címzettnek szóló meghívót, mielőtt elfogadná azt. Ez megakadályozza őket abban, hogy megosztási előfizetést hozzanak létre, és soha nem fognak megosztott adatokat kapni. Ha törli a meghívót egy olyan fogyasztónak, aki már létrehozott egy megosztási előfizetést, a meghívó törlésének nincs hatása.

## <a name="navigate-to-a-sent-data-share"></a>Navigálás elküldött adatmegosztásra

Az Azure Data Share-ben keresse meg az elküldött megosztást, és válassza a **Meghívók** lapot. Kattintson azon címzettek melletti mezőkre, akiknek a meghívóit törölni szeretné, majd kattintson a **Törlés gombra.**

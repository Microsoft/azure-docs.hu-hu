# <a name="billing-questions"></a>Számlázással kapcsolatos kérdések

## <a name="how-does-billing-work-for-azure-security-center"></a>Hogyan működik a számlázás az Azure Security Centerben?
A Security Center szolgáltatást két szinten biztosítjuk:

- Az **ingyenes szint** betekintést nyújt az Azure-erőforrások biztonsági állapotába, az alapvető biztonsági házirendbe, a biztonsági javaslatokba, valamint a partnerek biztonsági termékeivel és szolgáltatásaival való integrációba.

- A **standard szint** olyan veszélyforrások elleni védelmi képességeket ad hozzá, amelyek biztonsági riasztásokat, fenyegetésfelderítést, viselkedéselemzést, anomáliaészlelést és fenyegetéshozzárendelési jelentéseket tartalmaznak. Elindíthategy szabványos szintű ingyenes próbaverziót. A frissítéshez válassza [a Tarifacsomag](https://docs.microsoft.com/azure/security-center/security-center-pricing) lehetőséget a biztonsági házirendben. További részletekért tekintse át az [árképzést ismertető oldalt](https://azure.microsoft.com/pricing/details/security-center/).

## <a name="how-can-i-track-who-in-my-organization-performed-pricing-tier-changes-in-azure-security-center"></a>Hogyan követhetem nyomon, hogy a szervezetben ki hajtotta végre a tarifacsomag-módosításokat az Azure Security Centerben?
Az Azure-előfizetések több rendszergazdai engedéllyel módosíthatja a tarifacsomag. Ha meg szeretné tudni, hogy melyik felhasználó hajtotta végre a tarifacsomag-módosítást, használja az Azure-tevékenységnaplót. További információ: [itt](https://techcommunity.microsoft.com/t5/Security-Identity/Tracking-Changes-in-the-Pricing-Tier-for-Azure-Security-Center/td-p/390832).

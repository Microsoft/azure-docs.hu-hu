# <a name="azure-policy-built-in-policy-definitions-for-azure-virtual-machine-scale-sets"></a>Az Azure virtuálisgép-méretezési csoportok beépített szabályzat-definícióinak Azure Policy

Ez az oldal az Azure virtuálisgép-méretezési csoportok beépített szabályzat-definícióinak [Azure Policy](../governance/policy/overview.md) indexe. További Azure Policy a más szolgáltatásokhoz készült beépített [definíciók: Azure Policy beépített definíciók](../governance/policy/samples/built-in-policies.md).

Az egyes beépített szabályzat-definíciók neve a Azure Portal házirend-definíciójának nevére hivatkozik. A **GitHub** oszlopban található hivatkozás használatával megtekintheti a forrást a [Azure Policy GitHub](https://github.com/Azure/azure-policy)-tárházban.

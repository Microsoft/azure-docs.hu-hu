# <a name="solutions-and-training-for-azure-active-directory-b2c"></a>Megoldások és képzés az Azure Active Directory B2C-hez

Az Azure Active Directory B2C (Azure AD B2C) lehetővé teszi a szervezetek számára, hogy olyan üzleti megoldásokat valósítsanak meg, amelyek segítenek az ügyfeleikkel való kapcsolattartásban. A következő megoldási útmutatók és betanításletölthető dokumentumok, amelyek végigvezetik ezeket a megoldásokat.

| Cím | Leírás |
| ----- | ----------- |

| [Szakértelme az Azure AD B2C-ben: Tanfolyam fejlesztőknek](https://aka.ms/learnAADB2C) | Ez a teljes körű tanfolyam a fejlesztők et az Azure AD B2C hitelesítési mechanizmusként való fejlesztésével egészül ki. Tíz részletes modul laborokkal mindent lefed az Azure-előfizetés beállításától kezdve az egyéni szabályzatok létrehozásáig, amelyek meghatározzák az ügyfeleket vonzó utazásokat. |
| [Partnerek, beszállítók és ügyfelek hozzáférésének engedélyezése az Azure Active Directoryval](https://aka.ms/aadexternalidentities) | Minden szervezet sikere, méretétől, iparágától vagy megfelelőségi és biztonsági helyzetétől függetlenül, a szervezeti képességen alapul, hogy együttműködjön más szervezetekkel és kapcsolatba lépjen az ügyfelekkel.<br><br>Az Azure AD, az Azure AD B2C és az Azure AD B2B Collaboration együttesen ismerteti az üzleti értéket és az olyan alkalmazás vagy webes élmény létrehozásának mechanizmusait, amelyek a környezetéhez igazított összevont hitelesítési élményt biztosítnak alkalmazottak, üzleti partnerek és beszállítók, valamint az ügyfelek számára. |

| [Általános adatvédelmi rendelet (GDPR) Az ügyfelekkel szemben imát leadott kérelmekkel kapcsolatos megfontolások](https://aka.ms/AADB2CandGDPR) | Minden olyan ügyfélesetében, amellyel alkalmazások szembesülnek, a GDPR-t minden olyan szervezetnek figyelembe kell vennie, amely személyes adatokat tartalmazó és uniós polgárokat kiszolgáló projektekbe kezd.<br><br>Ez a megoldásútmutató arra összpontosít, hogy az Azure AD B2C hogyan használható a teljes GDPR-megfelelőségi megközelítés rugalmas elemeként, beleértve azt is, hogy az Azure AD B2C összetevői hogyan támogatják az egyének számára az egyes kulcsfontosságú GDPR-jogokat. |

# <a name="help-and-support-options-for-azure-communication-services"></a>Súgó és támogatás lehetőségek az Azure kommunikációs szolgáltatásokhoz

Ha probléma merül fel, vagy kérdése van az Azure kommunikációs szolgáltatások bevonásával kapcsolatban, akkor számos lehetősége van.

## <a name="create-an-azure-support-request"></a>Azure-támogatási kérelem létrehozása
Az Azure-ban számos [támogatási lehetőség és csomag](https://azure.microsoft.com/support/plans/) érhető el, amelyeket felderítheti és áttekintheti. Támogatási jegyet a [Azure Portal](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview)hozhat létre.

## <a name="post-a-question-to-microsoft-qa"></a>Kérdés közzététele a Microsoft Q&A

Ha szeretne gyorsan és megbízhatóan választ adni a termékre vagy technikai kérdésekre, előfordulhat, hogy a Microsoft-mérnököktől, az Azure-ban a legértékesebb szakemberektől (MVP) vagy a Közösségtől származó Azure kommunikációs szolgáltatásokkal foglalkozunk, a [Microsoft Q&a](https://docs.microsoft.com/answers/products/azure). 

Ha nem talál választ a problémára a kereséssel, küldjön új kérdést a Microsoft Q&A alkalmazásnak. Kérdés létrehozásakor ügyeljen arra, hogy az [Azure kommunikációs szolgáltatások címkét](https://docs.microsoft.com/answers/topics/azure-communication-services.html)használja.

## <a name="post-a-question-on-stack-overflow"></a>Kérdés közzététele Stack Overflow

Kipróbálhatja a kérdéseit Stack Overflow is, amely egy nagy közösségi fejlesztői Közösséggel és ökoszisztémával rendelkezik. Az Azure kommunikációs szolgáltatásainak van egy [dedikált címkéje](https://stackoverflow.com/questions/tagged/azure-communication-services) is.

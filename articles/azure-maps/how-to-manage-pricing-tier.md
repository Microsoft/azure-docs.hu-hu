# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Az Azure Maps-fiók tarifacsomagja kezelése

Kezelheti az Azure Portalon az Azure Maps-fiók tarifacsomagját. Miután létrehozott egy [fiók](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), megtekintheti vagy módosíthatja a fiókhoz tartozó tarifacsomagot.

Tarifacsomag, a jobb kiválasztásával kapcsolatban további információt talál, [jobb tarifacsomag kiválasztása](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier)

## <a name="view-pricing-tier"></a>A tarifacsomag megtekintése

A választott tarifacsomag megtekintéséhez lépjen a **Tarifacsomag** lehetőséget a beállítások menüben.

![Kiválasztott tarifacsomag megtekintése](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>A tarifacsomag módosítása

Miután létrehozta az Azure Maps-fiók, frissítése, vagy az Azure Maps-fiók tarifacsomagja visszaminősítésére. Frissítés vagy visszaminősítését, keresse meg a **Tarifacsomag** lehetőséget a beállítások menüben, majd kattintson a tarifacsomag nem kiemelt. Kattintson a Mentés gombra kattintva mentse a választott díjszabási csomag lehetőséget.

![A tarifacsomag módosítása](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Új előfizetési kulcsok előállításához, ha be- és Visszaléptetés a a tarifacsomagot az Azure Maps-fiók nem rendelkezik.

# <a name="manage-the-pricing-tier-of-your-azure-maps-account"></a>Az Azure Maps-fiók tarifacsomagját kezelése

Kezelheti az Azure Portalon az Azure Maps-fiók tarifacsomagját. Miután létrehozott egy [fiók](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), megtekintheti vagy módosíthatja a fiókhoz tartozó tarifacsomagot.

További információk [a jobb oldalon, az Azure Maps-tarifacsomag kiválasztása](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier).

## <a name="view-your-pricing-tier"></a>A tarifacsomag megtekintése

A választott tarifacsomag megtekintéséhez lépjen a **Tarifacsomag** lehetőséget a beállítások menüben.

![Kiválasztott tarifacsomag megtekintése](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-a-pricing-tier"></a>Tarifacsomag módosítása

Az Azure Maps-fiók létrehozását követően frissítse, vagy az Azure Maps-fiók tarifacsomagja visszaminősítésére. Frissítés vagy visszaminősítését, keresse meg a **Tarifacsomag** lehetőséget a beállítások menüben. Válassza ki a tarifacsomagot, amely nincs kijelölve. Válassza ki a **mentése** gombra kattintva mentse a választott díjszabási csomag lehetőséget.

![Tarifacsomag módosítása](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!NOTE]
> Nem kell létrehozni új előfizetési kulcsok vagy ügyfél-Azonosítót (az Azure AD-hitelesítés), ha be- és Visszaléptetés a a tarifacsomagot az Azure Maps-fiók.

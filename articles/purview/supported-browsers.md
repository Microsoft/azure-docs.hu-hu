# <a name="supported-browsers"></a>Támogatott böngészők 

Az Azure-beli hatáskörébe a következő böngészők használhatók. Javasoljuk, hogy használja az operációs rendszerével kompatibilis böngésző legújabb verzióját. 

* Microsoft Edge (legújabb verzió)
* Safari (csak Mac, legújabb verzió)
* Chrome (legújabb verzió)
* Firefox (legújabb verzió)

## <a name="chrome-incognito-mode"></a>Krómozott inkognitóban mód

 A Chrome inkognitóban blokkolja a harmadik féltől származó cookie-kat, hogy azok működjenek a hatáskörébe Studióban.

:::image type="content" source="./media/supported-browsers/incognito-chrome.png" alt-text="Képernyőfelvétel a Chrome-ról.":::

## <a name="chromium-edge-inprivate-mode"></a>Króm Edge InPrivate-üzemmód

A belső króm Edge-et a szigorú nyomkövetés-megelőzés használatával le kell tiltani a hatáskörébe Studio működéséhez.

:::image type="content" source="./media/supported-browsers/incognito-edge.png" alt-text="Képernyőkép a peremhálózat megjelenítéséről.":::

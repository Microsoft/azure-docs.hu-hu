# <a name="make-resources-more-secure-with-azure-advisor"></a>Az Azure Advisor segítségével biztonságosabbá teheti az erőforrásokat

Az Azure Advisor egységes, összevont nézetet biztosít az összes Azure-erőforrással kapcsolatos javaslatokról. Az Azure Security Centerrel integrálva biztonsági javaslatokat nyújt. A biztonsági javaslatokat az Advisor **irányítópultján** található Biztonság lapon kaphatja meg.

A Security Center az Azure-erőforrások biztonsági felügyeletének átláthatóbbá és szabályozhatóbbá tételével megkönnyíti a fenyegetések megelőzését, észlelését és elhárítását. Rendszeresen elemzi az Azure-erőforrások biztonsági állapotát. A Security Center javaslatokat hoz létre, amikor lehetséges biztonsági réseket észlel. A javaslatok végigvezetik a szükséges vezérlők konfigurálásának folyamatán. 

A biztonsági javaslatokról az [Azure Security Center biztonsági javaslatainak kezelése](https://azure.microsoft.com/documentation/articles/security-center-recommendations/)című témakörben talál további információt.

## <a name="how-to-access-security-recommendations-in-azure-advisor"></a>A biztonsági javaslatok elérése az Azure Advisorban

1. Jelentkezzen be az [Azure Portalra](https://portal.azure.com), majd nyissa meg [az Advisor](https://aka.ms/azureadvisordashboard)t.

2.  Az Advisor irányítópultján kattintson a **Biztonság** fülre.

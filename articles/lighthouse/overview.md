# <a name="what-is-azure-lighthouse"></a>Mi az Azure világítótoronyig?

Az Azure világítótoronyig kínál szolgáltatók egy egyetlen vezérlősík megtekintése és kezelése az Azure között ügyfeleik magasabb szintű automation, a méretezési csoport és a továbbfejlesztett cégirányítási. Azure világítótoronyig átfogó és hatékony eszközök az Azure platform beépített használatával felügyelt szolgáltatások szolgáltatói is szolgáltatásként. Ezt az ajánlatot is hasznot vállalati informatikai szervezetek több bérlőre kiterjedő-erőforrások kezeléséhez.

![Az Azure világítótoronyig – áttekintő ábra](media/azure-lighthouse-overview.jpg)

## <a name="benefits"></a>Előnyök

Azure világítótoronyig többieknél és hatékonyan hozhatók létre, és az ügyfelek számára a felügyelt szolgáltatások segítségével. Az előnyök:

- **A felügyeletet**: Customer engagement és életciklus-ügyfél-erőforrások kezeléséhez műveletekre egyszerűbb és jobban skálázható.
- **Nagyobb láthatóságot és az ügyfelek pontosság**: Ügyfelek, amelynek felügyel erőforrásokat kell nagyobb betekintést a műveletek és a hatókör pontosan szabályozhatja azok delegálása Management, közben a rendszer megőrzi az IP.
- **Átfogó és egységes platform eszközök**: A fejlesztőeszközeit helyzetekre kulcsos szolgáltató, többek között például az EA, CSP és az utólagos elszámolású több licencmodellek szükségesek. Az új képességek együttműködik a meglévő eszközökkel és API-kkal, modelleket, licencelési és a programok például partneri a [Cloud Solution Provider (CSP) program](https://docs.microsoft.com/partner-center/csp-overview). Az Azure világítótoronyig beállításoktól integrálható a meglévő folyamatok és az alkalmazások és az ügyfélesetekből szerint követheti nyomon a hatás [a Partnerazonosító csatolása](https://docs.microsoft.com/azure/billing/billing-partner-admin-link-started).

Nincsenek további költségek nélkül társított Azure világítótoronyig használatával az ügyfelek Azure-erőforrások kezeléséhez.

## <a name="capabilities"></a>Funkciók

Az Azure világítótoronyig customer engagement és a felügyeleti csomagokba több lehetőség is tartalmazza:

- **Az Azure resource management delegált**: Az ügyfelek Azure-erőforrások kezeléséhez biztonságosan a saját bérlőn belül környezetet és-irányítási kapcsoló nélkül. További információ: [Azure erőforrás-kezelés delegált](./concepts/azure-delegated-resource-management.md).
- **Új Azure Portalon élmény**: Több-bérlős adatok megtekintése az új **ügyfeleim** lapját a [az Azure portal](https://portal.azure.com). Egy megfelelő **szolgáltatók** panel lehetővé teszi, hogy az ügyfelek tekintheti meg és kezelheti a service provider hozzáférést. További információ: [megtekintése és kezelheti az ügyfelek](./how-to/view-manage-customers.md) és [megtekintése és kezelheti a szolgáltatók](./how-to/view-manage-service-providers.md).
- **Az Azure Resource Manager-sablonok**: Könnyen végrehajthat felügyeleti feladatokat, beleértve a felhasználók beléptetése az Azure erőforrás-kezelés delegált. További információ: a [tárház minták](https://github.com/Azure/Azure-Lighthouse-samples/tree/master/Azure-Delegated-Resource-Management/templates) és [felvétele a az Azure-bA egy ügyfél meghatalmazott erőforrás-kezelés](how-to/onboard-customer.md).
- **Felügyelt szolgáltatásokat kínál az Azure Marketplace-en**: A szolgáltatások ajánlása a magán- vagy nyilvános ajánlatokat ügyfeleket, valamint azok automatikusan előkészítve az Azure-delegált erőforrások kezelése, Azure Resource Manager-sablonok használatával bevezetési alternatívájaként. További információ: [felügyelt szolgáltatásokat kínál az Azure Marketplace-en](./concepts/managed-services-offers.md).
- **Azure által felügyelt alkalmazások**: Csomag és a szállítási alkalmazások egyszerűen, az ügyfelek üzembe helyezése és használata a saját előfizetések számára. Helyezzen egy erőforráscsoportban elérni a bérlő, ami lehetővé teszi a szolgáltatás kezelése az Azure világítótoronyig összteljesítmény részeként a rendszer telepíti az alkalmazást. További információ: [Azure managed applications áttekintése](https://docs.microsoft.com/azure/managed-applications/overview).

> [!NOTE]
> A fent leírt funkciók jelenleg érhető el a nyilvános felhő. Régiónkénti rendelkezésre állás az egyes szolgáltatások, lásd: [elérhető termékek régiók szerint](https://azure.microsoft.com/global-infrastructure/services/).

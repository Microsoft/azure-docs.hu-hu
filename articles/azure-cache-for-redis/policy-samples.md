# <a name="azure-policy-built-in-policy-definitions-for-azure-cache-for-redis"></a>Az Azure cache Redis-hez készült beépített szabályzat-definícióinak Azure Policy

Ez az oldal [Azure Policy](../governance/policy/overview.md) beépített szabályzat-definíciók indexe a Redis készült Azure cache-hez. További Azure Policy a más szolgáltatásokhoz készült beépített [definíciók: Azure Policy beépített definíciók](../governance/policy/samples/built-in-policies.md).

Az egyes beépített hivatkozások neve a Azure Portal házirend-definíciójában. **A forrás oszlopban található** hivatkozás használatával megtekintheti a forrást a [Azure Policy GitHub](https://github.com/Azure/azure-policy)-tárházban.

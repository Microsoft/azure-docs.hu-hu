# <a name="what-is-microsoft-genomics"></a>Mi az a Microsoft Genomics?
A Microsoft Genomics a Burrows-Wheeler-igazítás (BWA) és a genom Analysis Toolkit (GATK) Felhőbeli implementációját kínálja a másodlagos elemzéshez. A szolgáltatás ISO-minősítéssel rendelkezik, és megfelel a HIPAA-előírásoknak, és a genom-szekvencia igényeinek megfelelő árat biztosít. Megtudhatja, hogyan használhatja a Microsoft Genomics szolgáltatást, és hogyan integrálhatja az API-t a rövid útmutatók, oktatóanyagok és dokumentáció beolvasásával.

## <a name="support-your-most-demanding-sequencing-needs"></a>A legigényesebb előkészítési igények támogatása
A saját adatközpontok kezelése helyett kihasználhatja a Microsoft méretének és tapasztalatának előnyeit a Exabyte-méretezési feladatok futtatásához. A BWA-GATK Felhőbeli megvalósítása nagy mértékben együttműködik a széles körű Intézet ajánlott eljárásainak folyamatával.


## <a name="keep-your-business-running"></a>Üzleti működésének fenntartása
A Microsoft Genomics 99,99%-os rendelkezésre állási szolgáltatói szerződést (SLA) biztosít a munkafolyamat-kérelmek fogadásához. További információ a Microsoft Genomics [SLA](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/)-ról.


## <a name="secure-your-data"></a>Az adatok védelme
A Microsoft Genomics szolgáltatás ISO 27001, ISO 27018 és ISO 9001 tanúsítvánnyal rendelkezik, és megfelel a HIPAA-előírásoknak. Az Azure platform biztonságáról az [Azure biztonsági és adatkezelési központban](https://www.microsoft.com/trustcenter/security) talál információkat.

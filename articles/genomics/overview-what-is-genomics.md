# <a name="what-is-microsoft-genomics"></a>Mi az a Microsoft Genomics?
A Microsoft Genomics a Burrows-Wheeler Aligner (BWA) és a Genoman Analysis Toolkit (GATK) felhőalapú megvalósítását kínálja másodlagos elemzésre. A szolgáltatás ISO-tanúsítvánnyal rendelkezik, és megfelel a HIPAA előírásoknak, és árkiszámíthatóságot kínál a genomszekvenálási igényeknek. Ismerje meg, hogyan használhatja a Microsoft Genomics szolgáltatást, és hogyan integrálható API-nkba a rövid útmutatóink, oktatóink és dokumentációink olvasásával.

## <a name="support-your-most-demanding-sequencing-needs"></a>Támogassa a legigényesebb szekvenálási igényeket
Ahelyett, hogy saját adatközpontokat kezelne, használja ki a Microsoft méretezési és tapasztalatát az exabájt-méretű számítási feladatok futtatásában. A BWA-GATK felhőalapú megvalósítása nagymértékben összhangban van a Broad Institute legjobb gyakorlatai val.


## <a name="keep-your-business-running"></a>Tartsa vállalkozását működésben
A Microsoft Genomics 99,99%-os rendelkezésre állási szolgáltatásiszint-szerződést (SLA) kínál a munkafolyamat-kérelmek fogadására. További információ a Microsoft Genomics [SLA-ról.](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/)


## <a name="secure-your-data"></a>Az adatok védelme
A Microsoft Genomics szolgáltatás ISO 27001, ISO 27018 és ISO 9001 tanúsítvánnyal rendelkezik, és megfelel a HIPAA előírásoknak. Az Azure platform biztonságáról az [Azure biztonsági és adatkezelési központban](https://www.microsoft.com/trustcenter/security) talál információkat.

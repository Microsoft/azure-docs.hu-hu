#### <a name="c-nuget-package"></a>C# NuGet-csomag

A .NET Speech SDK a következő paranccsal telepíthető a **a .net Core parancssori felületeból** `dotnet add` .

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

A .NET Speech SDK a következő paranccsal telepíthető a **Package managerből** `Install-Package` .

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```

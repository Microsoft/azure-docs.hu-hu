# <a name="what-is-the-linguistic-analysis-api-preview"></a>Mi a Linguistic Analysis API? (Előzetes verzió)

> [!IMPORTANT]
> A Linguistic Analysis előzetes verzióját 2018. augusztus 9-én visszavontuk. A szövegek feldolgozásához és elemzéséhet az [Azure Machine Learning szövegelemzési moduljait](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics) javasoljuk használni.

A Linguistic Analysis API természetes nyelvi feldolgozási (NLP) eszközökhöz biztosít hozzáférést a szövegek szerkezetének azonosításához. A jelenlegi kiadás az elemzés három típusát teszi lehetővé:

- [Mondatok szétválasztása és lexikális elemzése](Sentences-and-Tokens.md)
- [Beszédrészlet-címkézés](POS-tagging.md)
- [Vevőkör-elemzés](Constituency-Parsing.md)

## <a name="getting-started"></a>Első lépések

Miután megismerkedett a fent sorolt főbb elemzéstípusokkal, megismerheti, hogyan [kérdezhető le az elérhető elemzők listája](AnalyzersMethod.md) a végponttal, hogyan vannak ezek [elnevezve](Analyzer-Names.md), és végül az [új bemenet elemzésére szolgáló](AnalyzeMethod.md) végponttal.

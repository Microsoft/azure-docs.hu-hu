# <a name="import-and-export-data-reference"></a>Adatok importálása és exportálása hivatkozás

Tekintse át ezt az importálási és exportálási hivatkozást, hogy a legjobb eredményeket kapja a tudásbázis biztonsági mentéséhez, tárolásához és cseréjéhez.

## <a name="import-and-export-knowledge-base"></a>Tudásbázis importálása és exportálása

Az exportált tudásbázisokt tartalmazó **TSV- és XLS-fájlok**csak a QnA Maker portál **Beállítások** lapjáról importálhatók. Nem használhatók adatforrásként a tudásbázis létrehozása során, illetve a **Beállítások** lap **+ Fájl hozzáadása** vagy + URL **hozzáadása** szolgáltatásából.

1. Másolja a parancsot egy szövegszerkesztőbe.
2. Hajtsa végre a következő módosításokat a parancs megfelelő területein:
    1. Cserélje le az értéket a `<your-text-analytics-key-here>` kulcsával.
    2. Cserélje le a kérelem URL-címének első részét `<your-text-analytics-endpoint-here>` a saját végpont URL-címére.
3. Nyisson meg egy parancsablakot.
4. Illessze be a szövegszerkesztőből a parancsot, majd futtassa.

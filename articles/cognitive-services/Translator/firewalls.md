# <a name="how-to-translate-behind-ip-firewalls-with-the-translator-text-api"></a>Az IP-tűzfalak mögötti lefordítása a Translator Text API

Translator Text API a tűzfalak mögött is lefordíthatja a tartományneveket vagy az IP-szűrést. A tartománynév-szűrés az előnyben részesített módszer. A Microsoft Translatort **nem ajánlott** IP-szűréses tűzfal mögött futtatni. A telepítés várhatóan megszakadhat a jövőben, előzetes értesítés nélkül.

## <a name="translator-ip-addresses"></a>Translator IP Addresses
A api.cognitive.microsofttranslator.com IP-címei – a Microsoft Translator Text API 2019-as augusztus 21-én:

* **Ázsia és a Csendes-óceáni térség:** 20.40.125.208, 20.43.88.240, 20.184.58.62, 40.90.139.163, 104.44.89.44
* **Európa** 40.90.138.4, 40.90.141.99, 51.105.170.64, 52.155.218.251
* **Észak-Amerika:** 40.90.139.36, 40.90.139.2, 40.119.2.134, 52.224.200.129, 52.249.207.163

# <a name="how-to-translate-behind-ip-firewalls-with-the-translator-text-api"></a>Hogyan lehet lefordítani az IP-tűzfalak mögött a Translator Text API-val

A Translator Text API a tűzfalak mögött is lefordítható tartománynév vagy IP-szűrés használatával. A tartománynév-szűrés az előnyben részesített módszer. **Nem javasoljuk a** Microsoft Translator futtatását IP-szűrő tűzfal mögül. A beállítás valószínűleg szünet a jövőben előzetes értesítés nélkül.

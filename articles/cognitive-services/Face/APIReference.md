# <a name="face-api-reference-list"></a>Face API hivatkozási listája

Az Azure Face egy felhőalapú szolgáltatás, amely algoritmusokat biztosít az arcfelismeréshez és -felismeréshez. A Face API-k a következő kategóriákba sorolhatók:

- Arcalgoritmus API-k: Olyan alapvető funkciók at fednek le, mint [az észlelés,](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395236) [a hasonló keresés,](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395237) [az ellenőrzés,](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f3039523a) [az azonosítás](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)és [a csoport.](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395238)
- [FaceList API-k](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f3039524b): FaceList-ek kezelésére szolgál a [Hasonló keresése esetén.](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395237)
- [LargePersonGroup person API-k](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/599adcba3a7b9412a4d53f40): A LargePersonGroup személyarcainak [kezelésére](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)szolgál azonosításra.
- [LargePersonGroup API-k](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/599acdee6ac60f11b48b5a9d): LargePersonGroup adatkészlet [azonosításra](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)szolgáló kezelésére szolgál.
- [LargeFaceList API-k](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/5a157b68d2de3616c086f2cc): LargeFaceList kezelésére szolgál a [hasonló keresése esetén.](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395237)
- [PersonGroup person API-k](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f3039523c): A PersonGroup személyarcok azonosítására szolgáló [kezelésére](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)szolgál.
- [PersonGroup API-k](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395244): A PersonGroup azonosító adatkészletének [kezelésére](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)szolgál.
- [Pillanatkép API-k:](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/snapshot-take)Az előfizetések közötti adatáttelepítés pillanatképének kezelésére szolgál.

# <a name="api-reference"></a>API-hivatkozás

Az Azure Face API egy olyan felhőalapú API, arcfelismerés és felismerés algoritmusokat biztosító. A Face API-k tartalmazzák a következő kategóriákban:

- [A Face API algoritmus](https://docs.microsoft.com/rest/api/cognitiveservices/face/face): Például ismerteti az alapvető funkciók [észlelési](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/detectwithstream), [hasonló](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar), [ellenőrzési](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/verifyfacetoface), [azonosító](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify), és [csoport ](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/group).
- [FaceList felügyeleti API-k](https://docs.microsoft.com/rest/api/cognitiveservices/face/facelist): Az egy FaceList kezelésére szolgáló [hasonló](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar).
- [LargePersonGroup személy felügyeleti API-k](https://docs.microsoft.com/rest/api/cognitiveservices/face/largepersongroupperson): Segítségével felügyelhető az arcokat LargePersonGroup, ha a [azonosító](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [LargePersonGroup felügyeleti API-k](https://docs.microsoft.com/rest/api/cognitiveservices/face/largepersongroup): LargePersonGroup adatkészlet kezelésére szolgáló [azonosító](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [LargeFaceList felügyeleti API-k](https://docs.microsoft.com/rest/api/cognitiveservices/face/largefacelist): Az egy LargeFaceList kezelésére szolgáló [hasonló](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar).
- [Személy is lehet PersonGroup felügyeleti API-k](https://docs.microsoft.com/rest/api/cognitiveservices/face/persongroupperson): Az arcokat személy is lehet PersonGroup kezelésére szolgáló [azonosító](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [Is lehet PersonGroup felügyeleti API-k](https://docs.microsoft.com/rest/api/cognitiveservices/face/persongroup): Az is lehet PersonGroup adatkészlet kezelésére szolgáló [azonosító](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [Pillanatkép-felügyeleti API-k](https://docs.microsoft.com/rest/api/cognitiveservices/face/snapshot): Több előfizetés között az adatáttelepítés pillanatkép kezelésére szolgál.

# <a name="natural-language-support-for-azure-cognitive-services"></a>Természetes nyelvi támogatás az Azure Cognitive Services

Az Azure Cognitive Services lehetővé teszi, hogy olyan alkalmazásokat építsen fel, amelyek megtekinthetik, meghallgatják és megértik a felhasználókat. Ezek a szolgáltatások között több mint három tucat nyelvet támogat, így a felhasználók természetes módon kommunikálnak az alkalmazással. Az alábbi hivatkozásokra kattintva megtekintheti a nyelv elérhetőségét szolgáltatás alapján.

Ezek a Cognitive Services a nyelvtől függetlenek, és nem rendelkeznek korlátozásokkal a nyelv alapján.

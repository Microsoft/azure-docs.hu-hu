# <a name="featured-user-generated-content-for-the-anomaly-detector-api"></a>Kiemelt, felhasználó által létrehozott tartalom az Anomáliadetektor API-hoz

Ebből a cikkből megtudhatja, hogy más ügyfelek hogyan gondolkodnak, és hogyan használják az Anomáliadetektor API-t. A következő erőforrásokat az Anomáliadetektor-felhasználók közössége hozta létre. Ide tartoznak a nyílt forráskódú projektek, valamint a Microsoft és a külső felhasználók által létrehozott egyéb hozzájárulások. Az alábbi hivatkozások némelyike a Microsofton kívüli webhelyeken található, és a Microsoft nem vállal felelősséget az ottani tartalomért. Körültekintően járjon el, amikor ezekre az erőforrásokra hivatkozik.

# <a name="upload-images"></a>Képek feltöltése

Megismerheti a felhasználói felület, egyszerre legfeljebb 2 MB méretű fájl egyes rendszerképek legfeljebb öt képek feltöltése. Mindenképpen ellenőrizze a fájlméretek feltöltése előtt.

![Képek feltöltése](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>A jóváhagyás küldése
Ha egyszerre több mint öt képek feltöltése, végül, nem kívánatos képek eltávolításához használja a Kuka ikont. Ezután küldje el a képek az automatizált moderálás előnyeit.

![Rendszerképek küldése](images/1-try-image-2.png)

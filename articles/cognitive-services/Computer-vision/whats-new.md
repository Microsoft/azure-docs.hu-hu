### <a name="read-api-30-preview"></a>API 3.0 előzetes verzió olvasása

Most már használhatja a 3.0-s verzió az Olvasás API kivonat nyomtatott vagy kézzel írt szöveget a képeket. A korábbi verziókhoz képest a 3.0 a következőket biztosítja:
* nagyobb pontosság
* új kimeneti formátum
* megbízhatósági pontszám minden egyes kivont szóhoz
* spanyol és angol nyelvek támogatása a kiegészítő nyelvi paraméterrel

Kövesse [a Szöveg kibontása rövid útmutató](https://docs.microsoft.com/azure/cognitive-services/computer-vision/quickstarts/csharp-hand-text?tabs=version-3) t a 3.0 API használatának megkezdéséhez.

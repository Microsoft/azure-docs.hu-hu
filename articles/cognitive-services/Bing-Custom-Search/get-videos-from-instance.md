# <a name="get-videos-from-your-custom-view"></a>Videók letöltése az egyéni nézetből

A Bing egyéni videók keresése lehetővé teszi az egyéni keresési élmény bővítését a videókkal. A webes találatokhoz hasonlóan az egyéni keresés a videókeresést is támogatja azokon a webhelyeken, amelyek a keresési példányhoz engedélyezve vannak. A videókat a Bing Egyéni videók keresési API-jával vagy a Hosztott felhasználói felület funkciójával szerezheti be. A Hosted ui funkció használata egyszerű, és ajánlott a keresési élmény rövid időn belül történő felfuttatásához. A házigazdája felhasználói felület videókra való konfigurálásáról [a Hosztolt felhasználói felület konfigurálása című](hosted-ui.md)témakörben talál.

Ha jobban szeretné szabályozni a keresési eredmények megjelenítését, használhatja a Bing egyéni videók keresési API-ját. Mivel az API-hívás hasonló a Bing Video Search API hívásához, a [Bing Video Search](../Bing-Video-Search/search-the-web.md) kivételezése példákat az API-t. De mielőtt ezt megtenné, ismerkedjen meg az [Egyéni videók keresés API referenciatartalmával.](https://docs.microsoft.com/rest/api/cognitiveservices-bingsearch/bing-custom-videos-api-v7-reference) A fő különbségek a támogatott lekérdezési paraméterek (tartalmaznia kell a customConfig lekérdezési paraméter) és a végpont, amelynek kéréseket küld.

A parancsmagok futtatásához használhatja az Azure Cloud Shell, egy ingyenes interaktív rendszerhéj. A fiókjával való használat érdekében a gyakran használt Azure-eszközök már előre telepítve és konfigurálva vannak rajta. A **Másolás** gombra kattintva másolja és illessze be a kódot a Cloud Shellbe, majd nyomja le az Enter billentyűt a futtatáshoz. Néhány módszer a Cloud Shell indításához:


|  |   |
|-----------------------------------------------|---|
| Nyissa meg a Cloud Shellt a böngészőben. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Kattintson a portál jobb felső részén található menü **Felhőhéj** gombjára. | [![A Cloud Shell a portálon](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Ha nem szeretné használni az Azure CloudShellt, telepítheti helyette a PowerShellt. Ha úgy dönt, hogy helyileg telepíti és használja a PowerShellt, telepítse a Resource Manager PowerShell-parancsmagok legújabb verzióját. A PowerShell-parancsmagok gyakran frissülnek, és általában frissítenie kell a PowerShell-parancsmagokat a legújabb funkciófunkciók lekéréséhez, ennek hiányában, amelyek nem fordulhatnak elő problémák.

A PowerShell helyileg futtatott verziójának megkereséséhez használja a "Get-Module -ListAvailable Az" parancsmag. A frissítéshez [olvassa el az Azure PowerShell-modul telepítése című témakört.](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps) További információt [az Azure PowerShell telepítésével és konfigurálásával](https://docs.microsoft.com/powershell/azure/azurerm/overview) foglalkozó témakörben talál.

Ha a PowerShellt macOS rendszeren használja, kövesse a [PowerShell Core macOS rendszeren történő telepítése](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6) című témakör lépéseit.

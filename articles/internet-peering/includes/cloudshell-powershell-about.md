A parancsmagok futtatásához használhatja az Azure Cloud Shell, egy ingyenes interaktív rendszerhéj. A fiókjával való használat érdekében a gyakran használt Azure-eszközök már előre telepítve és konfigurálva vannak rajta. Válassza a **Másolás** lehetőséget a kód másolásához, majd illessze be a Cloud Shellbe. Ezután a futtatásához válassza az **Enter** lehetőséget. A Cloud Shell indításának több módja is van:


|  |   |
|-----------------------------------------------|---|
| Nyissa meg a Cloud Shellt a böngészőben. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Válassza a **Cloud Shell** gombot az Azure Portal jobb felső részén lévő eszköztáron. | [![A Cloud Shell a portálon](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Ha nem szeretné használni az Azure Cloud Shellt, telepítheti helyette a PowerShellt. Ha úgy dönt, hogy helyileg telepíti és használja a PowerShellt, telepítse az Azure Resource Manager PowerShell-parancsmagok legújabb verzióját. A PowerShell-parancsmagok gyakran frissülnek. Általában frissítenie kell a PowerShell-parancsmagokat a legújabb funkciófunkciók lekéréséhez. Ha nem, akkor problémákba ütközhet.

A PowerShell helyileg futtatott verziójának megkereséséhez használja a **"Get-Module -ListAvailable Az"** parancsmag. A frissítéshez [olvassa el az Azure PowerShell-modul telepítése című témakört.](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps) További információt az [Azure PowerShell telepítéséről és konfigurálásáról talál.](https://docs.microsoft.com/powershell/azure/azurerm/overview)

Ha a PowerShellt macOS rendszeren használja, kövesse a [PowerShell macOS rendszeren történő telepítése](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6) című témakör lépéseit.

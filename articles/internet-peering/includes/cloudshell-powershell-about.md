A parancsmagok futtatásához használhatja a Azure Cloud Shellt, amely egy ingyenes interaktív rendszerhéj. A fiókjával való használat érdekében a gyakran használt Azure-eszközök már előre telepítve és konfigurálva vannak rajta. Válassza a **Másolás** elemet a kód másolásához, majd illessze be Cloud Shellba. Ezután válassza az **ENTER billentyűt** a futtatásához. A Cloud Shell elindításának néhány módja van:


| Indítási módszer | Képernyőfelvétel  |
|-----------------------------------------------|---|
| Nyissa meg a Cloud Shellt a böngészőben. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Kattintson a **Cloud Shell** gombra a Azure Portal jobb felső sarkában található eszköztáron. | [![A Cloud Shell a portálon](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Ha nem kívánja használni a Azure Cloud Shellt, helyette a PowerShellt helyileg is telepítheti. Ha a PowerShell helyi telepítése és használata mellett dönt, telepítse a Azure Resource Manager PowerShell-parancsmagok legújabb verzióját. A PowerShell-parancsmagok gyakran frissülnek. A legújabb funkciók beszerzéséhez általában frissítenie kell a PowerShell-parancsmagokat. Ha nem, problémák merülhetnek fel.

A helyileg futtatott PowerShell-verzió megkereséséhez használja a **"Get-Module-ListAvailable az"** parancsmagot. A frissítéshez tekintse meg [a Azure PowerShell modul telepítését](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)ismertető témakört. További információ: [Azure PowerShell telepítése és konfigurálása](https://docs.microsoft.com/powershell/azure/azurerm/overview).

Ha macOS rendszeren használja a PowerShellt, kövesse a [PowerShell telepítése MacOS rendszeren](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6) című témakör lépéseit.

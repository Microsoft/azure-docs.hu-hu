# <a name="azure-storage-compliance-offerings"></a>Az Azure Storage megfelelőségi ajánlatai

Annak érdekében, hogy a szervezetek megfeleljenek az egyének adatainak gyűjtésére és felhasználására vonatkozó nemzeti, regionális és iparág-specifikus követelményeknek, a Microsoft Azure & az Azure Storage bármely felhőszolgáltató legátfogóbb tanúsítványkészletét és tanúsítványait kínálja.

Az azure storage-beli megfelelőségi ajánlatok at alább találva biztosíthatja, hogy az Azure Storage-szolgáltatás használatával szabályozott szolgáltatása. Ezek a következő Azure Storage-ajánlatok: Blobok, fájlok, várólisták, táblák, lemezek, cool storage és prémium szintű storage.

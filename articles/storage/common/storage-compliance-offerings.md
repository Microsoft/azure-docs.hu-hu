# <a name="azure-storage-compliance-offerings"></a>Az Azure Storage-megfelelőségi ajánlatok

Megkönnyíti a szervezetek számára ahhoz, hogy a gyűjtemény és a egyének adatok, Microsoft Azure és az Azure Storage használatát szabályozó országos, regionális és iparág-specifikus követelmények kínálnak a legátfogóbb minősítései közül is bemutat, és a tanúsítványokat, bármely felhőszolgáltató.

Megfelelőségi ajánlatok az Azure Storage annak biztosítása érdekében a szolgáltatás az Azure Storage szolgáltatás segítségével szabályozott alább találja. Azok alkalmazhatók a következő Azure Storage-ajánlatok: Blobok, fájlok, üzenetsorok, táblák, lemezek, ritkán használt adatok tárolási és a Premium Storage.

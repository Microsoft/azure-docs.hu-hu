# <a name="azcopy-doc"></a>azcopy doc

Markdown formátumú dokumentációt hoz létre az eszközhöz.

## <a name="synopsis"></a>Áttekintés

Létrehozza az eszköz dokumentációját Markdown formátumban, és a kijelölt helyen tárolja őket.

Alapértelmezés szerint a fájlok egy "doc" nevű mappában vannak tárolva az aktuális könyvtárban.

```azcopy
azcopy doc [flags]
```

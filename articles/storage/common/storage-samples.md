# <a name="azure-storage-samples"></a>Azure Storage-minták

Az alábbi hivatkozásoksegítségével megtekintheti és letöltheti az Azure Storage mintakódját és alkalmazásait.

## <a name="azure-code-samples-library"></a>Azure-kódminták tára

Az [Azure Code Samples](https://azure.microsoft.com/documentation/samples/?service=storage) függvénytár mintákat tartalmaz az Azure Storage, amely helyileg letölthető és futtatható. A Kódmintatár .zip formátumú mintakódot biztosít. Azt is megteheti, hogy minden egyes minta github-tárháza tallózik és klónozhatja.

## <a name="net-samples"></a>.NET-minták

A .NET-minták megismeréséhez töltse le a [.NET storage ügyfélkönyvtárat](https://www.nuget.org/packages/WindowsAzure.Storage/) a NuGet programból. A .NET storage-ügyfélkódtár az [Azure SDK for .NET](https://azure.microsoft.com/downloads/)szolgáltatásban is elérhető.

* [.NET-keretrendszert használó Azure Storage-minták](storage-samples-dotnet.md)

## <a name="java-samples"></a>Java-példák

A Java-minták felfedezéséhez töltse le a [Java Storage Ügyfélkönyvtárat.](https://github.com/azure/azure-storage-java)

* [Javát használó Azure Storage-minták](storage-samples-java.md)

## <a name="python-samples"></a>Python-példák

A Python-minták megismeréséhez töltse le a [Python Storage-ügyfélkönyvtárat.](https://github.com/azure/azure-storage-python)

* [Azure Storage-minták python használatával](storage-samples-python.md)

## <a name="nodejs-samples"></a>Node.js-példák

A Node.js minták feltárásához töltse le a [Node.js storage ügyfélkönyvtárát.](https://github.com/Azure/azure-storage-node)

* [Azure Storage-minták JavaScript/Node.js használatával](storage-samples-javascript.md)

## <a name="c-samples"></a>C++ minták

A C++ minták felfedezéséhez töltse le a [C++ storage ügyfélkönyvtárat](https://www.nuget.org/packages/wastorage/) a NuGet ből.

* [A blobok – első lépések](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/BlobsGettingStarted)
* [A táblázatok – első lépések](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/TablesGettingStarted)
* [Bevezetés az üzenetsorok használatába](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/QueuesGettingStarted)

## <a name="azure-cli"></a>Azure CLI

* [Ismerkedés az Azure CLI-vel](/cli/azure/get-started-with-azure-cli)

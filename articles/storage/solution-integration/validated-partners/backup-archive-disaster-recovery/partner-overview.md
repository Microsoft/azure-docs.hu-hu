# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Azure Storage-Archívum, biztonsági mentési és vész-helyreállítási partnerek

Ez a cikk az Azure Storage szolgáltatással integrált Microsoft-partnereket ismerteti az archiválás, a biztonsági mentés, valamint az üzletmenet-folytonosság és a vész-helyreállítási (BCDR) munkaterhelések esetében. Ezek a partneri megoldások kihasználhatják az Azure Storage méretezési és költséghatékonysági előnyeit. A megoldások segítségével megoldhatja a biztonsági mentési kihívásokat, helyreállíthatja a vész-helyreállítási helyet, vagy archiválhatja a nem használt tartalmakat a hosszú távú megőrzés érdekében. Az Azure Storage összes megfelelőségi szabványa, valamint az Azure-szolgáltatások, például a [megváltoztathatatlan tárolási](../../../blobs/storage-blob-immutable-storage.md) és [életciklus-kezelés](../../../blobs/storage-lifecycle-management-concepts.md)révén ezek a megoldások egyszerűen lecserélhetik a szalagos biztonsági mentéseket, és igény szerinti gazdaságos helyreállítási helyet kínálnak.

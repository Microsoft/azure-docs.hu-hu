# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Azure Storage-partnerek elsődleges és másodlagos tároláshoz

Ez a cikk azokat a Microsoft-partneri vállalatokat mutatja be, amelyek olyan Azure-beli vagy hibrid megoldáson alapuló NAS-vagy SAN-megoldásokat biztosítanak, amelyek költséghatékony szinten használják az Azure Storage szolgáltatást. Ezek a megoldások lehetővé tehetik, hogy az ügyfelek ugyanazt a megoldást használják bármely környezetében.

# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Az Azure Storage adatirányítási, felügyeleti és áttelepítési partnerei

Ez a cikk az Azure Storage szolgáltatással integrált Microsoft-partneri vállalatokat emeli ki, amelyek javítják a teljes adatkezelési képességeket. Ezek a partneri megoldások támogatják a tárolás értékelését és a jelentéskészítést, a platform-független migrációt, a replikálást, a Felhőbeli rétegeket és az adatszabályozást.

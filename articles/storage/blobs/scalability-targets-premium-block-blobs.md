# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>Méretezhetőségi célok a prémium szintű blokkblob-tárfiókokhoz

A prémium szintű teljesítményű blokkblob-tárfiók kisebb, kilobájtos objektumokat használó alkalmazásokhoz van optimalizálva. Ideális olyan alkalmazásokhoz, amelyek magas tranzakciós arányt vagy konzisztens kis késésű tárolást igényelnek. Prémium szintű teljesítmény blokk blob storage célja, hogy az alkalmazásokkal való méretezés. Ha a forgatókönyv megköveteli, hogy telepítse az alkalmazások(ok), amelyek több százezer kérelmek másodpercenként vagy petabájt nyi tárolási kapacitás, forduljon a Microsoft az [Azure Portalon](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)támogatási kérelem benyújtásával.

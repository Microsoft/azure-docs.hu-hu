# <a name="azure-files-and-azure-netapp-files-comparison"></a>Azure Files és Azure NetApp Files összehasonlítása

Ez a cikk összehasonlítja a Azure Files és Azure NetApp Files. 

A felhőalapú fájltárolást igénylő legtöbb számítási feladat Azure Files vagy Azure NetApp Files. A számítási feladathoz leginkább illeszkedő megoldás meghatározásához tekintse át az ebben a cikkben található információkat. További információért tekintse meg a Azure Files [és](https://docs.microsoft.com/azure/storage/files/) [Azure NetApp Files](https://docs.microsoft.com/azure/azure-netapp-files/) dokumentációját.

# <a name="azure-remote-rendering-asset-tool-arrt"></a>Azure távoli renderelési eszköz (ARRT)

Az Azure távoli renderelési eszköz (ARRT) egy, a C++/QT.-ben kifejlesztett, nyílt forráskódú asztali alkalmazás

![ARRT](./media/azure-remote-rendering-asset-tool.png "ARRT képernyőkép")

## <a name="features"></a>Szolgáltatások

Az alkalmazás az alábbiakra használható:

* 3D modell feltöltése
* A modell átalakításának szabályozása
* Távoli renderelési munkamenet létrehozása és kezelése
* 3D-modell betöltése
* 3D modell előzetes verziójának megtekintése
* Az anyagok módosítása

Nyílt forráskódú mintaként is használható, amelyből megismerheti, hogyan valósítható meg az Azure Remote rendering C++ SDK kezelőfelülete az Azure Storage ügyféloldali kódtár használatával a 3D-modell átalakításának kezeléséhez.

# <a name="azure-remote-rendering-tutorials"></a>Azure távoli renderelési oktatóanyagok

Üdvözöljük az Azure távoli renderelési oktatóanyagokban. Az Azure Remote rendering legfontosabb alapfogalmai mellett ezek az interaktív leckék a távolról renderelt modellek megtekintését, kezelését és testreszabását, valamint a biztonságos, kereskedelmi használatra kész alkalmazások kiemelésének szempontjait ismertetik.

### <a name="azure-remote-rendering-tutorials"></a>Azure távoli renderelési oktatóanyagok

Javasoljuk, hogy az alábbi oktatóanyagokat az előzőekben ismertetett módon végezze el.

#### <a name="viewing-a-remotely-rendered-model"></a>[Távolról renderelt modell megtekintése](view-remote-models/view-remote-models.md)

Ebben az oktatóanyagban egy alapszintű alkalmazást fog létrehozni, amely képes a távolról renderelt modellek megtekintésére az Azure távoli renderelés használatával.

#### <a name="interfaces-and-custom-models"></a>[Felületek és egyéni modellek](custom-models/custom-models.md)

Ezután megtudhatja, hogyan töltheti be saját modelljeit az Azure távoli Renderelésbe, és megtekintheti őket az alkalmazásban. A [távolról megjelenített modellek megtekintése](view-remote-models/view-remote-models.md) az oktatóanyag előfeltétele.

#### <a name="manipulating-models"></a>[Modellek módosítása](manipulate-models/manipulate-models.md)

Ebben az oktatóanyagban a távolról renderelt modelljeit fogja kezelni és kezelni. Javasoljuk, hogy az oktatóanyag előtt végezzen el [egy távolról renderelt modellt](view-remote-models/view-remote-models.md) , [felületet és egyéni modellt](custom-models/custom-models.md) .

#### <a name="refining-materials-lighting-and-effects"></a>[Az anyagok, a megvilágítás és az effektusok finomítása](materials-lighting-effects/materials-lighting-effects.md)

Ezután olyan fontos fogalmakat fogunk kimutatni, mint például az anyagok módosítása, a megvilágítás módosítása és a hatások alkalmazása a távolról renderelt modelleken. Az oktatóanyaghoz [egy távolról renderelt modell](view-remote-models/view-remote-models.md), [interfész és egyéni modell](custom-models/custom-models.md), valamint a [modellek módosítása](manipulate-models/manipulate-models.md) ajánlott.

#### <a name="securing-azure-remote-rendering-and-model-storage"></a>[Az Azure Remote Rendering biztonságossá tétele és a modellek tárolása](security/security.md)

Itt megismerheti az Azure távoli renderelési fiók és a blob Storage biztonságossá tételét. Ez ismerteti az Azure Remote rendering szolgáltatást használó kereskedelmi alkalmazások kiadásának első lépéseit. Ez az oktatóanyag feltételezi, hogy végrehajtotta a fenti leckéket.


#### <a name="creating-a-commercial-ready-azure-remote-rendering-application"></a>[Forgalmazásra kész Azure Remote Rendering-alkalmazás létrehozása](commercial-ready/commercial-ready.md)

Végezetül megbeszéljük a kereskedelmi alkalmazások Azure-beli távoli rendereléssel való előkészítésével kapcsolatos fontos szempontokat és tippeket. Ez akkor hasznos, ha az erőforrás kijelentkezése előtt végrehajtotta az összes Azure-beli távoli renderelési oktatóanyagot.

**Ismerkedés a** [távolról renderelt modellek megtekintésével](view-remote-models/view-remote-models.md).

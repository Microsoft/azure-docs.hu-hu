# <a name="microsoft-azure-documentation"></a>Microsoft Azure-dokumentáció

Üdvözli a nyílt forráskódú [dokumentáció](https://docs.microsoft.com/azure) , [Microsoft Azure](https://azure.microsoft.com). Tekintse át az információs fájl megérteni, milyen segítséget nyújthat a Microsoft Azure-dokumentáció hozzájárulni.

## <a name="getting-started"></a>Első lépések

Közreműködés a nyílt forráskód több, mint hogy frissítések, akkor is megosztja velünk, ha probléma van. Olvassa el a [közreműködői útmutató](.github/CONTRIBUTING.md) további.

### <a name="prerequisites"></a>Előfeltételek

Biztosít, amely nagyszerű hozzájárulás döntött! Van néhány eszközök, amely a dokumentációban való közreműködéshez szükségesek.

Hozzájárul a dokumentáció GitHub-fiókra van szükség, ha nincs fiókja, kövesse az utasításokat a [GitHub-fiók beállítása](https://docs.microsoft.com/contribute/get-started-setup-github) a közreműködői útmutatóban.

#### <a name="download"></a>Letöltés

Telepítse az alábbi eszközöket:

* [Git](https://git-scm.com/download)
* [Visual Studio Code](https://code.visualstudio.com/Download)
* [Közreműködői csomag a docs](https://marketplace.visualstudio.com/items?itemName=docsmsft.docs-authoring-pack) Visual Studio Code-bővítmény

#### <a name="install"></a>Telepítés

Kövesse az utasításokat a [tartalomkészítő eszközök telepítése](https://docs.microsoft.com/contribute/get-started-setup-tools) a közreműködői útmutatóban.
